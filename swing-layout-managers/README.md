# Java Swing Layout Managers

Master layout management in Java Swing applications.

## Prerequisites

- Java 21+ (JDK)
- Maven 3.9+
- IDE with Java support (IntelliJ IDEA or VS Code)
- Basic understanding of Swing components

## Setup

```bash
# Navigate to starter code
cd initial/

# Build the project
mvn compile

# Run the application
mvn exec:java
```

## Learning Objectives

By completing this tutorial, you will learn:

- FlowLayout for simple sequential layouts
- BorderLayout for regional positioning
- GridLayout for grid-based layouts
- GridBagLayout for complex layouts
- BoxLayout for vertical/horizontal stacking
- Combining layout managers
- Custom layout strategies
