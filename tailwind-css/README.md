# Tailwind CSS Examples

Working examples demonstrating Tailwind CSS utility classes and patterns.

## Prerequisites

- Node.js 18+
- npm or pnpm

## Setup

```bash
cd examples/

# Install dependencies
npm install

# Start development server
npm run dev
```

## Examples

| File | Description |
|------|-------------|
| `buttons.html` | Button variants and states |
| `cards.html` | Card component patterns |
| `forms.html` | Form styling and validation states |
| `layout.html` | Flexbox and Grid layouts |
| `responsive.html` | Responsive design patterns |
| `dark-mode.html` | Dark mode implementation |
