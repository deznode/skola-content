# Introdusun a Python

Bem-vindu pa es tutorial! Nos ta kumesa ku Python, un lingua di programasun simples ma poderoza.

## O ki é Python?

Python é un lingua di programasun ki ta uza pa:
- Dezenvolvimentu web
- Analizi di dadus
- Inteligensa artifisial
- Automatizasun

## Bo primeiru programa

```python
# Es é un komentariu
print("Ola Mundu!")
```

Kuandu bu kori es kodigo, el ta imprime "Ola Mundu!" na konsola.

## Variavel

Variavel é komu kaixa ki ta guarda valur:

```python
nomi = "Maria"
idadi = 25
print(f"Nha nomi é {nomi} i N tene {idadi} anu")
```

## Siginti pasu

Na prosimo tutorial, nos ta aprendi sobri:
- Tipu di dadus
- Operadores
- Estruturas di kontroli
