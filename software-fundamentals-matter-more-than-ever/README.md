## Description

A comprehensive tutorial series in Kabuverdianu (Creole) exploring why software engineering fundamentals — data structures, algorithms, design patterns, SOLID principles, testing, performance, and security — hold enduring value across an entire engineering career, regardless of which frameworks come and go.

Derived from a source lecture covering the perils of trend-chasing and the compounding returns of deep, principled knowledge investment.

## Learning Outcomes (Module 1)

After completing Module 1, learners will be able to:

1. Explain why software engineering fundamentals retain value across entire careers
2. Recognize the pattern of technological hype cycles and their recurrence
3. Build a personal mental model for filtering trend noise from lasting knowledge
4. Identify the psychological pull of Resume-Driven Development and its hidden costs
5. Define a deliberate skill-investment strategy prioritizing depth over breadth
6. Name and briefly describe each of the 7 core fundamental areas
7. Understand how the 7 areas interconnect and reinforce one another
