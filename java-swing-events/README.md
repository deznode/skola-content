# Java Swing Events

Learn event handling in Java Swing applications.

## Prerequisites

- Java 21+ (JDK)
- Maven 3.9+
- IDE with Java support (IntelliJ IDEA or VS Code)

## Setup

```bash
# Navigate to starter code
cd initial/

# Build the project
mvn compile

# Run the application
mvn exec:java
```

## Learning Objectives

By completing this tutorial, you will learn:

- ActionListener for button clicks
- MouseListener and MouseMotionListener
- KeyListener for keyboard events
- WindowListener for window events
- Event adapters and inner classes
- Lambda expressions for event handlers
