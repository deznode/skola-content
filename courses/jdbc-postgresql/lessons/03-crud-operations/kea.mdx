### Ler dadus (Read)

`executeQuery()` ta devolve un `ResultSet` ki bu ta perkore ku `rs.next()`:

```java
public static void listaUzuariusTudu(Connection conn) throws SQLException {
    String sql = "SELECT id, nomi_uzuariu, email FROM uzuarius";

    try (PreparedStatement pstmt = conn.prepareStatement(sql);
         ResultSet rs = pstmt.executeQuery()) {

        while (rs.next()) {
            System.out.printf("ID: %d | Uzuáriu: %s | Email: %s%n",
                              rs.getInt("id"),
                              rs.getString("nomi_uzuariu"),
                              rs.getString("email"));
        }
    }
}
```

Pa atxa un fila spesífiku, djunta un `WHERE` ku un `?`:

```java
public static void atxaUzuariuPorEmail(Connection conn, String email) throws SQLException {
    String sql = "SELECT id, nomi_uzuariu FROM uzuarius WHERE email = ?";

    try (PreparedStatement pstmt = conn.prepareStatement(sql)) {
        pstmt.setString(1, email);
        try (ResultSet rs = pstmt.executeQuery()) {
            if (rs.next()) {
                System.out.printf("Atxadu: %s (ID: %d)%n",
                                  rs.getString("nomi_uzuariu"), rs.getInt("id"));
            } else {
                System.out.println("Uzuáriu ka atxadu.");
            }
        }
    }
}
```

:::callout{type=tip}
Sénpri uza nomi di koluna (`rs.getString("email")`) en bez di pozisan — é más lejível i ta sobrevive reordenasan di koluna.
:::

## Update i Delete

```java
public static int atualizaEmailUzuariu(Connection conn, int uzuariuId, String nobuEmail)
        throws SQLException {

    String sql = "UPDATE uzuarius SET email = ? WHERE id = ?";

    try (PreparedStatement pstmt = conn.prepareStatement(sql)) {
        pstmt.setString(1, nobuEmail);
        pstmt.setInt(2, uzuariuId);
        return pstmt.executeUpdate();   // númeru di fila atualizadu
    }
}

public static int apagaUzuariu(Connection conn, int uzuariuId) throws SQLException {
    String sql = "DELETE FROM uzuarius WHERE id = ?";

    try (PreparedStatement pstmt = conn.prepareStatement(sql)) {
        pstmt.setInt(1, uzuariuId);
        return pstmt.executeUpdate();
    }
}
```

:::callout{type=danger}
`UPDATE`/`DELETE` sen `WHERE` ta afeta **TUDU** fila. Kore `UPDATE uzuarius SET email = '...'` sen `WHERE` ta muda email di tudu uzuáriu. Verifika dublu `WHERE`.
:::

:::callout{type=success}
Na es pontu bu debe konsigi konekta, kria tabela, inseri, ler, atualiza, i apaga. Si algun ka ta funsiona, konsulta lisan **Rezolve Problema**.
:::
