# O que é Kotlin?

Kotlin é uma linguagem de programação moderna criada pela JetBrains. Ela é:

- **Concisa**: Escreva menos código, faça mais
- **Segura**: Evite erros comuns com null safety
- **Interoperável**: Funciona com Java perfeitamente

## Por que usar Kotlin?

1. Desenvolvimento Android (recomendação oficial do Google)
2. Backend com Spring Boot
3. Multiplataforma (iOS, Web, Desktop)

## Exemplo

```kotlin
fun main() {
    println("Olá Mundo!")
}
```
