# Kuzé ki Kotlin?

Kotlin é un linguajen di programasaun modernu kriadu pa JetBrains. El é:

- **Konsizu**: Skrebi menus kódigu, fazi más
- **Seguru**: Evita errus komus ku null safety
- **Interoperável**: Funsiona ku Java perfeitamenti

## Pa ki uza Kotlin?

1. Dezenvolvimentu Android (rekomendu ofisal di Google)
2. Backend ku Spring Boot
3. Multiplataforma (iOS, Web, Desktop)

## Esemplu

```kotlin
fun main() {
    println("Ola Mundu!")
}
```
