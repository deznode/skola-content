# Kuzé ki Kotlin?

Bu kaba di prende bu primeru linguajen di programasan. Gosi, bu sta li pa kumesa un novu — **Kotlin**. Un pergunta natural bu pode ta faze: pamodi meste prende otu linguajen, si bu ja sabe programa?

Resposta é sinplis: kada linguajen ta abri portas diferenti. Kotlin ta poi-bu na un di kes ekosistema más grandi di dizenvolvimentu di software di mundu — di apps Android te sistema backend di grandi enpresa. I midjór di tudu: kes konseitu ki bu ja prendi (variavel, tipu, kondisan, loop) ta transferi direitamenti pa Kotlin.

<SectionHeading variant="concept" seq={1}>Kuzé ki Kotlin É</SectionHeading>

Kotlin é un linguajen di programasan modernu, kriadu pa **JetBrains** — un enpresa konxidu pa si ferramentas di dizenvolvimentu, sima IntelliJ IDEA. Kotlin kore riba di un "mákina virtual" txomadu **JVM** (Java Virtual Machine) — mesmu mákina unde Java ta kore.

<SectionHeading variant="concept" seq={2}>Kotlin i Java: Dos Linguajen, Un Só Mákina</SectionHeading>

Es é un di kes fatu más distintu di Kotlin: el ka é só "primu" di Java — el ta kore na **mesmu JVM**, i pode uza kódiku Java direitamenti, sen konversan, sen adapter. Es propriedadi ta txomadu **interoperability**, na Inglés. Kel funsiona pamodi Kotlin i Java, tudu dos, ta kompiladu pa mesmu formatu — bytecode JVM — antis di kore. JVM ka ta liga pa ki linguajen kel bytecode nase, kel é pamodi Kotlin i Java pode partilha kódiku. Ma Python i JavaScript ka pode faze kel-li — kada un ta kore riba di un mákina virtual diferenti, ku formatu internu diferenti. Na prátika, kel signifika:

- Bu pode txoma un funsan Java dentu di un programa Kotlin.
- Bu pode txoma un funsan Kotlin dentu di un programa Java.
- Tudu biblioteka Java — i ten txeu, dezdi web te dadus — sta disponível pa Kotlin tanbe, sen trabadju extra.

Es é pamodi txeu empresa ki ja ten sistema Java grandi sta ta adisiona Kotlin dja — es pode kombina dos linguajen dentu di mesmu prujetu, sen perde tudu kódiku Java ki es ja ten.

<SectionHeading variant="concept" seq={3}>Unde Kotlin Ta Uzadu</SectionHeading>

Kotlin ka é un linguajen di un só área — el ta kobri tres kaminhu prinsipal:

- **Dizenvolvimentu Android** — dezdi 2019, Kotlin é **linguajen ofisialmenti rekomendadu pa Google** pa kria apps Android.
- **Backend** — ku framework sima **Spring Boot**, Kotlin ta kria servidor i **API** (Application Programming Interface — un interface pa otu sistema uza si servisu) pa aplikasan web.
- **Multiplataforma** — ku Kotlin Multiplatform, un só kódiku pode partilha lójika entri **iOS, Web i Desktop**, ka só Android.

<SectionHeading variant="concept" seq={4}>Pamodi Kotlin É Popular</SectionHeading>

Dos propriedadi ta distingui Kotlin di txeu otu linguajen:

- **Menus kódiku**: Kotlin ta skrebe menus kódiku pa faze mesmu kuza ki Java. Un programa ki na Java ta leba txeu linha, na Kotlin txeu bez ta leba menus di metadi.
- **Null safety**: Kotlin ta djuda evita un kategoria interu di erru — famozu `NullPointerException` (ou "NPE") — pamodi el ta obriga-bu deklara di manera klaru kual variavel pode ser `null` i kual ka pode.

Es dos propriedadi ka é só un splika kurtu — es ta paresi na tudu lisan di es kursu. Kada bez ki bu ta prende un konseitu novu (variavel, fluxu di kontrolu, funsan, koleson, ou klasis), bu ta odja kumo Kotlin ta faze-l di manera más kurtu i más seguru ki manera tradisional.

<SectionHeading variant="concept" seq={5}>Bu Purmeru Programa Kotlin</SectionHeading>

Tudu linguajen di programasan ten un tradisan: primeru programa ki bu ta skrebe é un ki ta imprimi un kumprimentu sinplis.

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Ola Mundu!");
    }
}
```
