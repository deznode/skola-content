This is a **TEXT lesson**. It exercises every Markdown element we support: headings, lists, tables, blockquotes, inline code, and a code block with *highlight* notation.

## Level 2 Heading

Normal text with **bold**, *italic*, and inline code like `print("hello")`.

### Level 3 Heading

2. Second item
3. Third item

An unordered list:

- Apple

## Code Block with Highlight

```python
def greeting(name):
    message = f"Hello, {name}!"  # [!code highlight]
    return message

print(greeting("World"))
```

In the block above, the `message` line should be marked with a highlight color.
