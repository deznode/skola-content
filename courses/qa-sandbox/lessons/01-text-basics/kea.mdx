Es lison é un **lison di tipu TEXT**. E ta esersi tudu elementu di Markdown ki nu ta suporta: títulus, listas, tabelas, blockquotes, kódigu inline, i un bloku di kódigu ku notason di *highlight*.

## Títulu di Nível 2

Tekstu normal ku **negritu**, *itáliku*, i kódigu inline tipu `print("ola")`.

### Títulu di Nível 3

Un lista ordenadu:

1. Priméru iten
2. Sigundu item
3. Terséru item

Un lista nan ordenadu:

- Mason
- Banana
- Laranja

## Blockquote

> Es é un blockquote. E ta uza pa testa renderizason di sitasan i nota importanti.

## Bloku di Kódigu ku Highlight

```python
def saudasan(nomi):
    mensajen = f"Ola, {nomi}!"  # [!code highlight]
    return mensajen

print(saudasan("Mundu"))
```

Na liña di riba, a liña di `mensajen` debe ta marka ku un kor di destake.
