# Parity-Gate Reference Lesson

Es lison TEXT ta esersi figura (infográfiku) ku lightbox, multi-linguajen na blocos di kódigu, tudu tres callouts, **i** tipografia kompletu + komponentes interativos (Quiz + CodeExample) — spec 048 T-09.

## Typography Spine

Es seksaun ta esersi h1 (di riba), h2 (es), h3, h4, inline `code`, anchor, i lista ka-ordenadu — pa garanti parity gate ta deteta drift na qualker konponenti tipográfiku.

### Sub-seksaun nível 3

Pa [skola.dev](https://skola.dev) renderiza es nivel di profundidade konsistente entri apps/web i pipeline-app, kada surface ten ki uza es mesmu komponentes lcc.

#### Sub-sub-seksaun nível 4

Mesmu na profundidade kuatu, klase Tailwind di lcc `typography.tsx` ta resolve igualmente.

- Item un di lista ka-ordenadu
- Item dos ku inline `value` na meiu
- Item tres ku link [pa MDN](https://developer.mozilla.org/)

## Blocos di Kódigu Multi-Linguajen

```python
# Python
numerus = [1, 2, 3]
total = sum(numerus)
print(f"Total: {total}")
```

```javascript
// JavaScript
const numbers = [1, 2, 3];
const total = numbers.reduce((a, b) => a + b, 0);
console.log(`Total: ${total}`);
```

```bash
# Bash
echo "Total: $((1 + 2 + 3))"
```

```sql
-- SQL
SELECT SUM(valor) AS total FROM numerus;
```

## Komponentes Interativos

A baxu ten un Quiz ku un CodeExample — anbus ta karega tras YAML sibling (`quiz.yaml`, `code-examples.yaml`):

<Quiz position={0} />

<CodeExample position={1} />

> Blockquote final pa testa renderizason djuntu ku tudu otru elementu.
