## Blocos di Kódigu Multi-Linguajen

```python
# Python
numerus = [1, 2, 3]
total = sum(numerus)
print(f"Total: {total}")
```

```javascript
// JavaScript
const numbers = [1, 2, 3];
const total = numbers.reduce((a, b) => a + b, 0);
console.log(`Total: ${total}`);
```

```bash
# Bash
echo "Total: $((1 + 2 + 3))"
```

```sql
-- SQL
SELECT SUM(valor) AS total FROM numerus;
```

## Tudu Callouts

:::callout{type=tip}
Callout di tipu **tip** djuntu ku figura i kódigu.
:::

:::callout{type=warning}
Callout di tipu **warning** — verifika a kor di avizu.
:::

:::callout{type=info}
Callout di tipu **info** — verifika a kor di informason.
:::

> Blockquote final pa testa renderizason djuntu ku tudu otru elementu.
