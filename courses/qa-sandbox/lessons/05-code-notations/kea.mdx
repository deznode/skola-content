Es lison ta esersi tudu tres transformadoris di Shiki ki nu ta suporta: **highlight**, **diff**, i **focus** — algun djuntu na mesmu bloku.

## Highlight + Diff Djuntu

```python
def proseso(dadus):
    resultadu = []
    for item in dadus:
        valor = item * 2  # [!code highlight]
        resultadu.append(valor)  # [!code --]
        resultadu.append(valor + 1)  # [!code ++]
    return resultadu
```

A linha di `valor` debe ta destaka; a linha di `append(valor)` debe ta vermedju; a linha di `append(valor + 1)` debe ta verdi.

## Focus

```javascript
function setup() {
  init();
  loadConfig();  // [!code focus]
  start();
}
```

So a linha ku `loadConfig()` debe sta na fokus; o restu debe ta más éskuru/desfokadu.
