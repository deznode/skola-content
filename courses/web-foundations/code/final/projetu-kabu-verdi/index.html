Kuzinha di Kabu Verdi

Katxupa, pastel, lagosta, grogue

Katxupa é pratu nasional — milhu, fijon i karni kuzinhadu lentamenti ti tudu sabor ta junta. Pastel di milhu pa sobremeza. Lagosta freska di mar pa otkazion speshal. I pa fin di refeson, un kopu di grogue.
