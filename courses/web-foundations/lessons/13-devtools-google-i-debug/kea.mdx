# DevTools, Google i debug

Ti gosi, kuandu un kuza dá erradu, bu ta fika paradu. Es lisan li bu ta aprende **debug** — kumo un kuzinheru ki ta proba sosa antis di sirvi. Ferramentas ki kada profisional di web ta uza tudu dia: DevTools, Google, MDN (Mozilla Developer Network). Kátia, ki ta aprende web na Praia, ja perdeu dos hora un dia pa odja ki problema éra so un erru di skritura na nomi di propriedadi. DevTools ta poupa-l tempu sima es.

## Chrome DevTools — tour

### Abri DevTools

- **Atallu:** `Cmd+Opt+I` (Mac) ou `Ctrl+Shift+I` (Windows/Linux).
- **Ku rato:** klika ku rato direita na kualkér elementu di pajina → **Inspect** / **Inspekciona**.

DevTools ta abri na un painel lateral ou na un janela separadu.

### Painel **Elements** — strutura HTML live

Ta mostra árvore di HTML kumo browser ta vé-l gosi — ka kumo bu skrebe-l. Si JavaScript adisiona ou remove elementu, Elements ta atualiza.

- Pasa rato sobre kualkér linha → kel elementu ta destaka na pajina.
- Klika ku rato direita → opsan pa edita HTML diretu, kopia, etc.

### Painel **Styles** — kuze ki ta aplika

Mostra **tudu regra CSS ki ta aplika** pa elementu selecionadu, ordenadu di mas spesífiku pa mas jeral. Na fundu, na itáliku, ta ben **user-agent styles** (kor i tamanhu padron di browser).

| Kuza ki bu ta odja | Signifikadu |
|---|---|
| Regra normal | Ta aplika gosi |
| Regra **tachiada** (riska na sentru) | Sobrepasadu pa otru regra mas spesífiku |
| Linha ku ikon di triángulu amarelu | Erru di skritura ou valor inválidu |

### Edita live ku flecha

Klika na kualkér valor numériku (egzemplu, `font-size: 16px`). Kumo kel valor ta keda azul (modu di edisan), pode:

- **↑ / ↓** — inkrementa ou dekrementa por 1.
- **Shift + ↑/↓** — inkrementa ou dekrementa por **10**.
- **Alt + ↑/↓** — inkrementa ou dekrementa por **0.1**.

Edisons na DevTools **ka ta salva na ficheru**. Tudu ta pirdi kuandu bu rikarrega pajina. Uza DevTools pa **eksperimenta**, dipos kopia valor finál pa `style.css`.

### Forsa stadu `:hover`

Pseudo-classes (`:hover`, `:focus`, `:active`) é difisil di debug pamodi es ta dezaparesi kuandu bu sai ku rato. Solusan: na topu di painel Styles, klika `:hov`. Marka kaixa pa `:hover`. Stadu ta keda "ligadu" — i bu pode inspekciona regra di hover ku kalmu.

### Box model na kantu direitu

Na painel Computed (ou embaxu di Styles), DevTools ta dezenha un diagrama vivu di box model di elementu selecionadu. Bu ta odja eksatamenti kuantu `padding`, `border` i `margin` el ten. Ferramenta krítika pa debug spasing.

:::callout{type=tip}
**Primeru kuza ki bu ta odja kuandu CSS ka funsiona:** sera ki `style.css` ta linka? Abri painel **Network**, rikarrega pajina, prokura `style.css`. Si **stadu é 404**, atributu `href` ou lokalizasan di ficheru ta eradu.
:::

## W3C HTML Validator

Pa kontrola ki HTML é váldidu (tag fixu korreta, atributus obrigatóriu prizenti), uza [validator.w3.org](https://validator.w3.org). Es ferramenta ofisial di W3C (organizasan ki manten standard di web).

1. Vai pa `validator.w3.org`.
2. Pega tab **"Validate by Direct Input"**.
3. Kopia tudu HTML di `cesaria.html` na area di testu.
4. Klika **Check**.
5. Es ta da-bu lista di erros, ku linha onde es ta sta.

Validador ka ta enspeta CSS ou JavaScript — só HTML.

## Googla di manera intelijenti

Mas di metadi di tempu di un dezenvolvedor é Google. É mistériu — ma **kumo** bu ta Googla ta faze diferensa.

### Mau query → mau resultadu

| Mau | Bon |
|---|---|
| "como sentrar coisa" | "CSS center element horizontally" |
| "html não funciona" | "HTML form action attribute missing" |
| "estilo errado" | "CSS specificity class vs id" |

**Regras:**

1. **Skrebe na inglés.** Dokumentasan di web é em inglés. Bu ta otxa 100× mas rezultadu.
2. **Komesa ku `CSS` ou `HTML`.** Ta filtra rezultadu pa web.
3. **Sé spesífiku sobre kuze ki bu kre faze**, ka kumo bu ta tenta-l.
4. **Adisiona "MDN" na fin** pa otxa MDN diretu: `CSS flexbox align-items MDN`.

### Onde klika

| Sitio | Konfiansa |
|---|---|
| **MDN Web Docs** (developer.mozilla.org) | **Altu** — manual ofisial. Sempri prefere. |
| **Stack Overflow** | Mediu — bon pa probléma spesífiku. Le resposta mas votadu, ka primeru. |
| **CSS-Tricks** | Bon — tutoriais sólidu, sobre tudu de Chris Coyier. |
| **W3Schools** | **Baxu** — frekuentement dezatualizadu. Evita ou kruza ku MDN. |

## Le MDN

Página tipiku di MDN ten es strutura:

1. **Diskrisan** — kuze ki é propriedadi/element.
2. **Try it** — demo interativu.
3. **Syntax** — sintaksi formal i valor posivel.
4. **Egzemplus** — kódiku kompletu.
5. **Browser compatibility** — tabela na fundu mostrando ki browser ta suportar (importanti pa projetu real).

Pa novatu, parti mas útil é **Try it** i **Egzemplus**. Sintaksi formal ta keda mas klaru ku tempu.

## Faze un bon pergunta (na fórum, Discord, Q&A di Skola)

Si bu prizizar di ajuda, pesoa ki bu ta perguntar ta presiza **kontekstu**. Sen kontekstu, resposta ka pode ben.

### Strutura di un bon pergunta

1. **Reprodusan minima** — un demo na **CodePen** ou **JSFiddle** ku **só problema**, sen restu di projetu.
2. **Kuze ki bu sperá** — "Bu sperá ki boton ta keda azul kuandu bu pasa rato".
3. **Kuze ki ta akontesi** — "Boton fika sen muda kor".
4. **Kuze ki bu dja tenta** — "Konfirma na DevTools ki `:hover` ta funsiona na otru elementu; selector é `.btn:hover { color: blue }`".

Es strutura ta resolvi 50% di problema só pa faze pergunta. Txeu bez, kuandu bu skrebe pergunta dretu, bu ta otxa resposta tu propi.

## Erus komun pa evita

- **Edisons na DevTools ka salva.** Se bu kre keda mudansa, kopia pa `style.css`. Si nu, ta pirdi ku rikarregamenti.
- **Konfiansa siga na W3Schools.** Ten muitu informasan dezatualizadu, prinsipalmenti pa Flexbox, Grid i propriedadis modernu. Kruza sempri ku MDN.
- **Perguntar sen reprodusan minima.** "Pamodi es ka ta funsiona?" — resposta probabal: silénsiu. Ku CodePen + ekspetativa + tentativa, resposta ta ben rápidu.
- **Inora avisu di DevTools.** Triángulu amarelu = mensajen di browser pa bu. Sempri kontrola-l antis di pirdi 30 minutu Googlando.
- **Ka kontrola Network panel.** Si `style.css` ta da 404, **nada di stilu** ka ta aplika. Es é primeru kuza pa kontrola.
