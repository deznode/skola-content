# Grid: pozisiona i alinha items

Ti gosi, kada grid item ta okupa un cell úniku — un linha + un koluna. Ma kuandu nu kre ki un item okupa **dos kolunas**, ou spani **tudu largura** di pajina (kumo un banner)? Pa kel-li, nu ta uza **line numbers**, **`span`** i un truk majiku ku **`-1`**.

## Line numbers — fundamentu

Pa un grid di 4 kolunas, ten **5 line numbers** vertikal:

```
| line 1 | line 2 | line 3 | line 4 | line 5 |
   col 1    col 2    col 3    col 4
```

Mêsmu pa linhas (horizontal). Pa un grid 4×3, ten 5 lines vertikal + 4 lines horizontal.

**Negativu konta di fin:** line `-1` é últimu, `-2` é penúltimu, etc. Es é parti **majika** — bu ka presiza sabe kuantu kolunas existi.

![Grid template: lines numeradu (1 ti N+1, -1 di fin), tracks, cells](../../infographics/11-css-grid-template.jpg)

## Pozisiona item ku line numbers

```css
.feature {
  grid-column: 1 / 3;      /* komesa na line 1, fin na line 3 → ta okupa col 1 i 2 */
  grid-row: 2 / 4;         /* komesa na line 2, fin na line 4 → ta okupa row 2 i 3 */
}
```

**Atensan — line di fin é eksklusivu.** `grid-column: 1 / 3` ta okupa kolunas 1 i 2, **ka 1, 2 i 3**. Studantes ta erra es kontaji konstantamenti.

## Span — pa "kuantu largu" só

Si bu ka kuida undi item ta komesa, só **kuantu largu el é**, uza `span`:

```css
.feature {
  grid-column: span 2;     /* spani 2 kolunas, undi kuze item ta otxa */
}

.tall {
  grid-row: span 3;        /* spani 3 linhas */
}
```

Mas kompinatu:

```css
.banner {
  grid-column: 1 / span 3;    /* komesa na col 1, spani 3 kolunas */
}
```

**Prefere `span` kuandu bu ka kuida pozisan eksatu.** Mas leji pa ler.

## Truk majiku — `1 / -1`

Kuandu un item ten ki **spani tudu largura** di grid:

```css
.hero {
  grid-column: 1 / -1;     /* komesa na primeru line, ta fin na **últimu** line */
}
```

**Pamodi es é majiku:** funsiona **independenti** di kuantas kolunas grid ten. Si bu muda di 3 koluna pa 5 koluna mas tarde, hero ta kontinua kompletu sen bu ten ki tocar el.

:::callout{type=tip}
**`grid-column: 1 / -1` é truk mas útil di tudu Grid.** Memoriza-l gosi. Bu ta uza-l pa banner, hero, footer, kualkér kuza ki "speni tudu largura".

**Evita** `grid-column: 1 / 6` pa un grid di 5 koluna — kuandu bu muda pa 6 koluna, layout ta kebra. Sempri `1 / -1`.
:::

## Alinhamentu dentru di cell — item-level

Kuandu un cell é **mas grandi ki kontéudu di item**, undi item ta poza dentru? CSS Grid ten kuatru propriedadi pa kel-li.

### No container (afeta tudu items)

| Propriedadi | Eixu | Valor |
|---|---|---|
| `justify-items` | Horizontal | `start`, `end`, `center`, `stretch` (padron) |
| `align-items` | Vertikal | `start`, `end`, `center`, `stretch` (padron) |

**Atensan — Grid uza `start`/`end`, ka `flex-start`/`flex-end`.** Mêsmu konfundi di Lisan 16, gosi bu sta skrebe-l verdadeiramenti.

### No item (sobrepasa container)

| Propriedadi | Eixu |
|---|---|
| `justify-self` | Horizontal |
| `align-self` | Vertikal |

### Shorthand — `place-items`

```css
.gallery {
  place-items: center center;    /* horizontal center, vertikal center */
}

/* ô forma mas kurtu kuandu dos ladu é igual: */
.gallery {
  place-items: center;           /* tudu dos eixu sentradu */
}
```

**`place-items: center`** é maneira mas rápida di sentra kontéudu di **kada** cell.

## Alinhamentu di grid inteiru — track-level

Kuandu **grid tudu** é mas pikinu ki container (egzemplu, ten 3 kolunas di 100px ma container ten 1000px), pa undi tracks ta poza?

```css
.gallery {
  justify-content: center;       /* sentra tudu grid horizontal */
  align-content: center;         /* sentra tudu grid vertikal */
}
```

Valor é igual ki Flexbox: `start`, `end`, `center`, `space-between`, `space-around`, `space-evenly`.

**Diferensa:** `justify-items` ta sentra **kada item dentru di kel cell**. `justify-content` ta sentra **tudu grid dentru di container**. Skopu diferenti.

### Pasu 3 — inspekciona line numbers ku DevTools

Abri DevTools. Na painel Elements, prokura `.gallery`. Klika batch "grid". Pajina ta resebi overlay ku **line numbers** vizível: 1, 2, 3, 4 horizontal i tanbé vertikal.

### Pasu 4 — eksperimenta `justify-items: center`

Adisiona temporalmenti:

```css
.gallery {
  /* ... */
  justify-items: center;
}
```

Salva. Kartas ta **enkolhe pa largura di kontéudu** i sentra horizontalmenti dentru di kel cells. Es é `justify-items` (item dentru di cell).

## Erus komun pa evita

- **`start`/`end` vs `flex-start`/`flex-end`** — pitfall di Lisan 16 ki ta repeti. Si alinhamentu di Grid ka ta funsiona, **primeru kuza ki bu kontrola é es**.
- **Off-by-one na line numbers.** `grid-column: 1 / 3` ta okupa kolunas 1 i 2, **ka 1 ti 3**. Line di fin é **eksklusivu**.
- **Skesi truk `1 / -1`.** Studantes ta skrebe `1 / 6` pa grid di 5 koluna i ta kebra kuandu adisiona 6ª koluna. **Sempri `1 / -1`** pa "tudu largura".
- **Konfundi `justify-items` ku `justify-content`.** `justify-items` = items dentru di kel cell. `justify-content` = tudu grid dentru di container. Si items ka ta moveu, bu probabal kre kel **otru**.
- **Spani mas tracks ki existi.** `grid-column: 1 / 10` na grid di 5 koluna ta kria kolunas implísitu i ta kebra layout. Uza `1 / -1` ou `span N` ku N ≤ kuantidadi di kolunas.
