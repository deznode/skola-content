# Testu i kor na CSS

Ti gosi, blog di Adilson ten un kor di fundu i un `<h1>` azul-mar. É un kumesu. Gosi nu ta torna-l bonitu di verdadi: font sériu, kor ku karater, spasing ki ta da konfortu pa ledjor di artigu sobre Cesária Évora.

## Propriedadis di testu

Es é "kavala di trabaju" di CSS — kuze ki bu ta uza tudu dia.

| Propriedadi | Kuze ki ta faze | Egzemplu di valor |
|---|---|---|
| `color` | Kor di testu | `#333`, `blue`, `rgb(50, 50, 50)` |
| `font-family` | Família di font | `sans-serif`, `Georgia, serif` |
| `font-size` | Tamanhu di font | `16px`, `1.2em`, `24px` |
| `font-weight` | Spesura di font | `bold`, `400`, `700` |
| `font-style` | Stilu di font | `italic`, `normal` |
| `text-transform` | Transforma maiúskula/minúskula | `uppercase`, `lowercase`, `capitalize` |
| `text-align` | Alinhamentu horizontal | `left`, `center`, `right`, `justify` |
| `line-height` | Altura di linha (spasing vertikal) | `1.5` (sen unidadi — múltiplu) |

**Sobre `line-height`:** uza sempri valor sen unidadi (`1.5`, `1.6`). Ka uza `px`. Múltiplu sen unidadi ta skala dretu ku `font-size`, pa kel-li é mas previzivel.

```css
p {
  color: #333;
  font-family: sans-serif;
  font-size: 16px;
  line-height: 1.5;
}
```

:::callout{type=tip}
**Pa testu kumpridu (paragrafu di artigu), uza `line-height` entri `1.4` i `1.7`.** Menus ki kel-li, linha ta keda apertadu i kansa ojo. Mas ki kel-li, linha ta sebra muitu i ledjor ta perdi fil. `1.5` é puntu di partida siguru pa kuazi tudu kazu.
:::

## Valor di kor

Ten kuatru manera prinsipal di skrebe un kor na CSS.

### Hex — `#RRGGBB`

Mas komun. Seis dígitu hexadesimal, em pares pa **vermedju (R)**, **verdi (G)**, **azul (B)**.

```css
color: #1098ad;        /* azul-mar — inspiradu na bandera di KV */
color: #f7f0e6;        /* areia klaru */
```

Hex tanbé ten **forma kurtu** kuandu dos dígitu di kada par é igual:

```css
color: #fff;     /* = #ffffff (branku) */
color: #000;     /* = #000000 (pretu) */
color: #f7f;     /* = #ff77ff */
```

### RGB — `rgb(R, G, B)`

Mêsmu kor di hex, ma na desimal (0–255 pa kada kanal).

```css
color: rgb(16, 152, 173);    /* mêsmu azul-mar di antes */
color: rgb(255, 255, 255);   /* branku */
```

### RGBA — `rgba(R, G, B, A)`

Igual ki RGB, ma ku un kuartu valor: **alpha** (0 = transparenti, 1 = sólidu). Útil pa kor semi-transparenti.

```css
background-color: rgba(16, 152, 173, 0.5);   /* azul-mar mei-transparenti */
```

### Nomi nomeadu — `red`, `blue`, `tomato`

CSS ten ~140 nomi pré-definidu. Bon pa demo rápidu, ma evita na projetu real (poku flexibilidadi).

```css
color: red;
color: tomato;
color: rebeccapurple;
```

### Truk di grey

Si R = G = B, kor ta keda grey. Tonalidadi ta depende di valor:

| Valor | Tonalidadi |
|---|---|
| `#fff` | Branku |
| `#f7f7f7` | Grey muitu klaru (fundu suavi) |
| `#777` | Grey médiu |
| `#444` | Grey sukuru |
| `#000` | Pretu |

```css
body { background-color: #f7f7f7; }
p     { color: #444; }
footer { color: #777; }
```

:::callout{type=tip}
VS Code ten **color picker** internu. Pasa rato sobri kualkér valor di kor i klika na kuadradu kolorida ki ta apareci. Bu pode skoji kor visualmenti i VS Code ta atualiza ficheru pa bu.
:::

## Background i border

### `background-color`

Ta pinta fundu di kualkér elementu.

```css
body              { background-color: #f7f7f7; }
.related-posts    { background-color: #fff; }
```

### `border`

Shorthand ku trez valor: **largura**, **stilu**, **kor**.

```css
border: 5px solid #1098ad;     /* 5px largura, sólidu, azul-mar */
border: 2px dotted #777;       /* 2px, ponteadu, grey */
border: 3px dashed red;        /* 3px, tracejadu, vermedju */
```

Stilus mas komun: `solid`, `dotted`, `dashed`, `double`, `none`.

Border pode tanbé ser **só na un lado**:

```css
border-top: 2px solid #1098ad;
border-bottom: 3px solid #1098ad;
border-left: 5px solid #777;
border-right: 1px dashed #ddd;
```

## Erus komun pa evita

- **Skesi `;` na fin di un deklarasan.** Próximu deklarasan ta keda silensiozamenti inoradu. Hábitu: skrebe sempri `;`.
- **Erru na nomi di propriedadi** (`colur` na bez di `color`, `font-sze` na bez di `font-size`). Browser ta inora linha. DevTools ta mostra un ikon di avisu na linha — nu ta odja kel-li na Lisan 13.
- **Misturá `px` i `em` sen kuidadu.** Pa gosi, fika ku `px` pa font i `1.5` (sen unidadi) pa `line-height`. Unidadi rilativu (`em`, `rem`, `%`) ta ben na un kursu di responsive design.
- **Uza kor nomeadu (`red`, `blue`) na projetu real.** Pinta-l so na demo. Pa produsan, sempri hex ou RGB.
- **Skesi alpha na `rgba`** kuandu bu kre transparénsia. `rgb(...)` é sólidu sempri.
