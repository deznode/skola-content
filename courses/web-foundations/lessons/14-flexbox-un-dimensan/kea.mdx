# Flexbox: un dimensan

Ti gosi, nu ka kontrola undi elementus ta sta na pajina ŌĆö es ta enpilha vertikal pamodi ├® padron di elementu bloku. **Flexbox** ├® forma modernu di nu organiza-l. Antes di Flexbox, programadores ta uza **floats** (durante muitu anu) ŌĆö el inda funsiona ma ├® legasi, i nu ta inora-l na kursu li.

## Sobre floats ŌĆö s├│ nota di literasia

Antigamenti, layout na CSS ├® fetu ku `float: left` i `float: right`. El ta funsiona, ma ├® trapadju, ten muitu bug, i k├│diku fika difisil di manten. Bu ta odja-l na websites bedju, ma pa k├│diku novu, sempri uza Flexbox ou CSS Grid.

## Primeru: koreta box model ku `box-sizing: border-box`

Lembra di Lisan 12 ŌĆö `width: 300px` + `padding: 40px` = 380px na pajina. Es ├® defualt **konfundenti**.

Solusan ├® un linha:

```css
* {
  box-sizing: border-box;
}
```

`box-sizing: border-box` ta faze `width` signifika **largura final** di elementu. Padding i border ta **subtrai** di kont├®udu, ka adisiona. Si bu defini `width: 300px` + `padding: 40px`, kaixa **ta okupa 300px**, ku padding **dentru** di kel 300px.

**Aplika es reset un bez na topu di kada folha di stilu ŌĆö i skesi.** Nu dja preve es regra na Lisan 11, gosi nu ta intende kuze ki el ta faze.

```css
/* style.css ŌĆö topu di ficheru */
* {
  box-sizing: border-box;
}

body {
  /* ... */
}
```

## Flexbox ŌĆö kumo el ta funsiona

### Aktiva ku `display: flex`

Aplika `display: flex` na un container. Tudu fidju **diretu** ta torna **flex items** automatikamenti.

```html
<div class="row">
  <p>Item 1</p>
  <p>Item 2</p>
  <p>Item 3</p>
</div>
```

```css
.row {
  display: flex;
}
```

Resultadu: trez paragrafu ta poza **un lado di otru** (horizontal), nau enpilhadu vertikal.

### Dos eixu ŌĆö main axis i cross axis

Kada flex container ten **dos eixu**:

| Eixu | Direksan padron |
|---|---|
| **Main axis** | Horizontal ŌĆö skerda ŌåÆ direita |
| **Cross axis** | Vertikal ŌĆö topu ŌåÆ fundu |

![Flexbox: main axis (horizontal) i cross axis (vertikal), ku justify-content i align-items](../../infographics/10-flexbox-axes.jpg)

Tudu propriedadi di Flexbox ta opera **na un di es dos eixu**.

## Propriedadis di container

### `justify-content` ŌĆö distribui items na main axis

| Valor | Lojika |
|---|---|
| `flex-start` | Items na komesu (padron) |
| `center` | Items sentradu |
| `flex-end` | Items na fin |
| `space-between` | Spasu entri items, sen spasu nas pontas |
| `space-around` | Spasu igual na roda di kada item |
| `space-evenly` | Spasu kompletamenti igual entri tudu items |

### `align-items` ŌĆö alinha items na cross axis

| Valor | Lojika |
|---|---|
| `stretch` | Items ta stika pa nxe altura (padron) |
| `center` | Items sentradu vertikalmenti |
| `flex-start` | Items na topu |
| `flex-end` | Items na fundu |

:::callout{type=tip}
**Sentra vertikal** ŌĆö antes di Flexbox, ├® un di problema mas frustranti di CSS. Gosi ├® un linha: `align-items: center`. Lembra: `justify-content` ├® horizontal (main), `align-items` ├® vertikal (cross). Si bu konfundi-l, troka es dos.
:::

### `gap` ŌĆö spasu entri items

`gap` ta da spasu **entri** items sen txuga padding ou margin individual.

```css
.row {
  display: flex;
  gap: 24px;
}
```

Mas limpu ki `margin-right` na kada item (ka ten trailing margin no ├║ltimu).

## Propriedadis di item

### `align-self` ŌĆö sobrepasa `align-items` pa un item

```css
.special-item {
  align-self: flex-end;     /* es item s├│ ta keda na fundu */
}
```

### `order` ŌĆö reorganiza vizualmenti

```css
.first-visible {
  order: -1;                /* ta moveu pa komesu, sen muda HTML */
}
```

`order` negativu ta mov├® pa komesu; pozitivu pa fin. Padron ├® `0`.

## Pr├Ītika 1 ŌĆö Flex header na `cesaria.html`

Abri `cesaria.html`. Adisiona class `main-header` na `<header>` di topu:

```html
<header id="main-header" class="main-header">
  <h1>Blog di Adilson</h1>
  <nav>
    <a href="index.html">In├Łsiu</a>
    <a href="blog.html">Blog</a>
    <a href="sobre.html">Sobre</a>
  </nav>
</header>
```

Na `style.css`, adisiona reset universal (na topu, antes di tudu) i regras pa header:

```css
* {
  box-sizing: border-box;
}

.main-header {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 20px 40px;
  background-color: #fff;
  border-bottom: 2px solid #1098ad;
}

.main-header nav {
  display: flex;
  gap: 24px;
}
```

Salva. Bu ta odja titulu na skerda, nav na direita, **alinhadu vertikalmenti**. Trez link di nav ku spasu igual entri kada un.

## Pr├Ītika 2 ŌĆö byline ku foto

Refaktoriza byline di artigu pa adisiona foto di autora:

```html
<header>
  <h1>Ces├Īria ├ēvora ŌĆö Reina di Morna</h1>
  <div class="author">
    <img
      class="author-photo"
      src="https://placehold.co/60"
      alt="Foto di Djamila Tavares"
      width="60"
      height="60"
    />
    <p class="author-name">Djamila Tavares</p>
  </div>
</header>
```

Na `style.css`:

```css
.author {
  display: flex;
  align-items: center;
  gap: 16px;
}

.author-photo {
  border-radius: 50%;       /* rondu */
}
```

Salva. Foto ta keda redondu, lado-a-lado ku nomi, **sentradu vertikalmenti**. 16px di spasu entri foto i nomi.

:::callout{type=tip}
**Flexbox pa byline ku foto + nomi ├® padron na web.** Kada blog di profisional ta uza es eksatu pattern: `display: flex; align-items: center; gap: 16px;`. Memoriza-l.
:::

## Erus komun pa evita

- **Skesi reset di box-sizing.** Sen `* { box-sizing: border-box; }`, kada `padding` ta silensiozamenti kebra largura. **Adisiona na kada projetu, kada bez.**
- **Troka `justify-content` ku `align-items`.** `justify-content` ├® **main axis** (horizontal padron). `align-items` ├® **cross axis** (vertikal padron). Si bu konfundi, sentra horizontal i vertikal ta troka.
- **`align-items: center` ka sentra nada viz├Łvel** kuandu container ten altura `auto`. Items ta determina altura, i ka ten nada pa sentra. Defini un altura na container ou aseita ki items ├® altura.
- **Uza `text-align: center` pa sentra un elementu inline** ŌĆö s├│ sentra **testu dentru**. Pa sentra elementu, uza Flexbox.
- **Tenta Flexbox kuandu un layout dos dimensan** (grid di kartas) ŌĆö vai presiza ti aninhar Flexbox dentru di Flexbox. Es ├® sinal di ki bu kre **CSS Grid** (Lisan 16).
