## Risumu di lisaun

Python é un **linguagen di programasun** ki foi kriadu na 1991 pa Guido van Rossum. Nomi ben di grupu *Monty Python* — komunidadi di Python ta gosta di bon humor.

### Pamodi Python é spesial

- **Fásil di lé i skrebi.** Sintaxi ta paresi ku inglês simples — sem pontu-i-vírgula, sem símbulus komplikadu.
- **Versátil.** Un só linguagen pa websites, dadus, AI, automatizasun, jogus.
- **Komunidadi enormi.** Milhoens di programadoris na mundu ta partilha bibliotecas grátis.

### Undi Python ta uza

- **Web** — Instagram, Spotify, Netflix tudu ta uza Python.
- **Dadus** — Pandas i NumPy ta fazi análizi di dadus fásil.
- **AI** — ChatGPT, TensorFlow i PyTorch tudu é Python.
- **Automatizasun** — renomia ficheirus, manda emails, extrai dadus di websites.

### Pamodi e ta inportanti pa karera na Kabu Verdi

Empresas internasional manera Tek Experts i Sonatel ta kontrata programadoris Python remotu — bo pode trabaja di Praia, Mindelu o Sal pa kliente na Lisboa, Paris o New York. Salário ta kumesa na 1500€/mês pa juniors i ta sobi rapidamenti.

### Próximu pasu

Na pursimu lisaun nôs ta instala Python i konfigura VS Code. Ten bo komputador prontu pa nôs kumesa skrebe kódigu di verdadi.
