## Risumu di lisan

Python é un **linguagen di programasan** ki foi kriadu na 1991 pa Guido van Rossum. Nomi ben di grupu *Monty Python* — komunidadi di Python ta gosta di bon humor.

### Pamodi Python é spesial

- **Faxi di ler i skrebi.** Sintaxi ta paresi ku inglês, simples — sem pontu-i-vírgula, sem símbulus komplikadu.
- **Versátil.** Un só linguagen pa websites, dadus, AI, automatizasan, jogus.
- **Komunidadi enormi.** Milhoens di programadoris na mundu ta partilha bibliotecas.

### Undi Python e uzado

- **Web** — Instagram, Spotify, Netflix tudu ta uza Python.
- **Dadus** — Pandas i NumPy ta fazi análizi di dadus.
- **AI** — ChatGPT, TensorFlow i PyTorch tudu é Python.
- **Automatizasan** — renomia ficheirus, manda emails, extrai dadus di websites.

### Pamodi Python na nos kontestu

Python ta djuda automatiza tarefas di dia-a-dia: kria relatóriu pa negósiu, fazi análizi di dadus pa projetu lokal, ou skrebi skripts ki ta poupa óras di trabaju.
