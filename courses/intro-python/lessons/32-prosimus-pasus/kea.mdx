## `@dataclass` — Klasis Mais Simples

Kuandu bo kria un klasi ki prinsipal trabadju é garda dadus (kuma Estudanti, Prodotu, Kontatu), bo ta skrebi sempri mesmu kódiku repetitivu: `__init__`, `__repr__`, `__eq__`...

Python tene un atalhu: `@dataclass` ta jera tudu es métodus automátikamenti!

### Klasi Normal vs @dataclass

**Sem dataclass (manéra tradisionál):**

```python
class Estudanti:
    def __init__(self, nomi, idadi, kursu):
        self.nomi = nomi
        self.idadi = idadi
        self.kursu = kursu

    def __repr__(self):
        return f"Estudanti(nomi='{self.nomi}', idadi={self.idadi}, kursu='{self.kursu}')"

    def __eq__(self, otru):
        return (self.nomi == otru.nomi and
                self.idadi == otru.idadi and
                self.kursu == otru.kursu)
```

**Ku dataclass (mesmu rezultadu, menus kódiku):**

```python
from dataclasses import dataclass

@dataclass
class Estudanti:
    nomi: str
    idadi: int
    kursu: str
```

**É só es!** Trés linhas na lugar di 12. `@dataclass` ta jera `__init__`, `__repr__`, i `__eq__` automátikamenti:

```python
# Kria objetu — __init__ jeradu automátikamenti
maria = Estudanti("Maria", 22, "Python")
joao = Estudanti("João", 25, "Data Science")

# __repr__ jeradu automátikamenti
print(maria)  # Estudanti(nomi='Maria', idadi=22, kursu='Python')

# __eq__ jeradu automátikamenti
maria2 = Estudanti("Maria", 22, "Python")
print(maria == maria2)  # True — kompara por valór!
```

:::callout{type=tip}
**Dika:** `@dataclass` é perfetu pa klasis ki prinsipal objetivu é "garda dadus". Si bo klasi tene mutu lójika kompleksu, klasi tradisionál pode ser melhor.
:::

## Opsoens di @dataclass

### Valoris Padrón

```python
from dataclasses import dataclass

@dataclass
class Prodotu:
    nomi: str
    presu: float
    kuantidadi: int = 1         # Valór padrón
    disponível: bool = True     # Valór padrón

# Pode omiti parâmetrus ku valór padrón
katchupa = Prodotu("Katchupa", 350)
print(katchupa)
# Prodotu(nomi='Katchupa', presu=350, kuantidadi=1, disponível=True)

grogue = Prodotu("Grogue di Santo Antão", 800, kuantidadi=6)
print(grogue)
# Prodotu(nomi='Grogue di Santo Antão', presu=800, kuantidadi=6, disponível=True)
```

### frozen — Objetu ki Ka Pode Muda

```python
from dataclasses import dataclass

@dataclass(frozen=True)
class Koordenada:
    latitude: float
    longitude: float

# Koordenadas di Praia
praia = Koordenada(14.9195, -23.5087)
print(praia)  # Koordenada(latitude=14.9195, longitude=-23.5087)

# Ka pode muda!
try:
    praia.latitude = 0.0
except AttributeError as e:
    print(f"Erru: {e}")  # Erru: cannot assign to field 'latitude'
```

### Adisiona Bo Propriu Métodus

`@dataclass` ka ta impedi-bu di adisiona métodus personalizadu:

```python
from dataclasses import dataclass

@dataclass
class Ilha:
    nomi: str
    populasun: int
    area_km2: float

    def densidadi(self):
        """Pesoas por km²"""
        return round(self.populasun / self.area_km2, 1)

    def __str__(self):
        """Override __str__ jeradu pa versun más bonitu"""
        return f"🏝️ {self.nomi} — {self.populasun:,} abitantis"

santiago = Ilha("Santiago", 313_912, 991)
sal = Ilha("Sal", 42_827, 216)

print(santiago)                # 🏝️ Santiago — 313,912 abitantis
print(f"Densidadi: {santiago.densidadi()} pesoas/km²")  # Densidadi: 316.8 pesoas/km²
print(repr(santiago))          # Ilha(nomi='Santiago', populasun=313912, area_km2=991)
```

Otru opsoens útil: `order=True` (permite `sorted()` automátiku) i `field(default_factory=list)` pa valoris padrón mutável.

## Projetu Final Sujeridu: Jestaun di Kontatu

Pa prátika tudu ki bo prende na es kursu, kria un **Jestor di Kontatu** na terminál. Es projetu ta uza:

- **Klasis i dataclasses** — `Kontatu`, `JestorKontatu`
- **Listas i disionárius** — garda kontatus
- **Funsoens** — adisiona, buska, elimina
- **Fixerus** — salva i karga di JSON
- **Tratamentu di erru** — try/except pa entradas inválidu
- **Métodus mádjiku** — `__str__`, `__repr__`, `__len__`

```python
from dataclasses import dataclass, field
import json

@dataclass
class Kontatu:
    nomi: str
    telefoni: str
    email: str = ""
    sidadi: str = "Praia"

    def __str__(self):
        return f"📇 {self.nomi} — {self.telefoni} ({self.sidadi})"

@dataclass
class JestorKontatu:
    kontatus: list[Kontatu] = field(default_factory=list)

    def adisiona(self, kontatu: Kontatu):
        self.kontatus.append(kontatu)
        print(f"✅ {kontatu.nomi} adisionadu!")

    def buska(self, nomi: str) -> list[Kontatu]:
        return [k for k in self.kontatus if nomi.lower() in k.nomi.lower()]

    def __len__(self):
        return len(self.kontatus)

    def __contains__(self, nomi: str):
        return any(nomi.lower() in k.nomi.lower() for k in self.kontatus)

# Ezemplu di uzu
jestor = JestorKontatu()
jestor.adisiona(Kontatu("Maria Santos", "991-2345", "maria@email.com", "Mindelo"))
jestor.adisiona(Kontatu("João Évora", "995-6789", "joao@email.com"))
jestor.adisiona(Kontatu("Ana Delgado", "993-1122", sidadi="Sal"))

print(f"\nTotal di kontatus: {len(jestor)}")
print("Maria" in jestor)   # True

for k in jestor.buska("maria"):
    print(k)  # 📇 Maria Santos — 991-2345 (Mindelo)
```

Es é un ponto di partida — bo pode kontinua adisionando funsoens kuma salva na fixeru JSON, menu interativu, edita kontatu, i mais!
