# Variáveis i Tipu Dinámiku

Imagina ki bu tene un kaxa. Bu pode po kualker kuza dentu — un númeru, un nomi, un presu. Kel kaxa é un **variável** (variable) na Python. Bu ta da un nomi pa kaxa, i Python ta garda valor dentu.

Nes lisan, nu ta prende kumo kria variáveis, ki regras di nomi tene, kumo resebe dados di utilizador, i kumo konverti entri tipus diferenti.

## Kria Variáveis

Na Python, bu ta kria un variável sinplismenti: bu ta atribui un valor ku `=`. Ka presiza deklara tipu — Python ta diskubri tipu automatikamenti.

```python
# Kria variáveis
nomi = "Amilcar"       # str (textu)
idadi = 25             # int (númeru interu)
altura = 1.75          # float (númeru desimal)
estudanti = True       # bool (verdaderu ó falsu)
```

Bu pode tanbe atribui vários valores na mesmu linha:

```python
# Atribuisan múltiplu
nomi, idadi, ilha = "Carla", 22, "Santiago"
print(nomi)   # Carla
print(idadi)  # 22
print(ilha)   # Santiago

# Mesmu valor pa vários variáveis
x = y = z = 0
print(x, y, z)  # 0 0 0
```

Variáveis pode muda di valor na kualker momentu:

```python
presu = 500
print(presu)   # 500

presu = 750    # Muda valor
print(presu)   # 750

presu = presu + 100  # Uza valor antigu pa kalkula novu
print(presu)   # 850
```

## Regras di Nomi

Python ten regras ki bu ten ki sigi pa nomis di variáveis:

### Regras Obrigatóriu

```python
# KORETU — komesa ku letra ó underscore
nomi = "Gil"
_idadi = 30
nomi_kompletu = "Gil Santos"
nota1 = 85

# ERADU — ka pode komesa ku númeru
# 2nomi = "Gil"       # SyntaxError!

# ERADU — ka pode uza hífens ó spasus
# nomi-kompletu = "Gil"    # SyntaxError!
# nomi kompletu = "Gil"    # SyntaxError!

# ERADU — ka pode uza palavras reservadas di Python
# if = 10        # SyntaxError!
# print = "ola"  # Funsiona ma ta kebra print()!
```

### Konvensons (PEP 8)

```python
# BON — snake_case pa variáveis i funsons
nomi_kompletu = "Edna Lima"
nota_final = 88.5
total_di_estudantis = 45

# EVITA — CamelCase é pa klasis (nu ta prende dipos)
# NomiKompletu = "Edna Lima"   # Reservadu pa klasis

# KONSTANTI — MAIÚSKULU ku underscore
TAXA_IVA = 0.15
MAX_ESTUDANTIS = 30
MOEDA = "ECV"
```

Nomi di variável ten ki ser **deskritivu**:

```python
# MALU — ki signifika x, n, t?
x = 350
n = "Katxupa"
t = x * 2

# SABI — klaru i fáxi di intendi
presu_katxupa = 350
nomi_pratu = "Katxupa"
total = presu_katxupa * 2
```

:::callout{type=tip}
Bo futuru ta agrádese kuandu bu uza nomis klaru. Imagina ler bo kódiku dipos di 6 mes — `presu_katxupa` é mutu midjór ki `x`!
:::

## Verifika Tipu ku `type()`

Funsan `type()` ta mostra tipu di kualker valor ó variável:

```python
nomi = "Lara"
idadi = 28
altura = 1.68
ativa = True

print(type(nomi))    # <class 'str'>
print(type(idadi))   # <class 'int'>
print(type(altura))  # <class 'float'>
print(type(ativa))   # <class 'bool'>
```

`type()` é útil pa debugging — kuandu algu ka ta funsiona, verifika si tipu é kel ki bu spera:

```python
valor = "25"
print(type(valor))  # <class 'str'> — é string, ka int!

# Kel splika pamodi kel ka funsiona:
# resultado = valor + 10  # TypeError!
```

Bu tanbe pode uza `type()` diretamenti ku valores:

```python
print(type(42))       # <class 'int'>
print(type(3.14))     # <class 'float'>
print(type("Praia"))  # <class 'str'>
print(type(True))     # <class 'bool'>
```

## Input di Utilizador ku `input()`

:::callout{type=tip}
Ka preokupa si bu ka kompriende tudu kódiku gosi. Es sesan-li ta uza `f"..."` (f-strings) pa mostra rezultadu — bu ta prende-l formalmenti na Lisan 7. Repara na `input()` i na konversan di tipu, ka na f-string sintaxi.
:::

Funsan `input()` ta para programa i spera utilizador skrebe algu. El **senpri ta retorna un string**.

```python
nomi = input("Kual é bo nomi? ")
print(f"Ola, {nomi}! Ben-vindu!")
```

Kuandu bu roda:
```
Kual é bo nomi? Maria
Ola, Maria! Ben-vindu!
```

:::callout{type=warning}
`input()` senpri ta retorna **string**, mesmu si utilizador skrebe un númeru:

```python
idadi = input("Kual é bo idadi? ")
print(type(idadi))  # <class 'str'> — É string!

# Kel ka ta funsiona kumo bu spera:
# resultado = idadi + 5  # TypeError: can only concatenate str to str
```
:::

Pa traba ku númerus, bu ten ki **konverti**:

```python
idadi = int(input("Kual é bo idadi? "))
print(type(idadi))  # <class 'int'> — Gosi é int!
print(f"Na 10 anus, bu ta tene {idadi + 10} anus")
```

:::callout{type=tip}
Senpri lenbra — `input()` ta retorna string. Si bu kre númeru, uza `int()` ó `float()` pa konverti.
:::

## Konversan di Tipu (Type Casting)

Python tene funsons pa konverti entri tipus:

| Funsan | Faze kuze | Izemplu |
|--------|-----------|---------|
| `int()` | Konverti pa interu | `int("25")` → `25` |
| `float()` | Konverti pa desimal | `float("3.14")` → `3.14` |
| `str()` | Konverti pa textu | `str(25)` → `"25"` |
| `bool()` | Konverti pa boolean | `bool(1)` → `True` |

```python
# String pa int
idadi_str = "25"
idadi_int = int(idadi_str)
print(idadi_int + 5)  # 30

# String pa float
presu_str = "149.99"
presu_float = float(presu_str)
print(presu_float * 2)  # 299.98

# Int pa string
populasan = 600000
mensajen = "Kabu Verdi tene " + str(populasan) + " pesoas"
print(mensajen)

# Melhor ku f-string (ka presiza konverti)
print(f"Kabu Verdi tene {populasan} pesoas")
```

### Kuidadu ku Konversan!

Nem tudu konversan é posível:

```python
# Kel ta funsiona
print(int("42"))      # 42
print(float("3.14"))  # 3.14

# Kel KA ta funsiona
# int("Maria")     # ValueError: invalid literal for int()
# int("3.14")      # ValueError: ka pode konverti float-string pa int diretamenti
# float("ola")     # ValueError: could not convert string to float

# Pa konverti "3.14" pa int, fazi en 2 pasus:
resultado = int(float("3.14"))  # Primeiru float, dipos int
print(resultado)  # 3
```

### Bool Konversan

Python tene regras simples pa konverti pa `bool`. Programadoris ta xoma valores ki ta vira `False` di **"falsy"** (vazi, zeru, nada), i valores ki ta vira `True` di **"truthy"** (ten algun kuza dentu).

```python
# Valores "falsy" (vira False)
print(bool(0))       # False
print(bool(0.0))     # False
print(bool(""))      # False (string vazio)
print(bool(None))    # False

# Tudu otu kuza é "truthy" (vira True)
print(bool(1))       # True
print(bool(-5))      # True
print(bool("Ola"))   # True
print(bool(0.001))   # True
```

## Izemplu Prátiku: Kalkuladora Simples

Nu ta kombina tudu ki nos prende — `input()`, konversan di tipu, i f-strings:

```python
# Kalkuladora simples
print("=== Kalkuladora di Skola ===")
print()

# Resebe númerus di utilizador
num1 = float(input("Primeiro númeru: "))
num2 = float(input("Segundu númeru: "))

# Kalkula
soma = num1 + num2
diferensa = num1 - num2
produtu = num1 * num2
divisan = num1 / num2

# Mostra resultadus
print()
print(f"{num1} + {num2} = {soma}")
print(f"{num1} - {num2} = {diferensa}")
print(f"{num1} * {num2} = {produtu}")
print(f"{num1} / {num2} = {divisan}")
```

Kuandu bu roda:
```
=== Kalkuladora di Skola ===

Primeiro númeru: 150
Segundu númeru: 45

150.0 + 45.0 = 195.0
150.0 - 45.0 = 105.0
150.0 * 45.0 = 6750.0
150.0 / 45.0 = 3.3333333333333335
```

## Izemplu Prátiku: Perfil di Estudanti

```python
# Programa ki kria un perfil simples
print("=== Kria Bo Perfil ===")
print()

nomi = input("Nomi: ")
ilha = input("Ilha: ")

nota_media = float(input("Nota média (0-100): "))

print()
print("--- Bo Perfil ---")
print(f"Nomi: {nomi}")
print(f"Ilha: {ilha}")
print(f"Idadi: {idadi} anus")

```

Kuandu bu roda:
```
=== Kria Bo Perfil ===

Nomi: Nilton
Ilha: São Vicente
Idadi: 19
Nota média: 82.5

--- Bo Perfil ---
Nomi: Nilton
Ilha: São Vicente
Idadi: 19 anus
Nota média: 82.5
Tipu di nomi: str
Tipu di idadi: int
Tipu di nota: float
```
