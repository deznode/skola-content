# Operasoens ku Ficheru

Programas di mundu real precisa guarda dadus — nota di estudantis, lista di klientes, konfigurasun di aplikasun. Tudu es ta vivi na **ficheru** na disku. Na es lisun, nôs ta prende kumo lé, skrebi, i adisiona konteúdu na fixerus uzandu Python.

Imajina ki bo ta kria un programa pa registra vendas na merkadu di Sukupira na Praia. Kada venda precisa fikа gravadu nun ficheru pa ka perdi kuandu programa fexá!

## Modus di Ficheru

![Modus di ficheru: r, w, a, r+, w+, rb, wb](../../infographics/12-modus-di-ficheru.png)

Antis di abri un ficheru, bo precisa dizi a Python **o ki bo kre fazi** ku el. Es é modus prinsipais:

| Modu | Signifikadu | Konportamentu |
|------|-------------|---------------|
| `'r'` | Read (lé) | Só lé. Ficheru ten ki ezisti |
| `'w'` | Write (skrebi) | Skrebi. **APAGA tudu ki ta existia!** |
| `'a'` | Append (adisiona) | Adisiona na final. Ka apaga nada |
| `'w+'` | Write + Read | Skrebi + lé. Apaga i kria novu |

:::callout{type=tip}
**Dika:** Modu padraun é `'r'`. Si bo ka espesifika nada, Python ta abri pa leitura.
:::

## Abri i Lé Ficheru

### Lé Tudu di un Vez

```python
# Lé ficheru interu pa un string
with open("notas.txt", "r") as ficheru:
    konteúdu = ficheru.read()
    print(konteúdu)
```

### Lé Linha por Linha

Pa ficheru grandi, lé linha por linha é más efisienti — ka ta karrega tudu pa memória:

```python
# Lé linha por linha (melhor pa ficheru grandi)
with open("estudantis.txt", "r") as ficheru:
    for linha in ficheru:
        print(linha.strip())  # strip() ta tira \n di final
```

### Lé Tudu Linhas pa un Lista

```python
# Kada linha ta vira un elementu di lista
with open("ilhas.txt", "r") as ficheru:
    linhas = ficheru.readlines()
    print(linhas)  # ['Santiago\n', 'São Vicente\n', 'Sal\n']
```

:::callout{type=tip}
**Dika:** `readlines()` ta mantene `\n` na final di kada linha. Uza `linha.strip()` pa tira.
:::

## O ki É `with` Statement?

`with` é un **context manager** — e ta garanti ki ficheru ta fexá automátikamenti, mesmu si erru akonteси. É manera seguru di trabadja ku ficheru.

**Sem `with` (modu antigu — ka rekomendadu):**

```python
# PERIGO: si erru akonteси antis di .close(), ficheru fika abertu!
ficheru = open("dadus.txt", "r")
konteúdu = ficheru.read()
ficheru.close()  # Si skeси es linha, ficheru fika abertu na memória
```

**Ku `with` (sempri uza es!):**

```python
# SEGURU: ficheru ta fexá automátikamenti kuandu bloku akaba
with open("dadus.txt", "r") as ficheru:
    konteúdu = ficheru.read()
# Aki ficheru dja fexá — mesmu si tinha erru!
```

Pensa na `with` manera un guarda ki ta abri porta, dixa bo entra, i ta fexá porta kuandu bo sai — mesmu si bo kai la dentru!

## Skrebi na Ficheru

### Modu `'w'` — Kuidadu! Ta Apaga Tudu!

```python
# ATENSAN: 'w' ta apaga tudu ki existia na ficheru!
with open("lista_kompras.txt", "w") as ficheru:
    ficheru.write("Katchupa ingredientis:\n")
    ficheru.write("- Milhu\n")
    ficheru.write("- Fejun\n")
    ficheru.write("- Banana verdi\n")
```

```python
# Si bo roda di novu ku 'w', konteúdu anteriu ta PERDI!
with open("lista_kompras.txt", "w") as ficheru:
    ficheru.write("Lista novu — tudu anteriu ta bai!\n")
```

:::callout{type=tip}
**Dika:** Modu `'w'` é manera "apagador" — e ta limpa tudu i kumesa di novu. Si bo kre mantene o ki dja stá, uza `'a'`!
:::

### Modu `'a'` — Adisiona Sen Apaga

```python
# 'a' ta adisiona na final sem apaga nada ki existia
with open("vendas.txt", "a") as ficheru:
    ficheru.write("2024-01-15: Grogue - 500 ECV\n")
    ficheru.write("2024-01-15: Ponche - 300 ECV\n")

# Roda di novu — ta adisiona más, ka ta apaga!
with open("vendas.txt", "a") as ficheru:
    ficheru.write("2024-01-16: Katchupa - 250 ECV\n")
```

### Skrebi Múltiplu Linhas di un Vez

```python
# writelines() ta skrebi un lista di strings
ilhas = ["Santiago\n", "São Vicente\n", "Sal\n", "Santo Antão\n", "Fogo\n"]

with open("ilhas_kv.txt", "w") as ficheru:
    ficheru.writelines(ilhas)
```

:::callout{type=tip}
**Dika:** `writelines()` ka ta adisiona `\n` automátikamenti — bo precisa inkluí na kada string!
:::

## Modu `'w+'` — Skrebi i Lé (ku seek)

`'w+'` dixa bo skrebi i dipos lé mesmu ficheru. Ma tene un armadilha: dipos di skrebi, **kursor ta na final** di ficheru. Bo precisa uza `seek(0)` pa volta pa kumesу!

```python
with open("notas_maria.txt", "w+") as ficheru:
    # Skrebi
    ficheru.write("Matimátika: 85\n")
    ficheru.write("Português: 92\n")
    ficheru.write("Istória: 78\n")

    # Tenta lé sem seek — ka ta sai nada!
    konteúdu = ficheru.read()
    print(f"Sem seek: '{konteúdu}'")  # Sem seek: ''

    # Volta pa kumesу i lé
    ficheru.seek(0)
    konteúdu = ficheru.read()
    print(f"Ku seek: '{konteúdu}'")  # Mostra tudu!
```

## Kaminhu di Fixeru ku `os.path`

Kuandu programa ta roda na diferenti sistéma (Windows, macOS, Linux), kaminhus di ficheru é diferenti:
- Windows: `C:\Users\Maria\documentus\notas.txt`
- macOS/Linux: `/home/maria/documentus/notas.txt`

Módulu `os.path` ta rezolve es problema:

```python
import os

# Kria kaminhu di forma seguru (funxiona na tudu sistéma)
kaminhu = os.path.join("documentus", "notas", "maria.txt")
print(kaminhu)
# Windows: documentus\notas\maria.txt
# macOS/Linux: documentus/notas/maria.txt

# Verifika si ficheru ezisti
si_ezisti = os.path.exists("vendas.txt")
print(f"Ficheru ezisti? {si_ezisti}")

# Verifika si é ficheru o pasta
print(os.path.isfile("vendas.txt"))    # True si é ficheru
print(os.path.isdir("documentus"))     # True si é pasta

# Kaminhu absolutu
print(os.path.abspath("vendas.txt"))
# /home/maria/projetu/vendas.txt

# Lista ficheru na un pasta
fixerus = os.listdir(".")
print(fixerus)  # ['vendas.txt', 'notas.txt', 'ilhas_kv.txt']
```

## Ezemplu Prátiku: Rejistu di Vendas

Vamus kria un programa konpletu ki ta registra vendas na un ficheru:

```python
import os

FIXERU_VENDAS = "vendas_sukupira.txt"

def registra_venda(prudotu, presu, klientе):
    """Adisiona un venda na ficheru"""
    with open(FIXERU_VENDAS, "a") as ficheru:
        ficheru.write(f"{prudotu} | {presu} ECV | Klienti: {klientе}\n")
    print(f"Venda registradu: {prudotu} pa {klientе}")

def mostra_vendas():
    """Mostra tudu vendas gravadu"""
    if not os.path.exists(FIXERU_VENDAS):
        print("Nenhun venda registradu inda.")
        return

    print("\n=== VENDAS DI SUKUPIRA ===")
    with open(FIXERU_VENDAS, "r") as ficheru:
        for linha in ficheru:
            print(f"  {linha.strip()}")
    print("========================\n")

def konta_vendas():
    """Konta kuantus vendas foi feitu"""
    if not os.path.exists(FIXERU_VENDAS):
        return 0

    with open(FIXERU_VENDAS, "r") as ficheru:
        linhas = ficheru.readlines()
    return len(linhas)

# Registra algunhas vendas
registra_venda("Grogue", 500, "Nilton")
registra_venda("Katchupa", 250, "Ana")
registra_venda("Ponche di koku", 300, "Pedro")

# Mostra tudu
mostra_vendas()
print(f"Total di vendas: {konta_vendas()}")
```

## Ezemplu: Kopia Ficheru ku Múltiplu Kontekstus

Bo pode abri dôs ficheru na mesmu `with`:

```python
# Kopia konteúdu di un ficheru pa otru
with open("orijinal.txt", "r") as fonte, open("kopia.txt", "w") as destinu:
    for linha in fonte:
        destinu.write(linha.upper())  # Kopia ku letra maiúskula

print("Kopia feitu!")
```

## Errus Kumun ku Ficheru

### 1. FileNotFoundError — Ficheru Ka Ezisti

```python
# Erru: ficheru ka ezisti
with open("ficheru_ki_ka_ezisti.txt", "r") as f:
    konteúdu = f.read()
# FileNotFoundError: [Errno 2] No such file or directory
```

**Soluson:** Verifika antis di abri, o uza try/except (prósimu lisun!):

```python
import os

if os.path.exists("vendas.txt"):
    with open("vendas.txt", "r") as f:
        print(f.read())
else:
    print("Ficheru inda ka foi kriadu.")
```

### 2. Modu 'w' Ta Destrui Konteúdu

```python
# PERIGO! Es ta apaga tudu ki tinha na ficheru!
with open("dadus_importanti.txt", "w") as f:
    f.write("Ops...")  # Tudu anteriu ta bai pa sempri!
```

**Soluson:** Uza `'a'` pa adisiona, o lé primeru i dipos skrebi.

### 3. Skesi `seek(0)` ku `'w+'`

```python
with open("testu.txt", "w+") as f:
    f.write("Ola Mundu!")
    # Kursor ta na final — lé ta retorna string váziu!
    print(f.read())  # '' (nada!)

    # Korresun:
    f.seek(0)
    print(f.read())  # 'Ola Mundu!'
```

## Rezumu

- **Modus prinsipais:** `'r'` (lé), `'w'` (skrebi — apaga!), `'a'` (adisiona), `'w+'` (skrebi + lé)
- **Sempri uza `with`** — ta garanti ki ficheru ta fexá mesmu si erru akonteси
- **Lé:** `read()` (tudu), `readline()` (un linha), `readlines()` (lista), o `for linha in ficheru`
- **Skrebi:** `write(string)` i `writelines(lista)` — ka ta adisiona `\n` automátikamenti
- **`seek(0)`** — presiza dipos di skrebi ku `'w+'` pa volta pa kumesу
- **`os.path`** — kria kaminhus kompatível, verifika si ficheru ezisti
- **Kuidadu ku `'w'`** — ta destrui tudu konteúdu anteriu!
