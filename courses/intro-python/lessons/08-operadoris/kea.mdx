# Operadoris

Operadoris é sinais ki ta dize Python pa faze un operasun ku valoris. Kuandu bo skrebe `10 + 5`, sinal `+` é operador e `10` ku `5` é operandus. Na ês lisun, nôs ta aprendi três tipu prinsipal di operadoris: aritmétiku (pa kalkula), komparasun (pa kompara), e lójiku (pa kombina kondisons). Es é ferramentas fundamentál ki bo ta uza na tudu programa ki bo skrebe.

## Operadoris Aritmétiku

Es operadoris ta faze kálkulus matemátiku. Si bo sabe matemátika báziku, bo djá konhese maioria.

```python
# Adisun i Subtrasun
print(10 + 5)    # 15
print(10 - 3)    # 7

# Multiplikasun
print(4 * 3)     # 12

# Divisun (SEMPRI retorna float!)
print(10 / 5)    # 2.0  ← nota: 2.0, KA 2!
print(7 / 2)     # 3.5
```

> **💡 Dika:** Divisun ku `/` SEMPRI ta retorna un float (nùmeru ku pontu desimal), mesmu kuandu rezultadu é "interu". `10 / 5` ta da `2.0`, ka `2`. Es é un armadilha komun!

```python
# Divisun Interu (floor division) — tira parti desimal
print(7 // 2)    # 3  (ka 3.5 — tira desimal)
print(10 // 3)   # 3
print(21 // 5)   # 4

# Módulu (restu di divisun)
print(10 % 3)    # 1  (10 ÷ 3 = 3 restu 1)
print(15 % 5)    # 0  (divisun exata — sem restu)
print(7 % 2)     # 1  (ímpar: restu é 1)
```

```python
# Esponensasun (potensia)
print(2 ** 3)    # 8   (2 × 2 × 2)
print(5 ** 2)    # 25  (5 × 5)
print(10 ** 0)   # 1   (kualker nùmeru elevadu a 0 é 1)
print(9 ** 0.5)  # 3.0 (rais kuadrada!)
```

### Ezemplu Prátiku: Kalkula Troku ku Eskudu

```python
# Kalkula troku na loja
presu = 750      # presu na Eskudus (ECV)
pagamentu = 1000 # klientu pagá ku nota di 1000

troku = pagamentu - presu
print(f"Troku: {troku} ECV")  # Troku: 250 ECV

# Kuantas notas di 100 i kuantas moedas?
notas_100 = troku // 100     # 2 notas di 100
restu = troku % 100          # 50 ECV sobra
print(f"{notas_100} nota(s) di 100 ECV i {restu} ECV na moeda")
```

### Ezemplu Prátiku: Konversor di Moeda

```python
# Konversor Eskudu → Euro
eskudus = 5000
taxa_kambiu = 110.265  # 1 EUR = 110.265 ECV

euros = eskudus / taxa_kambiu
print(f"{eskudus} ECV = {euros:.2f} EUR")
# 5000 ECV = 45.35 EUR
```

### Par o Ímpar?

```python
# Truque ku módulu: verifica si nùmeru é par o ímpar
numeru = 17

if numeru % 2 == 0:
    print(f"{numeru} é par")
else:
    print(f"{numeru} é ímpar")
# 17 é ímpar
```

### Operadoris di Atribuisun Konposta

Bo pode kombina operadoris aritmétiku ku atribuisun pa atalhu.

```python
# Atalhu: operador + atribuisun
saldo = 1000

saldo += 500   # mesmu ki: saldo = saldo + 500
print(saldo)   # 1500

saldo -= 200   # mesmu ki: saldo = saldo - 200
print(saldo)   # 1300

saldo *= 2     # mesmu ki: saldo = saldo * 2
print(saldo)   # 2600

saldo //= 3    # mesmu ki: saldo = saldo // 3
print(saldo)   # 866
```

| Atalhu | Ekivalenti |
|--------|------------|
| `x += 5` | `x = x + 5` |
| `x -= 3` | `x = x - 3` |
| `x *= 2` | `x = x * 2` |
| `x /= 4` | `x = x / 4` |
| `x //= 2` | `x = x // 2` |
| `x %= 3` | `x = x % 3` |
| `x **= 2` | `x = x ** 2` |

## Operadoris di Komparasun

Es operadoris ta kompara dois valoris e ta retorna `True` o `False` (boolean). Es é essensial pa kondisionais (`if/elif/else`) ki bo ta aprendi na prósimu lisun.

```python
# Igual i Diferenti
print(10 == 10)   # True   (é igual?)
print(10 == 5)    # False
print(10 != 5)    # True   (é diferenti?)
print(10 != 10)   # False

# Maior i Menor
print(10 > 5)     # True
print(10 < 5)     # False
print(10 >= 10)   # True   (maior O igual)
print(10 <= 9)    # False  (menor O igual)
```

> **💡 Dika:** Ka konfundi `=` (atribuisun) ku `==` (komparasun)! `x = 5` ta guarda valor 5 na `x`. `x == 5` ta pergunta "x é igual a 5?".

```python
# Komparasun di strings — é CASE SENSITIVE!
print("Praia" == "Praia")    # True
print("Praia" == "praia")    # False! (P maiúskula ≠ p minúskula)
print("Praia".lower() == "praia".lower())  # True (soluson!)
```

```python
# Komparasun ku variáveis
idadi_joao = 25
idadi_ana = 30

print(f"João é más velhu ki Ana? {idadi_joao > idadi_ana}")   # False
print(f"Ana é más velhu ki João? {idadi_ana > idadi_joao}")    # True
print(f"Es tene mesmu idadi? {idadi_joao == idadi_ana}")       # False
```

### Ezemplu Prátiku: Verifikasun di Idadi

```python
# Verifica si pesoa pode vota (>= 18 anu)
nomi = "Nilton"
idadi = 17

pode_vota = idadi >= 18
print(f"{nomi} pode vota? {pode_vota}")  # Nilton pode vota? False
```

## Operadoris Lójiku

Operadoris lójiku ta kombina vários kondisons. Pensa es komu "e", "o", "ka" na Kriolu.

### `and` — AMBOS tene ki ser True

```python
# and: tudu kondisons tene ki ser verdaderu
idadi = 25
tene_bilheti = True

print(idadi >= 18 and tene_bilheti)  # True (25 >= 18 É tene bilheti)
print(idadi >= 30 and tene_bilheti)  # False (25 ka >= 30)
```

### `or` — ALGUN tene ki ser True

```python
# or: pelo menus UN kondisun tene ki ser verdaderu
é_studanti = False
é_idosu = True

tene_diskontu = é_studanti or é_idosu
print(tene_diskontu)  # True (pelo menus un é True)
```

### `not` — inverte True/False

```python
# not: inverte valor lójiku
ta_xobe = True
print(not ta_xobe)  # False

loja_aberta = False
print(not loja_aberta)  # True
```

### Kombina Tudu Djuntu

```python
# Ezemplu kompletu: validasun di bilheti
idadi = 22
é_studanti = True
tene_dinheru = True

# Pode entra si: (tene 18+ anu I tene dinheru) O é studanti
pode_entra = (idadi >= 18 and tene_dinheru) or é_studanti
print(f"Pode entra? {pode_entra}")  # True
```

```python
# Ezemplu prátiku: validasun di senha
senha = "Py2026!"

tene_tamanhu = len(senha) >= 6
tene_numeru = any(c.isdigit() for c in senha)
tene_letra = any(c.isalpha() for c in senha)

senha_forte = tene_tamanhu and tene_numeru and tene_letra
print(f"Senha forte? {senha_forte}")  # True
```

## Tabela di Verdadi (Truth Table)

Pa intende `and`, `or`, e `not`, ê tabela ta ajuda:

```python
# Tabela di verdadi pa 'and'
print(True and True)    # True
print(True and False)   # False
print(False and True)   # False
print(False and False)  # False
# Rezumu: and só é True kuandu AMBOS é True

# Tabela di verdadi pa 'or'
print(True or True)     # True
print(True or False)    # True
print(False or True)    # True
print(False or False)   # False
# Rezumu: or só é False kuandu AMBOS é False

# Tabela di verdadi pa 'not'
print(not True)    # False
print(not False)   # True
```

## Presedensia di Operadoris (Orden di Exekusun)

Python ta sigi un orden espesífiku kuandu tene vários operadoris na mesmu expresun. É igualzinhu ki matemátika — multiplikasun antis di adisun.

```python
# Sem paréntesis: Python sigi presedensia
resultado = 2 + 3 * 4
print(resultado)  # 14 (ka 20! — * antis di +)

# Ku paréntesis: bo kontrola orden
resultado = (2 + 3) * 4
print(resultado)  # 20
```

**Orden di presedensia (di más altu pa más baixu):**

| Priuridad | Operador | Descrisun |
|-----------|----------|-----------|
| 1 (más altu) | `**` | Esponensasun |
| 2 | `not` | Negasun lójika |
| 3 | `*`, `/`, `//`, `%` | Multiplikasun, divisun, módulu |
| 4 | `+`, `-` | Adisun, subtrasun |
| 5 | `==`, `!=`, `>`, `<`, `>=`, `<=` | Komparasun |
| 6 | `and` | E lójiku |
| 7 (más baixu) | `or` | Ou lójiku |

```python
# Ezemplu di presedensia
resultado = 2 ** 3 + 4 * 2 - 1
# Pasu 1: 2 ** 3 = 8       (esponensasun primeiru)
# Pasu 2: 4 * 2 = 8        (multiplikasun)
# Pasu 3: 8 + 8 - 1 = 15   (adisun i subtrasun)
print(resultado)  # 15
```

> **💡 Dika:** Na dúvida, uza **paréntesis**! `(2 + 3) * 4` é más klaru ki depende di presedensia. Kódiku klaru é más importante ki kódiku kurtu.

## Armadilhas Komun

```python
# ARMADILHA 1: / sempri retorna float
print(type(10 / 5))   # <class 'float'> — 2.0, ka 2!
print(type(10 // 5))  # <class 'int'>   — 2 (si bo kre interu, uza //)

# ARMADILHA 2: = vs ==
x = 5      # atribuisun (guarda valor)
x == 5     # komparasun (pergunta si é igual) → True

# ARMADILHA 3: komparasun di string é case-sensitive
nomi = "Pedro"
print(nomi == "pedro")  # False!
print(nomi.lower() == "pedro")  # True (soluson)

# ARMADILHA 4: divisun interu ku negativus
print(7 // 2)    #  3  ← normal
print(-7 // 2)   # -4  ← arredonda pa BAIXU, ka pa zeru!
```

## Rezumu

- **Aritmétiku:** `+`, `-`, `*` funsiona komu esperadu; `/` SEMPRI retorna float; `//` pa divisun interu; `%` pa restu; `**` pa potensia
- **Atribuisun konposta:** `+=`, `-=`, `*=` é atalhu pa `x = x op valor`
- **Komparasun:** `==` (igual), `!=` (diferenti), `>`, `<`, `>=`, `<=` — tudu retorna True/False
- **Lójiku:** `and` (ambos True), `or` (algun True), `not` (inverte)
- **Presedensia:** `**` > `not` > `* / // %` > `+ -` > komparasun > `and` > `or`
- **Ka konfundi** `=` (atribuisun) ku `==` (komparasun)
- **Na dúvida, uza paréntesis** pa torna kódiku más klaru
