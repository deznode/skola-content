# Sintaxi i Semantika

Tudu lingua tene regras. Kriolu tene gramátika, i Python tene **syntax** (sintaxi). Si bo skrebe un frazi ku palavras fora di orden, ninguén ta intende. Mesmu kuza ku kódiku — si bu ka sigi regras di Python, el ta da eru.

Nes lisan, nu ta prende regras fundamentais di Python: kumo skrebe komentárius, pamodi letras maiúskulas i minúskulas importa, i kumo Python uza indentasan pa organiza kódiku.

## Syntax vs. Semantics

**Syntax** é regras di kumo bo skrebe kódiku. **Semantics** é signifikadu di kódiku — kuze ki el ta faze kuandu bo roda.

Pensa assin:
- **Syntax** = gramátika di kriolu ("Nha nomi é Maria" — regra di frazi)
- **Semantics** = signifikadu ("Maria é nha nomi" — nu ta intende)

```python
# Syntax koretu, semantics koretu
print("Ola, Mundu!")  # Ta imprimi: Ola, Mundu!

# Syntax INKORRETU — Python ka ta intendi
# print("Ola, Mundu!"  # SyntaxError: falta parentesis di fitxa
```

Si bo skrebe kódiku ku syntax eradu, Python ta da **SyntaxError** antis di roda kualker kuza. Si syntax sta koretu ma kódiku ta faze algu diferenti di ki bo kria, kel é un problema di **semantics**.

```python
# Syntax koretu, ma semantics eradu
# Bo kria kalkula média, ma bo dividi pa 3 en bez di 2
nota1 = 85
nota2 = 92
media = (nota1 + nota2) / 3  # Eru di lójika! Devia ser / 2
print(media)  # 59.0 — resultado eradu, ma Python ka reklama
```

:::callout{type=tip}
**Dika:** Python só ta deteta erus di syntax. Erus di lójika (semantics) é bo responsabilidadi — testa senpri bo kódiku!
:::

## Komentárius ku `#`

Komentárius é notus ki bo skrebe pa bo mesmu ó pa otu programadoris. Python ta ignora tudu kuza dipos di `#`.

```python
# Kel é un komentáriu — Python ta ignora kel
nomi = "Cesária"  # Nomi di estudanti

# Kalkula média di notus
nota1 = 85
nota2 = 92
media = (nota1 + nota2) / 2  # 88.5
print(f"Média di {nomi}: {media}")
```

Komentárius é sabi pa:
- Splika pamodi bo skrebe kódiku di un sertu manera
- Djuda otu pesoas (ó bo futuru!) intende bo kódiku
- Desativa kódiku temporariamenti (pa testa)

```python
# Bo podi desativa un linha di kódiku pon # na frenti
presu = 1500
# presu = presu * 1.10  # Desativadu — ka ta aplika desconto gosi
print(f"Presu: {presu} ECV")
```

:::callout{type=tip}
**Dika:** Komentárius sabi ta splika **pamodi**, ka **kuze**. En bez di `# soma a + b`, skrebe `# kalkula total di venda`.
:::

## Case Sensitivity

Python é **case sensitive** — el ta distingui letras maiúskulas di minúskulas. `nomi`, `Nomi` i `NOMI` é **trez variáveis diferenti**.

```python
nomi = "Maria"
Nomi = "João"
NOMI = "Pedro"

print(nomi)  # Maria
print(Nomi)  # João
print(NOMI)  # Pedro
```

Kel é importanti pamodi:

```python
# print() funsiona — é funson built-in
print("Ola!")

# Print() ka funsiona — P maiúskulu
# Print("Ola!")  # NameError: name 'Print' is not defined
```

```python
# True i False tene T i F maiúskulu
ativo = True   # Koretu
# ativo = true  # NameError: name 'true' is not defined
```

:::callout{type=tip}
**Dika:** Na prátika, skrebe variáveis senpri ku letras minúskulas: `nomi`, `idadi`, `nota_final`. É konvensan di Python (PEP 8).
:::

## Indentasan — Blokus di Kódiku

Na otu linguajens (Java, C), bu ta uza `{ }` pa defini blokus di kódiku. Na Python, nu ta uza **indentasan** — spasus na kumessu di linha.

Regra é simples: **4 spasus** pa kada nível di indentasan.

:::callout{type=tip}
**Ka preokupa si bu ka kompriende tudu kódiku gosi.** Lisan-li é so pa odja kumo indentasan ta defini blokus — `if`, `>=`, i variáveis ta ben más tardi (Módulu 1 Lisan 5, 8 i 9). Repara na spasus, ka na lójika.
:::

```python
idadi = 20

if idadi >= 18:
    print("Bo é adultu")      # 4 spasus — dentu di if
    print("Bo podi vota")     # 4 spasus — inda dentu di if

print("Programa termina")     # Sen indentasan — fora di if
```

Kuandu bo roda kel kódiku ku `idadi = 20`:
```
Bo é adultu
Bo podi vota
Programa termina
```

Kuandu bo roda ku `idadi = 15`:
```
Programa termina
```

Si bo skesi indentasan, Python ta da eru:

```python
if idadi >= 18:
print("Bo é adultu")  # IndentationError: expected an indented block
```

Indentasan tanbe ta funsiona ku vários níveis:

```python
nota = 85

if nota >= 70:
    print("Bo pasa!")           # nível 1 (4 spasus)
    if nota >= 90:
        print("Exselenti!")     # nível 2 (8 spasus)
    else:
        print("Bon trabadjo!")  # nível 2 (8 spasus)
```

:::callout{type=tip}
**Dika:** Konfigura bo VS Code pa mostra spasus: Settings → "Render Whitespace" → "all". Assin bu ta odja si bo indentasan sta koretu.
:::

## Kontinuasan di Linha ku `\`

Ás vezis un linha di kódiku é mutu longu. Bu pode kebra el ku **backslash** (`\`) pa kontinua na prósimu linha.

```python
# Linha mutu longu — difísil di le
total = 100 + 200 + 300 + 400 + 500 + 600 + 700 + 800 + 900 + 1000

# Melhor — uza \ pa kebra
total = 100 + 200 + 300 + 400 + 500 + \
        600 + 700 + 800 + 900 + 1000
print(total)  # 5500
```

Dentu di parentesis, kolxetis ó xavis, bu ka presiza `\` — Python ta intende automátikamenti:

```python
# Parentesis ta permiti kebra di linha sen \
ilhas = (
    "Santiago",
    "São Vicente",
    "Sal",
    "Fogo",
    "Santo Antão",
    "Boa Vista"
)
print(ilhas)

# Mesmu ku lista (kolxetis)
notus = [
    85, 92, 78,
    90, 88, 95
]
```

*(Tupla `(...)` i lista `[...]`: bu ta prende más na Módulu 2, Lisan 11 i 12. Pa gosi, so fixa ki dentu di parentesis Python ka presiza `\`.)*

:::callout{type=tip}
**Dika:** Na prátika, é midjór uza parentesis pa kebra linhas longas. É más limpu ki `\`.
:::

## Dynamic Typing

Na Python, bu ka presiza deklara tipu di un variável. Python ta diskubri tipu automátikamenti — kel ta txoma **dynamic typing**.

```python
valor = 10
print(type(valor))  # <class 'int'>

valor = "Mindelo"
print(type(valor))  # <class 'str'>

valor = 3.14
print(type(valor))  # <class 'float'>

valor = True
print(type(valor))  # <class 'bool'>
```

Mesmu variável `valor` muda di `int` pa `str` pa `float` pa `bool` — Python ka reklama!

Kel é diferenti di linguajens kumo Java ó C++, undi bu ten ki deklara tipu:

```python
# Na Python (dynamic typing):
nomi = "Ana"          # Python sabi ki é str

# Na Java (static typing) — só pa komparasan:
# String nomi = "Ana";  // Bo ten ki dizi ki é String
```

Funsan `type()` é bo amigu pa verifika tipu di kualker valor:

```python
presu_katxupa = 350
nomi_pratu = "Katxupa Guisada"
ten_grogu = True

print(type(presu_katxupa))  # <class 'int'>
print(type(nomi_pratu))     # <class 'str'>
print(type(ten_grogu))     # <class 'bool'>
```

:::callout{type=tip}
**Dika:** Dynamic typing ta da liberdadi, ma tanbe responsabilidadi. Tenta ka muda tipu di un variável sen razun — kódiku ta fika konfuzu.
:::

## Erus Komuns di Sintaxi

Kel é erus ki tudu prinsipianti ta enkontra. Ka ten problemas — é normal!

### 1. IndentationError

```python
# ERADU — falta indentasan dipos di if
if True:
print("Ola")  # IndentationError!

# KORETU
if True:
    print("Ola")
```

### 2. SyntaxError — Parentesis

```python
# ERADU — falta parentesis di fitxa
print("Ola"  # SyntaxError!

# KORETU
print("Ola")
```

### 3. NameError — Variável ka definidu

```python
# ERADU — bo uza variável antis di kria el
print(mensajen)  # NameError: name 'mensajen' is not defined

# KORETU
mensajen = "Ola!"
print(mensajen)
```

### 4. Case sensitivity na funsons

```python
# ERADU
Print("Ola")  # NameError: name 'Print' is not defined

# KORETU
print("Ola")
```

## Rezumu

- **Syntax** é regras di kumo skrebe kódiku; **semantics** é kuze ki kódiku signifika
- **Komentárius** (`#`) ta djuda dokumenta kódiku — Python ta ignora es
- Python é **case sensitive**: `nomi` i `Nomi` é variáveis diferenti
- **Indentasan** (4 spasus) ta defini blokus di kódiku — ka ten `{ }` na Python
- Uza `\` ó parentesis pa **kontinua linhas longas**
- **Dynamic typing**: variáveis pode muda di tipu; uza `type()` pa verifika
