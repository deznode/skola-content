# Eransa (Simples i Múltiplu)

Djina tene olus verdi di si mai i altura di si pai. Ela erda karakterístikas di dos — ka presiza "inventa" olus o altura di zero, es ven di si famía. I Djina inda tene si própriu trabus únikis: si rizu, si manera di fala, si talentu pa toka violun.

Na programasan, **eransa** funsiona mesmu manera. Un klasi filhu ta **erda** tudu atributus i metodus di klasi pai, i ainda pode adisiona si própriu koisa o muda kel ki erda. Isu ta evita repetisan di kódiku i ta kria un ierárkia naturál — mesmu kumo árvori genealójika.

## Eransa Simples

Na eransa simples, un klasi filhu ta erda di **un só** klasi pai. Bu ta indika pai dentu di paréntesis:

```python
# Klasi pai (bazi)
class Viatúra:
    def __init__(self, marka, modelu, anu):
        self.marka = marka
        self.modelu = modelu
        self.anu = anu

    def liga(self):
        print(f"🚗 {self.marka} {self.modelu} ta ligá!")

    def deskrebe(self):
        return f"{self.marka} {self.modelu} ({self.anu})"

# Klasi filhu — erda di Viatúra
class Táksi(Viatúra):
    pass  # Ka adisiona nada novu — só erda tudu di Viatúra

# Táksi tene TUDU ki Viatúra tene!
taksi = Táksi("Toyota", "Hilux", 2020)
taksi.liga()              # 🚗 Toyota Hilux ta ligá!
print(taksi.deskrebe())   # Toyota Hilux (2020)
print(taksi.marka)        # Toyota
```

Mesmu ku `Táksi` tene `pass` (váziu), el erda `__init__`, `liga()`, i `deskrebe()` di `Viatúra`. É kumo un fidju ki tene tudu trabus di pai sen faze nada estraordinária.

## `super()` — Txoma Konstrutor di Pai

Normalmenti, klasi filhu kre **adisiona** atributus novu alémd di kel ki pai dja tene. Pa isu, bo uza `super().__init__()` pa txoma konstrutor di pai primeru, dipos adisiona bo koisa:

```python
class Viatúra:
    def __init__(self, marka, modelu, anu):
        self.marka = marka
        self.modelu = modelu
        self.anu = anu

    def liga(self):
        print(f"🚗 {self.marka} {self.modelu} ta ligá!")

    def deskrebe(self):
        return f"{self.marka} {self.modelu} ({self.anu})"


class Táksi(Viatúra):
    def __init__(self, marka, modelu, anu, lisensa, zona):
        # Txoma konstrutor di pai pa inisializa marka, modelu, anu
        super().__init__(marka, modelu, anu)
        # Adisiona atributus própriu di Táksi
        self.lisensa = lisensa
        self.zona = zona
        self.korridas = 0

    def fazi_korrida(self):
        """Rejista un korrida"""
        self.korridas += 1
        print(f"🚕 Korrida #{self.korridas} na zona {self.zona}")

    def deskrebe(self):
        """Sobreskrebe métudu di pai ku informasan extra"""
        bazi = super().deskrebe()  # Txoma versan di pai
        return f"{bazi} — Táksi Lisensa: {self.lisensa} (Zona: {self.zona})"


# Kria táksi
taksi_praia = Táksi("Toyota", "Hilux", 2020, "TX-1234", "Praia")
taksi_praia.liga()             # 🚗 Toyota Hilux ta ligá! (erdadu di Viatúra)
taksi_praia.fazi_korrida()     # 🚕 Korrida #1 na zona Praia
taksi_praia.fazi_korrida()     # 🚕 Korrida #2 na zona Praia
print(taksi_praia.deskrebe())  # Toyota Hilux (2020) — Táksi Lisensa: TX-1234 (Zona: Praia)
```

```python
# Más un ezemplu: Veíkulu elétriku
class ViatúraElétrika(Viatúra):
    def __init__(self, marka, modelu, anu, bateria_kwh):
        super().__init__(marka, modelu, anu)
        self.bateria_kwh = bateria_kwh
        self.karga_atuál = 100  # Perséntu

    def karega(self):
        """Karega bateria pa 100%"""
        self.karga_atuál = 100
        print(f"🔋 {self.marka} {self.modelu} karegadu! Bateria: {self.bateria_kwh} kWh")

    def liga(self):
        """Sobreskrebe — veíkulu elétriku é silensiozu"""
        if self.karga_atuál > 0:
            print(f"⚡ {self.marka} {self.modelu} ta ligá silensiozamenti...")
        else:
            print(f"🪫 Bateria váziu! Karega primeiru.")

evu = ViatúraElétrika("BYD", "Dolphin", 2025, 60)
evu.liga()       # ⚡ BYD Dolphin ta ligá silensiozamenti...
evu.karega()     # 🔋 BYD Dolphin karegadu! Bateria: 60 kWh
```

:::callout{type=tip}
`super()` ta dize "txoma versan di nha pai pa es métudu". Uza `super().__init__()` pa ka repeti kódiku ki pai dja ta faze. Depois, adisiona só kel ki é novu.
:::

## Eransa Múltiplu

Python ta permiti un klasi erda di **vários pais** (diferenti di algunus línguas kumo Java). Pa eransa múltiplu, bu ta uza txomada **direta** a kada pai:

```python
class Artista:
    def __init__(self, nomi, jeneru_muzikál):
        self.nomi = nomi
        self.jeneru_muzikál = jeneru_muzikál

    def atua(self):
        print(f"🎤 {self.nomi} ta atua {self.jeneru_muzikál}!")


class Kompozitor:
    def __init__(self, instrumentu, núm_kompozisons):
        self.instrumentu = instrumentu
        self.núm_kompozisons = núm_kompozisons

    def kompoe(self, títulu):
        self.núm_kompozisons += 1
        print(f"🎼 Kompozisan #{self.núm_kompozisons}: '{títulu}'")


class MúzikuKompletu(Artista, Kompozitor):
    def __init__(self, nomi, jeneru_muzikál, instrumentu, núm_kompozisons):
        # Txomada direta a kada pai — pa eransa múltiplu
        Artista.__init__(self, nomi, jeneru_muzikál)
        Kompozitor.__init__(self, instrumentu, núm_kompozisons)
        self.álbuns = []

    def lansa_álbun(self, nomi_álbun):
        """Lansa un álbun novu"""
        self.álbuns.append(nomi_álbun)
        print(f"💿 Álbun '{nomi_álbun}' lansadu! Total: {len(self.álbuns)} álbuns")

    def perfil(self):
        """Mostra perfil kompletu"""
        print(f"\n{'='*40}")
        print(f"  🎵 {self.nomi}")
        print(f"  Jéneru: {self.jeneru_muzikál}")
        print(f"  Instrumentu: {self.instrumentu}")
        print(f"  Kompozisons: {self.núm_kompozisons}")
        print(f"  Álbuns: {len(self.álbuns)}")
        print(f"{'='*40}\n")


# Kria un múziku konpletu
cesaria = MúzikuKompletu("Cesária Évora", "Morna", "Voz", 50)
cesaria.atua()                         # 🎤 Cesária Évora ta atua Morna!
cesaria.kompoe("Mar Azul")             # 🎼 Kompozisan #51: 'Mar Azul'
cesaria.lansa_álbun("Voz d'Amor")      # 💿 Álbun 'Voz d'Amor' lansadu!
cesaria.lansa_álbun("Cabo Verde")      # 💿 Álbun 'Cabo Verde' lansadu!
cesaria.perfil()
# ========================================
#   🎵 Cesária Évora
#   Jéneru: Morna
#   Instrumentu: Voz
#   Kompozisons: 51
#   Álbuns: 2
# ========================================
```

:::callout{type=tip}
Pa eransa **simples** (un só pai), uza `super().__init__()`. Pa eransa **múltiplu** (vários pais), uza `Pai.__init__(self, ...)` pa kada pai. É regra di ouru!
:::

## Method Resolution Order (MRO)

Kuandu un klasi erda di vários pais, Python presiza sabi **ki orden** pa buska metodus. Es orden txoma **MRO** (Method Resolution Order). Python ta sigi algoritmu C3 — di filhu pa pai, di skerda pa direita:

```python
class A:
    def saudasan(self):
        print("Ola di A!")

class B(A):
    def saudasan(self):
        print("Ola di B!")

class C(A):
    def saudasan(self):
        print("Ola di C!")

class D(B, C):
    pass  # Ka defini saudasan — vai buska na pais

d = D()
d.saudasan()  # Ola di B!  ← pamodi B ven antis di C

# Verifika MRO
print(D.__mro__)
# (<class 'D'>, <class 'B'>, <class 'C'>, <class 'A'>, <class 'object'>)
# Orden: D → B → C → A → object
```

```python
# Ezemplu prátiku di MRO ku kultura CV
class Morabeza:
    def kualidadi(self):
        return "Morabeza (ospitalidadi)"

class Kriativdadi:
    def kualidadi(self):
        return "Kriativdadi"

class Kaboverdianu(Morabeza, Kriativdadi):
    pass  # Erda di dos

kv = Kaboverdianu()
print(kv.kualidadi())   # Morabeza (ospitalidadi)  ← Morabeza ven primeiru

# Pa ve orden di buska
for klasi in Kaboverdianu.__mro__:
    print(f"  → {klasi.__name__}")
#   → Kaboverdianu
#   → Morabeza
#   → Kriativdadi
#   → object
```

:::callout{type=tip}
Si bu ka sabe ki métudu ta ser txomadu, uza `Klasi.__mro__` o `Klasi.mro()` pa ve orden ezata di buska.
:::

## `isinstance()` i `issubclass()`

Python ta fornese dos funsons pa verifika relasan entri objetis i klasis:

```python
class Viatúra:
    pass

class Táksi(Viatúra):
    pass

class Autokarru(Viatúra):
    pass

# Kria objetis
taksi = Táksi()
bus = Autokarru()

# isinstance() — verifica si objetu é instánsia di un klasi
print(isinstance(taksi, Táksi))      # True — taksi É un Táksi
print(isinstance(taksi, Viatúra))    # True — taksi TAMBÉN é un Viatúra (erdansa!)
print(isinstance(taksi, Autokarru))  # False — taksi KA é un Autokarru

# issubclass() — verifica si un klasi é filhu di otru
print(issubclass(Táksi, Viatúra))      # True — Táksi é filhu di Viatúra
print(issubclass(Autokarru, Viatúra))  # True — Autokarru é filhu di Viatúra
print(issubclass(Táksi, Autokarru))    # False — Táksi ka é filhu di Autokarru
print(issubclass(Viatúra, Viatúra))    # True — klasi é subklasi di si mesmu
```

```python
# Uzu prátiku: funsan ki ta aseta diferentis tipu
def prosesa_viatúra(v):
    """Prosesa kualker tipu di viatúra"""
    if not isinstance(v, Viatúra):
        print("❌ Isu ka é un viatúra!")
        return

    print(f"✅ Prosesandu: {type(v).__name__}")

prosesa_viatúra(taksi)     # ✅ Prosesandu: Táksi
prosesa_viatúra(bus)        # ✅ Prosesandu: Autokarru
prosesa_viatúra("minibus")  # ❌ Isu ka é un viatúra!
```

## Izemplu Prátiku: Sistema di Trabadjoris

Nu ta kria un sistema ierárkiku pa un enpresa na Praia:

```python
class Trabadjor:
    """Klasi bazi pa tudu trabadjoris"""
    def __init__(self, nomi, idadi, saláriu):
        self.nomi = nomi
        self.idadi = idadi
        self.saláriu = saláriu    # Saláriu mensál na ECV

    def apresenta(self):
        return f"{self.nomi}, {self.idadi} anus"

    def saláriu_anuál(self):
        """Kalkula saláriu anuál (12 mezis + subsídiu di férias)"""
        return self.saláriu * 13  # 12 mezis + 1 subsídiu


class Profesór(Trabadjor):
    """Profesór — erda di Trabadjor"""
    def __init__(self, nomi, idadi, saláriu, matéria, skola):
        super().__init__(nomi, idadi, saláriu)
        self.matéria = matéria
        self.skola = skola
        self.alúnus = []

    def adisiona_alúnu(self, nomi_alúnu):
        self.alúnus.append(nomi_alúnu)

    def apresenta(self):
        bazi = super().apresenta()
        return f"{bazi} — Profesór di {self.matéria} na {self.skola}"


class Médiku(Trabadjor):
    """Médiku — erda di Trabadjor"""
    def __init__(self, nomi, idadi, saláriu, espesialidadi, ospitál):
        super().__init__(nomi, idadi, saláriu)
        self.espesialidadi = espesialidadi
        self.ospitál = ospitál
        self.pasiéntis_oji = 0

    def atende_pasiénti(self):
        self.pasiéntis_oji += 1
        print(f"🏥 Dr. {self.nomi} ta atende pasiénti #{self.pasiéntis_oji}")

    def apresenta(self):
        bazi = super().apresenta()
        return f"{bazi} — Médiku di {self.espesialidadi} na {self.ospitál}"


class Enjenheru(Trabadjor):
    """Enjenheru — erda di Trabadjor"""
    def __init__(self, nomi, idadi, saláriu, área, enpresa):
        super().__init__(nomi, idadi, saláriu)
        self.área = área
        self.enpresa = enpresa

    def apresenta(self):
        bazi = super().apresenta()
        return f"{bazi} — Enjenheru di {self.área} na {self.enpresa}"
```

```python
# Kria trabadjoris
prof = Profesór("Amilcar", 45, 65000, "Matimátika", "Liséu Domingos Ramos")
prof.adisiona_alúnu("Ana")
prof.adisiona_alúnu("Kelvin")
prof.adisiona_alúnu("Edna")

med = Médiku("Carla", 38, 120000, "Pediatria", "Hospital Agostinho Neto")
med.atende_pasiénti()   # 🏥 Dr. Carla ta atende pasiénti #1
med.atende_pasiénti()   # 🏥 Dr. Carla ta atende pasiénti #2

enj = Enjenheru("Nelson", 32, 90000, "Software", "Skola.dev")

# Tudu é Trabadjor!
trabadjoris = [prof, med, enj]

print("\n📋 Ekipa:")
for t in trabadjoris:
    print(f"  {t.apresenta()}")
    print(f"    Saláriu anuál: {t.saláriu_anuál():,} ECV")

# 📋 Ekipa:
#   Amilcar, 45 anus — Profesór di Matimátika na Liséu Domingos Ramos
#     Saláriu anuál: 845,000 ECV
#   Carla, 38 anus — Médiku di Pediatria na Hospital Agostinho Neto
#     Saláriu anuál: 1,560,000 ECV
#   Nelson, 32 anus — Enjenheru di Software na Skola.dev
#     Saláriu anuál: 1,170,000 ECV
```

```python
# isinstance() ta funsiona ku ierarkia
print(isinstance(prof, Profesór))    # True
print(isinstance(prof, Trabadjor))   # True  ← Profesór É un Trabadjor
print(isinstance(prof, Médiku))      # False ← Profesór KA é un Médiku

# Konta tipu di trabadjoris
médikus = [t for t in trabadjoris if isinstance(t, Médiku)]
print(f"\nTotal médikus: {len(médikus)}")  # Total médikus: 1
```

## Rezumu
<KeyTakeaways>
  <RezumuItem variant="gold" term="Regra di oru">Pa eransa **simples** (un pai), uza `super().__init__()`; pa eransa **múltiplu** (vários pai), uza `Pai.__init__(self, ...)` direta pa kada pai.</RezumuItem>
  <RezumuItem term="Eransa">`class Filhu(Pai):` ta erda tudu os atributu i metodu di o pai — reuza kódiku sen repetisan.</RezumuItem>
  <RezumuItem term="super()">Ta txoma o konstrutor (o un metodu) di o pai — bu adisiona só o ki é novu.</RezumuItem>
  <RezumuItem term="MRO">Method Resolution Order ta determina a orden di buska: filhu → pai (skerda pa direita) → avós. `Klasi.__mro__` ta mostra-l.</RezumuItem>
  <RezumuItem term="isinstance / issubclass">`isinstance(obj, Klasi)` (inklui eransa); `issubclass(Filhu, Pai)`.</RezumuItem>
  <RezumuItem variant="tip" term="Pista">Bu pode **sobreskrebe** un metodu na o filhu pa mudá o komportamentu erdadu — i txoma a versan di o pai ku `super().metodu()`.</RezumuItem>
</KeyTakeaways>
