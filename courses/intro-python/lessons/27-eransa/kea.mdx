# Eransa Simples

Djina tene odjus verdi di se mai i altura di se pai. El ta erda karakterístikas di dos — ka presiza "inventa" odjus ou altura di zero, es ven di se famía. I Djina inda tene se própriu trabus únikis: se rizu, se manera di fala, se talentu pa toka violun.

Na programasan, **eransa** ta funsiona mesmu manera. Un klasi filhu ta **erda** tudu atributus i metodus di klasi pai, i ainda pode adisiona se própriu kuza ou muda kel ki erda. Isu ta evita repetisan di kódiku i ta kria un ierárkia naturál — mesmu kumo árvori genealójika.

## Eransa Simples

Na eransa simples, un klasi filhu ta erda di **un só** klasi pai. Bu ta indika pai dentu di paréntesis:

```python
# Klasi pai (bazi)
class Viatúra:
    def __init__(self, marka, modelu, anu):
        self.marka = marka
        self.modelu = modelu
        self.anu = anu

    def liga(self):
        print(f"🚗 {self.marka} {self.modelu} ta ligá!")

    def deskrebe(self):
        return f"{self.marka} {self.modelu} ({self.anu})"

# Klasi filhu — erda di Viatúra
class Taxi(Viatúra):
    pass  # Ka adisiona nada novu — só erda tudu di Viatúra

# Taxi tene TUDU ki Viatúra tene!
taxi = Taxi("Toyota", "Hilux", 2020)
taxi.liga()              # 🚗 Toyota Hilux ta ligá!
print(taxi.deskrebe())   # Toyota Hilux (2020)
print(taxi.marka)        # Toyota
```

Mesmu ku `Taxi` tene `pass` (váziu), el ta erda `__init__`, `liga()`, i `deskrebe()` di `Viatúra`. É kumo un fidju ki tene tudu trabus di pai sen faze nada estraordinária.

## `super()` — Txoma Konstrutor di Pai

Normalmenti, klasi filhu kre **adisiona** atributus novu alén di kel ki pai dja tene. Pa isu, bo uza `super().__init__()` pa txoma konstrutor di pai primeru, dipos adisiona bo kuza:

```python
class Viatúra:
    def __init__(self, marka, modelu, anu):
        self.marka = marka
        self.modelu = modelu
        self.anu = anu

    def liga(self):
        print(f"🚗 {self.marka} {self.modelu} ta ligá!")

    def deskrebe(self):
        return f"{self.marka} {self.modelu} ({self.anu})"


class Taxi(Viatúra):
    def __init__(self, marka, modelu, anu, lisensa, zona):
        # Txoma konstrutor di pai pa inisializa marka, modelu, anu
        super().__init__(marka, modelu, anu)
        # Adisiona atributus própriu di Taxi
        self.lisensa = lisensa
        self.zona = zona
        self.korridas = 0

    def fazi_korrida(self):
        """Rejista un korrida"""
        self.korridas += 1
        print(f"🚕 Korrida #{self.korridas} na zona {self.zona}")

    def deskrebe(self):
        """Sobreskrebe métudu di pai ku informasan extra"""
        bazi = super().deskrebe()  # Txoma versan di pai
        return f"{bazi} — Taxi Lisensa: {self.lisensa} (Zona: {self.zona})"


# Kria taxi
taxi_praia = Taxi("Toyota", "Hilux", 2020, "TX-1234", "Praia")
taxi_praia.liga()             # 🚗 Toyota Hilux ta ligá! (erdadu di Viatúra)
taxi_praia.fazi_korrida()     # 🚕 Korrida #1 na zona Praia
taxi_praia.fazi_korrida()     # 🚕 Korrida #2 na zona Praia
print(taxi_praia.deskrebe())  # Toyota Hilux (2020) — Taxi Lisensa: TX-1234 (Zona: Praia)
```

```python
# Más un ezemplu: Veíkulu elétriku
class ViatúraElétrika(Viatúra):
    def __init__(self, marka, modelu, anu, bateria_kwh):
        super().__init__(marka, modelu, anu)
        self.bateria_kwh = bateria_kwh
        self.karga_atuál = 100  # Perséntu

    def karega(self):
        """Karega bateria pa 100%"""
        self.karga_atuál = 100
        print(f"🔋 {self.marka} {self.modelu} karegadu! Bateria: {self.bateria_kwh} kWh")

    def liga(self):
        """Sobreskrebe — veíkulu elétriku é silensiozu"""
        if self.karga_atuál > 0:
            print(f"⚡ {self.marka} {self.modelu} ta ligá silensiozamenti...")
        else:
            print(f"🪫 Bateria váziu! Karega primeiru.")

evu = ViatúraElétrika("BYD", "Dolphin", 2025, 60)
evu.liga()       # ⚡ BYD Dolphin ta ligá silensiozamenti...
evu.karega()     # 🔋 BYD Dolphin karegadu! Bateria: 60 kWh
```

:::callout{type=tip}
`super()` ta dize "txoma versan di nha pai pa es métudu". Uza `super().__init__()` pa ka repeti kódiku ki pai dja ta faze. Depois, adisiona só kel ki é novu.
:::

## Izemplu Prátiku: Sistema di Trabadjoris

Nu ta kria un sistema ierárkiku pa un enpresa na Praia. Un klasi bazi `Trabadjor` ta guarda o ki tudu trabadjor tene en komun; kada tipu di trabadjor ta **erda** di el i ta **sobreskrebe** `apresenta()`:

```python
class Trabadjor:
    """Klasi bazi pa tudu trabadjoris"""
    def __init__(self, nomi, idadi, saláriu):
        self.nomi = nomi
        self.idadi = idadi
        self.saláriu = saláriu    # Saláriu mensál na ECV

    def apresenta(self):
        return f"{self.nomi}, {self.idadi} anus"

    def saláriu_anuál(self):
        """Kalkula saláriu anuál (12 mezis + subsídiu di férias)"""
        return self.saláriu * 13  # 12 mezis + 1 subsídiu


class Profesór(Trabadjor):
    """Profesór — erda di Trabadjor"""
    def __init__(self, nomi, idadi, saláriu, matéria, skola):
        super().__init__(nomi, idadi, saláriu)
        self.matéria = matéria
        self.skola = skola
        self.alúnus = []

    def adisiona_alúnu(self, nomi_alúnu):
        self.alúnus.append(nomi_alúnu)

    def apresenta(self):
        bazi = super().apresenta()
        return f"{bazi} — Profesór di {self.matéria} na {self.skola}"


class Médiku(Trabadjor):
    """Médiku — erda di Trabadjor"""
    def __init__(self, nomi, idadi, saláriu, espesialidadi, ospitál):
        super().__init__(nomi, idadi, saláriu)
        self.espesialidadi = espesialidadi
        self.ospitál = ospitál
        self.pasiéntis_oji = 0

    def atende_pasiénti(self):
        self.pasiéntis_oji += 1
        print(f"🏥 Dr. {self.nomi} ta atende pasiénti #{self.pasiéntis_oji}")

    def apresenta(self):
        bazi = super().apresenta()
        return f"{bazi} — Médiku di {self.espesialidadi} na {self.ospitál}"


class Enjenheru(Trabadjor):
    """Enjenheru — erda di Trabadjor"""
    def __init__(self, nomi, idadi, saláriu, área, enpresa):
        super().__init__(nomi, idadi, saláriu)
        self.área = área
        self.enpresa = enpresa

    def apresenta(self):
        bazi = super().apresenta()
        return f"{bazi} — Enjenheru di {self.área} na {self.enpresa}"
```

```python
# Kria trabadjoris
prof = Profesór("Amilcar", 45, 65000, "Matimátika", "Liséu Domingos Ramos")
prof.adisiona_alúnu("Ana")
prof.adisiona_alúnu("Kelvin")
prof.adisiona_alúnu("Edna")

med = Médiku("Carla", 38, 120000, "Pediatria", "Hospital Agostinho Neto")
med.atende_pasiénti()   # 🏥 Dr. Carla ta atende pasiénti #1
med.atende_pasiénti()   # 🏥 Dr. Carla ta atende pasiénti #2

enj = Enjenheru("Nelson", 32, 90000, "Software", "Skola.dev")

# Tudu é Trabadjor!
trabadjoris = [prof, med, enj]

print("\n📋 Ekipa:")
for t in trabadjoris:
    print(f"  {t.apresenta()}")
    print(f"    Saláriu anuál: {t.saláriu_anuál():,} ECV")

# 📋 Ekipa:
#   Amilcar, 45 anus — Profesór di Matimátika na Liséu Domingos Ramos
#     Saláriu anuál: 845,000 ECV
#   Carla, 38 anus — Médiku di Pediatria na Hospital Agostinho Neto
#     Saláriu anuál: 1,560,000 ECV
#   Nelson, 32 anus — Enjenheru di Software na Skola.dev
#     Saláriu anuál: 1,170,000 ECV
```

Kada klasi filhu ta erda `saláriu_anuál()` sen repeti-l, i kada un ta **sobreskrebe** `apresenta()` pa mostra o ki é spesífiku di el. Isu é eransa simples na prátika: reuza o ki é komun, personaliza o ki é diferenti.

:::callout{type=note}
Eransa inda tene más: **eransa múltiplu** (un klasi ku vários pai), **MRO** (orden di buska di metodus) i `isinstance()`. Pa gosi, foka na eransa di **un só** pai ku `super()` — dipos nu ta ba más lonji.
:::
