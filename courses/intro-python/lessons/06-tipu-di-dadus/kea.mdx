# Tipu di Dadus

Na mundu real, nôs ta lida ku diferentes tipus di informasun: númerus, textus, sim/nun. Python tanbé tene diferentes **data types** pa representa kada tipu di informasun.

Nês lisun, nôs ta esplora 4 tipus fundamentais di Python — `int`, `float`, `str` e `bool` — e aprendi kuandu uza kada un e kumu evita errus kuandu mistra es.

## 4 Tipus Fundamentais

### Integer (`int`) — Númerus Interus

Númerus sen puntu desimal: positivu, negativu ó zeru.

```python
# Ezemplus di int
populasun = 600000
temperatura = -5
anu = 2026
zero = 0

print(type(populasun))  # <class 'int'>
```

Undi uza `int`:
- Kontadoris (kantidadi di estudantis, anus, pájinas)
- Idadis
- Indexis (posisun na un lista)
- Kualker númeru ki ka presiza puntu desimal

```python
# Python ta lida ku númerus mutu grandi sen problema
numeru_grandi = 1_000_000_000  # Underscore pa lejilidadi
print(numeru_grandi)           # 1000000000
print(type(numeru_grandi))     # <class 'int'>
```

> **💡 Dika:** Bo podi uza underscore (`_`) dentu di númerus grandi pa lejilidadi: `1_000_000` é mesmu ki `1000000`.

### Float — Númerus Desimais

Númerus ku puntu desimal.

```python
# Ezemplus di float
presu_cachupa = 350.50
altura = 1.75
pi = 3.14159
temperatura = -2.5

print(type(presu_cachupa))  # <class 'float'>
```

Undi uza `float`:
- Presus e valores monetárius
- Medidas (altura, pezu, distánsia)
- Resultadus di divisun
- Kualker valor ki presiza prezisun desimal

```python
# ATENSUN: Divisun senpri ta retorna float!
resultado = 10 / 5
print(resultado)       # 2.0 (ka 2!)
print(type(resultado)) # <class 'float'>

# Mesmu kuandu resultado podi ser interu
resultado = 100 / 10
print(resultado)       # 10.0
print(type(resultado)) # <class 'float'>
```

**Kuidadu ku prezisun di float:**

```python
# Float tene limitasoens di prezisun
print(0.1 + 0.2)  # 0.30000000000000004 (ka 0.3 ezatu!)
```

Kel é un limitasun di kumu komputador ta armazena númerus desimais. Pa maioria di kazus, ka é problema. Pa valores monetárius ezatus, ezisti módulu `decimal`, ma kel é avanasadu.

### String (`str`) — Textu

Textu dentu di aspas (simples ó dublu).

```python
# Ezemplus di str
nomi = "Cesária Évora"
ilha = 'São Vicente'
mensajen = "Morabeza é alma di Kabu Verdi"
vazio = ""  # String vazio — tanbé é válidu

print(type(nomi))  # <class 'str'>
```

Bo podi uza aspas simples (`'`) ó dublu (`"`) — resultado é mesmu:

```python
# Tudu kel é válidu
greeting1 = "Ola!"
greeting2 = 'Ola!'
print(greeting1 == greeting2)  # True

# Sabi pa kuandu textu tene aspas dentu
frazi = "Funana é músika di 'batuku' kabu-verdianu"
frazi2 = 'El dizi "morabeza" pa tudu pesoa'
```

Strings tene propriedadis spesiais:

```python
nomi = "Djina"

# Komprimantu
print(len(nomi))  # 5

# Konkatenasun (djunta strings)
saudason = "Ola, " + nomi + "!"
print(saudason)  # Ola, Djina!

# Repetisun
print("-" * 30)  # ------------------------------

# f-string (melhor manera di mistra textu ku variáveis)
idadi = 20
print(f"{nomi} tene {idadi} anus")  # Djina tene 20 anus
```

> **💡 Dika:** Senpri uza **f-strings** pa mistra textu ku variáveis. É más limpu ki konkatenasun ku `+`.

### Boolean (`bool`) — Verdaderu ó Falsu

Só tene 2 valores: `True` ó `False` (ku T e F maiúskulu!).

```python
# Ezemplus di bool
é_estudanti = True
ten_passaporti = False
ta_xobi = True

print(type(é_estudanti))  # <class 'bool'>
```

Booleans ta sai naturalmenti di **komparasoens**:

```python
idadi = 20
print(idadi >= 18)      # True
print(idadi == 15)      # False
print(idadi != 20)      # False

nomi = "Pedro"
print(nomi == "Pedro")  # True
print(nomi == "pedro")  # False (case sensitive!)

presu = 350.50
print(presu > 300)      # True
print(presu < 100)      # False
```

```python
# Bool na kondisionais
nota = 85
pasou = nota >= 70

if pasou:
    print("Parabéns! Bo pasa!")
else:
    print("Bo tene ki studia más...")
```

## Kuandu Uza Kada Tipu

| Situasun | Tipu | Ezemplu |
|----------|------|---------|
| Kontador, kantidadi, idadi | `int` | `estudantis = 45` |
| Presu, medida, persentajen | `float` | `presu = 149.99` |
| Nomi, mensajen, textu | `str` | `nomi = "Ana"` |
| Sim/nun, ativu/inativu | `bool` | `ativu = True` |

```python
# Ezemplu prátiku: Rejistru di un estudanti
nomi = "Kelvin"           # str
idadi = 21                # int
nota_media = 82.5         # float
é_bolsista = True         # bool
ilha = "Fogo"             # str
kreditusECV = 15000       # int

print(f"Estudanti: {nomi}")
print(f"Idadi: {idadi} anus")
print(f"Nota média: {nota_media}")
print(f"Bolsista: {'Sim' if é_bolsista else 'Nun'}")
print(f"Ilha: {ilha}")
print(f"Kréditus: {kreditusECV} ECV")
```

## TypeError: Kuandu Tipus Ka Ta Kombina

Un di errus más komuns pa prinsipianti é **TypeError** — kuandu bo tenta fazi un operasun ku tipus inkompatíveis.

### Erru #1: Soma di String ku Númeru

```python
nomi = "Maria"
idadi = 25

# ERADU — ka podi soma string ku int
# mensajen = nomi + " tene " + idadi + " anus"  # TypeError!

# SOLUSON 1: Konverti int pa string
mensajen = nomi + " tene " + str(idadi) + " anus"
print(mensajen)  # Maria tene 25 anus

# SOLUSON 2 (MELHOR): Uza f-string
mensajen = f"{nomi} tene {idadi} anus"
print(mensajen)  # Maria tene 25 anus
```

### Erru #2: Operasun Matemátika ku String

```python
presu = "350"    # É string, ka int!
taxa = 0.15

# ERADU
# total = presu * taxa  # TypeError: can't multiply str by float

# KORETU — konverti primeiru
total = float(presu) * taxa
print(f"Taxa: {total} ECV")  # Taxa: 52.5 ECV
```

### Erru #3: Komparasun di Tipus Diferentes

```python
# String "5" ka é mesmu ki int 5
print("5" == 5)     # False
print(type("5"))    # <class 'str'>
print(type(5))      # <class 'int'>

# Pa kompara, konverti pa mesmu tipu
print(int("5") == 5)  # True
```

> **💡 Dika:** Kuandu bo enkontra `TypeError`, verifica tipus ku `type()`. Kuazi senpri, soluson é konverti un valor ku `int()`, `float()` ó `str()`.

## Tabela Rezumu di Konversun

```python
# int() — pa interu
print(int(3.14))       # 3 (trunca, ka arredonda!)
print(int("42"))       # 42
print(int(True))       # 1
print(int(False))      # 0

# float() — pa desimal
print(float(42))       # 42.0
print(float("3.14"))   # 3.14
print(float(True))     # 1.0

# str() — pa textu
print(str(42))         # "42"
print(str(3.14))       # "3.14"
print(str(True))       # "True"

# bool() — pa boolean
print(bool(0))         # False
print(bool(42))        # True
print(bool(""))        # False (string vazio)
print(bool("Ola"))     # True (kualker string ku konteúdu)
```

> **💡 Dika:** `int()` ta **trunca** (korta parti desimal), ka ta arredonda. `int(3.9)` da `3`, ka `4`! Pa arredonda, uza `round(3.9)` ki da `4`.

## Ezemplu Prátiku: Loja di Cachupa

Vamos kombina tudu tipus na un programa real:

```python
# === Loja di Cachupa di Nha Cesária ===
print("=" * 35)
print("  Loja di Cachupa di Nha Cesária")
print("=" * 35)
print()

# Dadus di pratu (diferentes tipus)
nomi_pratu = "Cachupa Rica"           # str
presu_unitáriu = 450.00               # float
porsun_disponível = 12                # int
é_spesialidadi = True                 # bool

# Mostra menu
print(f"Pratu: {nomi_pratu}")
print(f"Presu: {presu_unitáriu} ECV")
print(f"Disponível: {porsun_disponível} porsoens")
print(f"Spesialidadi di kaza: {'Sim' if é_spesialidadi else 'Nun'}")
print()

# Resebe pedidu
kantidadi = int(input("Kantu porsun bo kre? "))

# Kalkula
subtotal = presu_unitáriu * kantidadi
taxa = subtotal * 0.15
total = subtotal + taxa
porsun_resta = porsun_disponível - kantidadi

# Mostra rezumu
print()
print("--- Rezumu di Pedidu ---")
print(f"Pratu: {nomi_pratu} x{kantidadi}")
print(f"Subtotal: {subtotal:.2f} ECV")
print(f"Taxa (15%): {taxa:.2f} ECV")
print(f"Total: {total:.2f} ECV")
print(f"Porsoens ki resta: {porsun_resta}")
```

Kuandu bo roda:
```
===================================
  Loja di Cachupa di Nha Cesária
===================================

Pratu: Cachupa Rica
Presu: 450.0 ECV
Disponível: 12 porsoens
Spesialidadi di kaza: Sim

Kantu porsun bo kre? 3

--- Rezumu di Pedidu ---
Pratu: Cachupa Rica x3
Subtotal: 1350.00 ECV
Taxa (15%): 202.50 ECV
Total: 1552.50 ECV
Porsoens ki resta: 9
```

## Verifikasun Rápida di Tipus

Kel é un truqui útil — uza `isinstance()` pa verifica si un valor é di un sertu tipu:

```python
valor = 42

print(isinstance(valor, int))     # True
print(isinstance(valor, float))   # False
print(isinstance(valor, str))     # False

# Verifica vários tipus di un vez
print(isinstance(valor, (int, float)))  # True — é int ó float
```

> **💡 Dika:** `isinstance()` é más profisional ki `type() ==`. Pa agora, `type()` ta sirvi, ma lenbra `isinstance()` pa futuru.

## Rezumu

- **`int`** — númerus interus (idadi, kontador, anu)
- **`float`** — númerus desimais (presu, altura, nota)
- **`str`** — textu (nomi, mensajen, ilha)
- **`bool`** — `True` ó `False` (komparasoens, kondisun)
- **TypeError** ta aparese kuandu bo mistra tipus inkompatíveis
- Uza **`int()`**, **`float()`**, **`str()`** pa konverti entre tipus
- **f-strings** é melhor manera di mistra textu ku variáveis — evita TypeError
- Divisun (`/`) senpri ta retorna **float**, mesmu ku númerus interus
