# Funsons Inkorporadus Úteis

Python tene muitísimas funsons **inkorporadus** (built-in) — funsons ki djá ben ku linguajen, sen presiza importa nada. Bo djá konxe `print()`, `len()`, `type()`, `int()`, `str()`. Na es lisan, nos ta aprende sinku funsons más ki é ESSENSIAL pa skrebe kódiku Pythonic — kódiku ki é limpu, lijível, i efisienti.

Es funsons ta muda manera ki bu ta itera sobri dadus. Dipos di es lisan, bo nunka más ta skrebe `for i in range(len(lista))` — promesa!

## `enumerate()` — Itera ku Índisi

Kuandu bo presiza índisi i valor na mesmu tempu, `enumerate()` é maneira koretu di faze na Python.

### Problema: Maneira "Ka Pythonic"

```python
# ❌ Ka Pythonic — range(len()) é feiu i verbózu
ilhas = ["Santiago", "São Vicente", "Sal", "Fogo", "Santo Antão"]

for i in range(len(ilhas)):
    print(f"{i + 1}. {ilhas[i]}")
```

### Soluson: `enumerate()`

```python
# ✅ Pythonic — limpu i lijível!
ilhas = ["Santiago", "São Vicente", "Sal", "Fogo", "Santo Antão"]

for indisi, ilha in enumerate(ilhas):
    print(f"{indisi + 1}. {ilha}")
# 1. Santiago
# 2. São Vicente
# 3. Sal
# 4. Fogo
# 5. Santo Antão
```

### Parámetru `start`

Bu pode komesa kontajen di kualker nùmeru:

```python
# Komesa di 1 en bez di 0
for num, ilha in enumerate(ilhas, start=1):
    print(f"{num}. {ilha}")
# 1. Santiago
# 2. São Vicente
# ...
```

### Ezemplus Prátikus

```python
# Enkontra pozisan di un elementu
notas = [14, 18, 9, 16, 11, 7, 15]

print("Studantis ki xunba (nota < 10):")
for pozisan, nota in enumerate(notas, start=1):
    if nota < 10:
        print(f"  Studanti #{pozisan}: nota {nota}")
# Studanti #3: nota 9
# Studanti #6: nota 7
```

```python
# Menu interativu
pratus = ["Katxupa", "Buzio grelhadu", "Lagosta", "Kaldeirada", "Pastel ku diabolus"]

print("=== Menu di Oji ===")
for num, pratu in enumerate(pratus, start=1):
    print(f"  {num}. {pratu}")
print("==================")

# Uza eskólha
eskólha = 3
print(f"\nBo skolhe: {pratus[eskólha - 1]}")  # Lagosta
```

```python
# Prosesamentu di ficheru (linha pa linha ku nùmeru)
linhas = ["# Títulu", "", "Primeiru parágrafu.", "Segundu parágrafu."]

for num_linha, linha in enumerate(linhas, start=1):
    if linha.strip():  # ignora linhas vazias
        print(f"L{num_linha}: {linha}")
# L1: # Títulu
# L3: Primeiru parágrafu.
# L4: Segundu parágrafu.
```

:::callout{type=tip}
Senpri ki bo pensa na skrebe `for i in range(len(lista))`, PARA i uza `enumerate()` en bez. É más Pythonic, más lijível, i menus propensu a eru.
:::

## `zip()` — Itera na Paralelu

`zip()` ta kombina dos ou más iteráveis, elementu pa elementu, kumo un zipper (féxu) di ropa.

```python
# Kombina nomis ku notas
nomis = ["Maria", "João", "Ana", "Pedro"]
notas = [16, 14, 19, 12]

for nomi, nota in zip(nomis, notas):
    print(f"{nomi}: {nota}")
# Maria: 16
# João: 14
# Ana: 19
# Pedro: 12
```

```python
# Kombina tres listas
studantis = ["Kelvin", "Edna", "Nilton", "Carla"]
notas_teste1 = [15, 18, 12, 16]
notas_teste2 = [17, 16, 14, 19]

print("=== Rezumu di Notas ===")
for nomi, n1, n2 in zip(studantis, notas_teste1, notas_teste2):
    media = (n1 + n2) / 2
    print(f"{nomi}: T1={n1}, T2={n2}, Média={media:.1f}")
# Kelvin: T1=15, T2=17, Média=16.0
# Edna: T1=18, T2=16, Média=17.0
# Nilton: T1=12, T2=14, Média=13.0
# Carla: T1=16, T2=19, Média=17.5
```

### Kria Disionárius ku `zip()`

```python
# Kombina xaves i valoris pa kria dict
ilhas = ["Santiago", "São Vicente", "Sal", "Fogo"]
kapitais = ["Praia", "Mindelo", "Espargos", "São Filipe"]

ilha_kapital = dict(zip(ilhas, kapitais))
print(ilha_kapital)
# {'Santiago': 'Praia', 'São Vicente': 'Mindelo', 'Sal': 'Espargos', 'Fogo': 'São Filipe'}
```

```python
# Kria lista di disionárius
nomis = ["Gil", "Lara", "Amilcar"]
idadis = [25, 30, 28]
profisóens = ["Dezenvolvedor", "Dizainera", "Profesór"]

pesoas = [
    {"nomi": n, "idadi": i, "profisan": p}
    for n, i, p in zip(nomis, idadis, profisóens)
]
for p in pesoas:
    print(p)
# {'nomi': 'Gil', 'idadi': 25, 'profisan': 'Dezenvolvedor'}
# {'nomi': 'Lara', 'idadi': 30, 'profisan': 'Dizainera'}
# {'nomi': 'Amilcar', 'idadi': 28, 'profisan': 'Profesór'}
```

### `zip()` Para na Más Kurtu

Si listas tene tamánhu diferenti, zip() para kuandu más kurtu akaba:

```python
nomis = ["Maria", "João", "Ana", "Pedro", "Carla"]
notas = [16, 14, 19]  # só 3 notas!

for nomi, nota in zip(nomis, notas):
    print(f"{nomi}: {nota}")
# Maria: 16
# João: 14
# Ana: 19
# (Pedro i Carla ka aparese!)
```

:::callout{type=tip}
Si bu kre ki `zip()` kontinua até más longu (prixenxe ku un valor predefinidu), uza `itertools.zip_longest()` di módulu `itertools`.
:::

## `sorted()` vs `.sort()`

Ambos ta ordena, ma di maneira diferenti:

| | `sorted(lista)` | `lista.sort()` |
|---|---|---|
| Retorna | **Nova lista** ordenadu | `None` (modifika original) |
| Original | Ka muda | Muda! |
| Funsiona na | Kualker iterável | Só listas |

```python
# sorted() — kria NOVA lista, original ka muda
numeros = [5, 2, 8, 1, 9, 3]

ordenadu = sorted(numeros)
print(ordenadu)   # [1, 2, 3, 5, 8, 9]
print(numeros)    # [5, 2, 8, 1, 9, 3] ← ka mudá!

# .sort() — MUDA lista original
numeros.sort()
print(numeros)    # [1, 2, 3, 5, 8, 9] ← mudá!
```

```python
# Ordena na orden inversu
presus = [350, 1200, 80, 5000, 750]

# Di más karu pa más baratu
print(sorted(presus, reverse=True))  # [5000, 1200, 750, 350, 80]
```

```python
# Ordena ku key — ordena pa un kritériu espesífiku
palavras = ["katxupa", "morabeza", "sol", "funana", "grogu"]

# Pa tamánhu di palavra
print(sorted(palavras, key=len))
# ['sol', 'grogu', 'funana', 'katxupa', 'morabeza']

# Alfabetikamenti (sen importa maiúskulas)
misturadu = ["Praia", "mindelo", "SAL", "fogo"]
print(sorted(misturadu, key=str.lower))
# ['fogo', 'mindelo', 'Praia', 'SAL']
```

```python
# Ordena lista di disionárius
studantis = [
    {"nomi": "Djina", "nota": 17},
    {"nomi": "Nuno", "nota": 12},
    {"nomi": "Cesária", "nota": 19},
    {"nomi": "Kelvin", "nota": 15},
]

# Pa nota (más altu primeiru)
ranking = sorted(studantis, key=lambda s: s["nota"], reverse=True)
for pos, s in enumerate(ranking, start=1):
    print(f"  {pos}° {s['nomi']}: {s['nota']}")
# 1° Cesária: 19
# 2° Djina: 17
# 3° Kelvin: 15
# 4° Nuno: 12
```

:::callout{type=tip}
Uza `sorted()` kuandu bu kre mantene lista original intaktu. Uza `.sort()` kuandu bu ka presiza original i kre evita kria kópia (más efizienti na memória).
:::

## `any()` i `all()` — Agregasan Boolean

Es funsons ta verifika kondisons sobri un iterável interu:

- **`any(iterável)`** — retorna `True` si **ALGUN** elementu é True (kumo `or`)
- **`all(iterável)`** — retorna `True` si **TUDU** elementu é True (kumo `and`)

```python
# any() — algun é verdaderu?
notas = [8, 12, 6, 15, 9]

algun_aprova = any(nota >= 10 for nota in notas)
print(f"Algun studanti aprová? {algun_aprova}")  # True

tudu_aprova = all(nota >= 10 for nota in notas)
print(f"Tudu studanti aprová? {tudu_aprova}")    # False
```

```python
# Verifica si lista tene algun nùmeru negativu
temperaturas = [28, 30, 25, 32, 27]
tene_negativu = any(t < 0 for t in temperaturas)
print(f"Tene temperatura negativa? {tene_negativu}")  # False

# Verifica si tudu presus é pozitivu
presus = [500, 1200, 350, -100, 800]
tudu_pozitivu = all(p > 0 for p in presus)
print(f"Tudu presu é pozitivu? {tudu_pozitivu}")  # False
```

```python
# Validasan di formuláriu
def formuláriu_válidu(dados):
    """Verifica si tudu kampu obrigatóriu ta prixenxidu."""
    kampus_obrigatóriu = ["nomi", "email", "ilha"]
    return all(dados.get(k) for k in kampus_obrigatóriu)

# Testa
dados1 = {"nomi": "Ana", "email": "ana@email.cv", "ilha": "Sal"}
dados2 = {"nomi": "Pedro", "email": "", "ilha": "Fogo"}

print(formuláriu_válidu(dados1))  # True (tudu prixenxidu)
print(formuláriu_válidu(dados2))  # False (email ta vazíu)
```

```python
# Verifikador di senha kombinu ku any/all
def verifica_senha(senha):
    """Verifica rekezitus di senha."""
    rezitus = {
        "8+ karáteris": len(senha) >= 8,
        "Tene maiúskula": any(c.isupper() for c in senha),
        "Tene minúskula": any(c.islower() for c in senha),
        "Tene nùmeru": any(c.isdigit() for c in senha),
        "Tene spesiál": any(c in "!@#$%&*_+-" for c in senha),
    }

    print("Rezitus di senha:")
    for rekezitu, pasa in rezitus.items():
        status = "✓" if pasa else "✗"
        print(f"  {status} {rekezitu}")

    return all(rezitus.values())

é_forti = verifica_senha("Pr@ia2026")
print(f"\nSenha forti? {é_forti}")
# Rezitus di senha:
#   ✓ 8+ karáteris
#   ✓ Tene maiúskula
#   ✓ Tene minúskula
#   ✓ Tene nùmeru
#   ✓ Tene spesiál
# Senha forti? True
```

## `isinstance()` — Verifika Tipu

`isinstance(objetu, tipu)` verifika si un objetu é di un sertu tipu. É más koretu ki `type(x) == int`.

```python
# Verifikasan báziku
print(isinstance(42, int))          # True
print(isinstance(3.14, float))      # True
print(isinstance("Praia", str))     # True
print(isinstance([1, 2], list))     # True
print(isinstance(True, bool))       # True
```

```python
# isinstance() vs type()
# Na Python, un bool ta konta sima un númeru interu
print(isinstance(True, int))    # True! (True ta funsiona sima 1, False sima 0)
print(type(True) == int)        # False (type() é striitu)
```

```python
# Verifica vários tipus di un bez
def prosesa_valor(valor):
    """Prosesa valor dependendu di si tipu."""
    if isinstance(valor, (int, float)):
        return f"Nùmeru: {valor * 2}"
    elif isinstance(valor, str):
        return f"Tekstu: {valor.upper()}"
    elif isinstance(valor, list):
        return f"Lista ku {len(valor)} elementu(s)"
    else:
        return f"Tipu deskonxedu: {type(valor).__name__}"

print(prosesa_valor(25))           # Nùmeru: 50
print(prosesa_valor("morabeza"))   # Tekstu: MORABEZA
print(prosesa_valor([1, 2, 3]))    # Lista ku 3 elementu(s)
```

```python
# Funsan ku validasan di tipu
def soma_seguru(*valores):
    """Soma SÓ nùmerus, ignora restu."""
    total = 0
    pa_ignora = []
    for v in valores:
        if isinstance(v, (int, float)):
            total += v
        else:
            pa_ignora.append(v)

    if pa_ignora:
        print(f"  (Ignoradu: {pa_ignora})")
    return total

rezultadu = soma_seguru(10, "oi", 20, [1,2], 30, "test")
# (Ignoradu: ['oi', [1, 2], 'test'])
print(f"Total: {rezultadu}")  # Total: 60
```

## Kombina Tudu — Izemplu Kompletu

```python
# Sistema di klasifikasan di studantis
studantis = ["Maria", "João", "Ana", "Pedro", "Carla", "Gil"]
notas_final = [17, 11, 19, 8, 14, 6]

# 1. Kombina nomis ku notas (zip)
# 2. Numera rezultadus (enumerate)
# 3. Klasifika kada studanti
print("=== Rezultadu Final ===\n")

for pos, (nomi, nota) in enumerate(zip(studantis, notas_final), start=1):
    if nota >= 16:
        status = "Distintu ⭐"
    elif nota >= 10:
        status = "Aprovadu ✓"
    else:
        status = "Xunbadu ✗"
    print(f"  {pos}. {nomi}: {nota} — {status}")

# Estatístikas
print(f"\nTudu aprová? {all(n >= 10 for n in notas_final)}")
print(f"Algun ku distintu? {any(n >= 16 for n in notas_final)}")
print(f"Média: {sum(notas_final) / len(notas_final):.1f}")

# Ranking (sorted)
ranking = sorted(zip(studantis, notas_final), key=lambda x: x[1], reverse=True)
print("\n=== Ranking ===")
for pos, (nomi, nota) in enumerate(ranking, start=1):
    print(f"  {pos}° {nomi}: {nota}")
```
