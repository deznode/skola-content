# Ler Erus (Tracebacks)

Tudu programador ta atxa erus — dezdi ki ta kumesa até ki ta vira espesialista. Diferensa é ki un programador sperienti **sabi ler** mensajen di eru i rezolve rápidu. Na es lisan, nu ta prende a ler tracebacks di Python manera un detetivi — di baxu pa riba — pa diskubri exatamenti o ki koreu mal i undi.

Erus ka é sinal di frakasu — é sinal di ki bu ta tenta. Nu ta prende djuntu, i erus é parti di prende.

:::callout{type=tip}
**Es lisan na 3 palavra:** **Ler** → **Atxa** → **Pensa**. Bu ta odja kel padran na fin di lisan. Pa gosi, fixa atensan na es 3 pasus enkuantu nu ta ler izemplus.
:::

## O ki é un Traceback?

Kuandu Python atxa un problema na bo kódiku, el ta para i mostra un **traceback** — un mensajen ki ta splika o ki koreu mal i undi. Odja es izemplu:

:::callout{type=tip}
**Ka preokupa si bu ka kompriende tudu kódiku gosi.** Lisan-li é so pa ler **mensajen di eru** — `def`, `return`, `f"..."` i otu kunseitus ta ben más tardi (Módulu 3 i Módulu 1 Lisan 7). Repara na traceback, ka na kódiku.
:::

```python
# fixeru: kalkula.py
def kalkula_media(notas):
    total = sum(notas)
    media = total / len(notas)
    return media

notas_di_ana = []
rezultadu = kalkula_media(notas_di_ana)
print(f"Média: {rezultadu}")
```

Kuandu bo roda es kódiku, Python ta mostra:

```
Traceback (most recent call last):
  File "kalkula.py", line 7, in <module>
    rezultadu = kalkula_media(notas_di_ana)
                ~~~~~~~~~~~~~~~~~~~~~~~~~~~
  File "kalkula.py", line 3, in kalkula_media
    media = total / len(notas)
            ~~~~~~^~~~~~~~~~~~
ZeroDivisionError: division by zero
```

Es parsesi komplikadu, ma ka é! Nu ta prende ler el pasu a pasu.

## Regra di Oru: Ler di Baxu pa Riba

Segridu más importanti: **senpri kumesa ler di última linha**. É la ki resposta ta stá.

### Pasu 1: Última Linha — O ki koreu mal

```
ZeroDivisionError: division by zero
```

Es linha tene **dos parti**:
- **`ZeroDivisionError`** — tipu di eru (nomi di exception)
- **`division by zero`** — mensajen ki ta splika problema

Tradusan: "Bu ta tenta dividi pa zero — Python ka pode faze kel."

### Pasu 2: Linhas di Riba — Undi koreu mal

```
  File "kalkula.py", line 3, in kalkula_media
    media = total / len(notas)
            ~~~~~~^~~~~~~~~~~~
```

Es ta dize-bu:
- **Fixeru:** `kalkula.py`
- **Linha:** 3
- **Funsan:** `kalkula_media`
- **Kódiku:** `media = total / len(notas)`
- **Seta `^`:** aponta direitamenti pa `/` — é operasan ki faya!

### Pasu 3: Subi más — Kemi txoma kemi

```
  File "kalkula.py", line 7, in <module>
    rezultadu = kalkula_media(notas_di_ana)
```

Es ta mostra ki linha 7 é undi bo txoma funsan ki faya. `<module>` signifika ki ta na nível prinsipál di programa (ka dentu di un funsan).

:::callout{type=tip}
**Dika:** Pensa na traceback manera un pilha di pratus — última pratu (última linha) é undi pratu kebra. Ma bu presiza odja pratus di baxu pa kompriende pamodi tudu kai.
:::

## Seta `^` — Bo Midjór Amigu (Python 3.11+)

Dezdi Python 3.11, erus tene un **seta `^`** ki ta aponta exatamenti pa parti di kódiku ki ta kauza problema. Es é un melhoria enormi!

```python
# Ezemplu: eru na espresan kompleksu
presu = {"katxupa": 350, "grogu": 200}
total = presu["katxupa"] + presu["ponche"]
```

```
Traceback (most recent call last):
  File "loja.py", line 3, in <module>
    total = presu["katxupa"] + presu["ponche"]
                                ~~~~~^^^^^^^^^^
KeyError: 'ponche'
```

Odja undi `^` ta aponta: `presu["ponche"]` — é exatamenti kel parti ki ta kauza eru! "ponche" ka ezisti na disionáriu `presu`.

Más un izemplu:

```python
nomis = ["Maria", "João", "Ana"]
print(nomis[5])
```

```
Traceback (most recent call last):
  File "nomis.py", line 2, in <module>
    print(nomis[5])
          ~~~~~^^^
IndexError: list index out of range
```

Seta `^` ta mostra ki `nomis[5]` é problema — lista so tene 3 elementus (índisi 0, 1, 2), ma bu ta pidi índisi 5.

## 5 Exceptions Más Kumun

Es é erus ki bu ta atxa más frekventi kuandu bu ta kumesa. Konxe kes i bu ta rezolve problemus mutu más rápidu.

### 1. `SyntaxError` — Eru di Sintaxi

Python ka kompriende o ki bo skrebe. É manera un frazi ku gramátika eradu.

```python
# Skesi dos pontus (:) después di if
idadi = 20
if idadi > 18
    print("Adultu")
```

```
  File "testa.py", line 3
    if idadi > 18
                 ^
SyntaxError: expected ':'
```

**Kauzas kumun:**
- Skesi `:` dipos di `if`, `for`, `while`, `def`, `class`
- Skesi fexá paréntesis `)`, kólxeti `]`, o xavi `}`
- Uza `=` (atribuisan) na lugar di `==` (komparasan) na kondisionais

:::callout{type=tip}
**Dika:** `SyntaxError` é úniku eru ki Python ta atxa **antis** di roda programa. Tudu otu erus ta aparesi kuandu programa ta roda.
:::

### 2. `NameError` — Nomi ka Ezisti

Bu ta uza un variável ki ka foi kriadu inda.

```python
print(mensajen)
```

```
Traceback (most recent call last):
  File "testa.py", line 1, in <module>
    print(mensajen)
          ^^^^^^^^
NameError: name 'mensajen' is not defined
```

**Kauzas kumun:**
- Variável ka foi atribuídu (skesi `mensajen = "Ola"`)
- Eru di ortografia: `mesnajen` na lugar di `mensajen`
- Uza variável fora di se scope (bu ta prende más na Lisan 16)

```python
# Korresan: defini variável antis di uza
mensajen = "Ola, mundu!"
print(mensajen)
```

### 3. `TypeError` — Tipu Eradu

Bu ta mistura tipus ki ka é kompatível.

```python
idadi = 25
mensajen = "Bo tene " + idadi + " anu"
```

```
Traceback (most recent call last):
  File "testa.py", line 2, in <module>
    mensajen = "Bo tene " + idadi + " anu"
               ~~~~~~~~~~~~^~~~~~
TypeError: can only concatenate str (not "int") to str
```

Seta `^` ta mostra exatamenti undi string ta bate ku int!

**Kauzas kumun:**
- Djunta string ku int: `"Ola" + 5` (uza `"Ola" + str(5)` o f-string)
- Txoma algu ki ka é funsan: `x = 5; x()` (TypeError: 'int' is not callable)
- Manda argumentu eradu pa funsan

*(Funsan: bu ta prende más na Lisan 16.)*

```python
# Korresan: uza f-string
idadi = 25
mensajen = f"Bo tene {idadi} anu"
print(mensajen)  # Bo tene 25 anu
```

### 4. `IndexError` — Índisi Fora di Rangu

Bu ta tenta asesa un pozisan ki ka ezisti na lista.

```python
ilhas = ["Santiago", "São Vicente", "Sal"]
print(ilhas[3])
```

```
Traceback (most recent call last):
  File "testa.py", line 2, in <module>
    print(ilhas[3])
          ~~~~~^^^
IndexError: list index out of range
```

Lista tene 3 elementus (índisi 0, 1, 2), ma bu ta pidi índisi 3.

**Kauzas kumun:**
- Skesi ki índisi ta kumesa na **0** (ka na 1)
- Uza `len(lista)` kumo índisi (último índisi válidu é `len(lista) - 1`)
- Lista váziu i bu ta tenta asesa kualker índisi

```python
# Korresan: verifika tamañu antis
ilhas = ["Santiago", "São Vicente", "Sal"]
indisi = 3
if indisi < len(ilhas):
    print(ilhas[indisi])
else:
    print(f"Índisi {indisi} ka ezisti! Lista so tene {len(ilhas)} elementus.")
```

### 5. `KeyError` — Xavi ka Ezisti na Disionáriu

Bu ta buska un xavi ki ka ezisti na disionáriu.

```python
estudanti = {"nomi": "Kelvin", "idadi": 22}
print(estudanti["nota"])
```

```
Traceback (most recent call last):
  File "testa.py", line 2, in <module>
    print(estudanti["nota"])
          ~~~~~~~~~^^^^^^^^
KeyError: 'nota'
```

**Korresan: uza `.get()` pa evita eru**

```python
estudanti = {"nomi": "Kelvin", "idadi": 22}

# Manera seguru — retorna None si ka ezisti
nota = estudanti.get("nota")
print(nota)  # None

# Manera seguru ku valór padran
nota = estudanti.get("nota", "Sen nota")
print(nota)  # Sen nota
```

## Rezimu Vizual: 5 Exceptions

| Exception | Signifikadu | Izemplu Rápidu | Korresan Kumun |
|-----------|-------------|-----------------|----------------|
| `SyntaxError` | Gramátika eradu | `if x > 5` (falta `:`) | Djunta `:`, `)`, `]` ki falta |
| `NameError` | Variável ka defini | `print(x)` sen `x = ...` | Defini variável o verifika ortografia |
| `TypeError` | Tipus inkompatível | `"Ola" + 5` | Uza `str()`, `int()` o f-string |
| `IndexError` | Índisi fora di rangu | `lista[10]` ku 3 elementus | Verifika `len()` antis |
| `KeyError` | Xavi ka ezisti | `d["xavi"]` ki ka ezisti | Uza `.get(xavi, padran)` |

## Estratéjia di 3 Pasus pa Rezolvi Eru

Kuandu bo odja un eru, sigi senpri es 3 pasus:

```
1. LER  →  Última linha di traceback (tipu + mensajen)
2. ATXA →  Fixeru i número di linha (seta ^ mostra lokalizasan)
3. PENSA →  "Pamodi Python ta pensa ki es é un problema?"
```

Nu pratika ku un izemplu:

:::callout{type=tip}
**O ki es kódiku ta tenta faze:** Soma kuatu presu (350, 200, 150, 400) i kalkula média. Ka preokupa ku `for` i `range` — bu ta prende-s na Lisan 10. Repara na traceback.
:::

```python
# Programa ku eru — bo pode atxa?
lista_di_presus = [350, 200, 150, 400]
total = 0

for i in range(1, 5):
    total = total + lista_di_presus[i]

media = total / 4
print(f"Média di presu: {media} ECV")
```

```
Traceback (most recent call last):
  File "presus.py", line 5, in <module>
    total = total + lista_di_presus[i]
                    ~~~~~~~~~~~~~~~^^^
IndexError: list index out of range
```

Aplika 3 pasus:
1. **LER:** `IndexError: list index out of range` — índisi fora di rangu
2. **ATXA:** Linha 5, `lista_di_presus[i]` — seta mostra ki problema é na asesu a lista
3. **PENSA:** `range(1, 5)` ta da `1, 2, 3, 4`. Ma `lista_di_presus[4]` ka ezisti pamodi índisis ta bai di 0 a 3. Soluson: `range(len(lista_di_presus))` o `range(0, 4)`.

```python
# Korrejidu!
lista_di_presus = [350, 200, 150, 400]
total = 0

for i in range(len(lista_di_presus)):
    total = total + lista_di_presus[i]

media = total / len(lista_di_presus)
print(f"Média di presu: {media} ECV")  # Média di presu: 275.0 ECV
```

:::callout{type=tip}
**Dika:** Na futuru, bu ta prende ki `sum(lista_di_presus)` é manera más simples di soma un lista. Ma pa gosi, importanti é kompriende kumo ler erus!
:::
