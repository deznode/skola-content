# Ler Erus (Tracebacks)

Tudu programador ta atxa erus — dezdi ki ta kumesa até ki ta vira espesialista. Diferensa é ki un programador sperienti **sabe ler** mensajen di eru i rezolve rápidu. Na es lisan, nu ta prende ler tracebacks di Python manera un detetivi — di baxu pa riba — pa diskubri exatamenti o ki koreu mal i unde.

:::callout{type=info title="Es lisan na 3 palavra"}
**Ler** → **Atxa** → **Pensa**. Bu ta odja kel padran na fin di lisan. Pa gosi, fixa atensan na es 3 pasus enkuantu nu ta ler izemplus.
:::

## O ki é un Traceback?

<GlossaryText
  text="Kuandu Python atxa un problema na bo kódiku, el ta para i mostra un [[traceback]] — un mensajen ki ta splika o ki koreu mal i unde."
  terms={{

/>

Odja es izemplu:

:::callout{type=tip}
**Ka preokupa si bu ka kompriende tudu kódiku gosi.** Lisan-li é so pa ler **mensajen di eru** — `def`, `return`, `f"..."` i otu kunseitus ta ben más tardi (Módulu 3 i Módulu 1 Lisan 7). Repara na traceback, ka na kódiku.
:::

```python
# fixeru: kalkula.py
def kalkula_media(notas):
    total = sum(notas)
    media = total / len(notas)
    return media

notas_di_ana = []
rezultadu = kalkula_media(notas_di_ana)
print(f"Média: {rezultadu}")
```

## Seta `^` — Bu Midjór Amigu (Python 3.11+)

Dezdi Python 3.11, erus tene un **seta `^`** ki ta aponta exatamenti pa parti di kódiku ki ta kauza problema. Es ta midjora txeu kuandu bu ta faze debug!

```python
# Izemplu: eru na espresan kompleksu
presu = {"katxupa": 350, "grogu": 200}
total = presu["katxupa"] + presu["ponche"]
```

<TerminalBlock
  title="python loja.py"
  lines={[
    { type: "err", t: "Traceback (most recent call last):" },
    { type: "trace", t: '  File "loja.py", line 3, in <module>' },
    { type: "code", t: '    total = presu["katxupa"] + presu["ponche"]' },
    { type: "caret", t: "                                ~~~~~^^^^^^^^^^" },
    { type: "err", t: "KeyError: 'ponche'" },
  ]}
/>

Odja unde `^` ta aponta: `presu["ponche"]` — é exatamenti kel parti ki ta kauza eru! "ponche" ka ezisti na disionáriu `presu`.

Más un izemplu:

```python
nomis = ["Maria", "João", "Ana"]
print(nomis[5])
```

<SectionHeading variant="concept">5 Exceptions Más Kumun</SectionHeading>

Es é erus ki bu ta atxa más frekventi kuandu bu ta kumesa. Konxe kes i bu ta rezolve problemus mutu más rápidu.

<SectionHeading level={3} variant="concept">1. `SyntaxError` — Eru di Sintaxi</SectionHeading>

Python ka kompriende o ki bo skrebe. É manera un frazi ku gramátika eradu.

```python
# Skise ':' dipos di if
idadi = 20
if idadi > 18
    print("Adultu")
```

<TerminalBlock
  title="python testa.py"
  lines={[
    { type: "trace", t: '  File "testa.py", line 3' },
    { type: "code", t: "    if idadi > 18" },
    { type: "caret", t: "                 ^" },
    { type: "err", t: "SyntaxError: expected ':'" },
  ]}
/>

**Kauzas kumun:**
- Skise `:` dipos di `if`, `for`, `while`, `def`, `class`
- Skise fitxa paréntesis `)`, brackets `]`, o braces `}`
- Uza `=` (atribuisan) na lugar di `==` (komparasan) na kondisionais

:::callout{type=tip}
`SyntaxError` é úniku eru ki Python ta atxa **antis** di roda programa. Tudu otu erus ta aparesi kuandu programa ta roda.
:::

<SectionHeading level={3} variant="concept">2. `NameError` — Nomi ka Ezisti</SectionHeading>

Bu ta uza un variável ki ka kriadu inda.

```python
print(mensajen)
```

<TerminalBlock
  title="python testa.py"
  lines={[
    { type: "err", t: "Traceback (most recent call last):" },
    { type: "trace", t: '  File "testa.py", line 1, in <module>' },
    { type: "code", t: "    print(mensajen)" },
    { type: "caret", t: "          ^^^^^^^^" },
    { type: "err", t: "NameError: name 'mensajen' is not defined" },
  ]}
/>

**Kauzas kumun:**
- Variável ka foi atribuídu (skise `mensajen = "Ola"`)
- Eru di ortografia: `mesnajen` na lugar di `mensajen`
- Uza variável fora di se scope (bu ta prende más na Lisan 16)

```python
# Koresan: defini variável antis di uza
mensajen = "Ola, mundu!"
print(mensajen)
```

<SectionHeading level={3} variant="concept">3. `TypeError` — Tipu Eradu</SectionHeading>

Bu ta mistura tipus ki ka é kompatível.

```python
idadi = 25
mensajen = "Bu tene " + idadi + " anu"
```

<TerminalBlock
  title="python testa.py"
  lines={[
    { type: "err", t: "Traceback (most recent call last):" },
    { type: "trace", t: '  File "testa.py", line 2, in <module>' },
    { type: "code", t: '    mensajen = "Bu tene " + idadi + " anu"' },
    { type: "caret", t: "               ~~~~~~~~~~~~^~~~~~" },
    { type: "err", t: 'TypeError: can only concatenate str (not "int") to str' },
  ]}
/>

Seta `^` ta mostra exatamenti unde string ta bate ku int!

**Kauzas kumun:**
- Djunta string ku int: `"Ola" + 5` (uza `"Ola" + str(5)` o f-string)
- Txoma algu ki ka é funsan: `x = 5; x()` (TypeError: 'int' is not callable)
- Manda argumentu eradu pa funsan

*(Funsan: bu ta prende más na Lisan 16.)*

```python
# Koresan: uza f-string
idadi = 25
mensajen = f"Bu tene {idadi} anu"
print(mensajen)  # Bu tene 25 anu
```

<SectionHeading level={3} variant="concept">4. `IndexError` — Índisi Fora di Rangu</SectionHeading>

Bu ta tenta asesa un pozisan ki ka ezisti na lista.

```python
ilhas = ["Santiago", "São Vicente", "Sal"]
print(ilhas[3])
```

<TerminalBlock
  title="python testa.py"
  lines={[
    { type: "err", t: "Traceback (most recent call last):" },
    { type: "trace", t: '  File "testa.py", line 2, in <module>' },
    { type: "code", t: "    print(ilhas[3])" },
    { type: "caret", t: "          ~~~~~^^^" },
    { type: "err", t: "IndexError: list index out of range" },
  ]}
/>

Lista tene 3 elementus (índisi 0, 1, 2), ma bu ta pidi índisi 3.

**Kauzas kumun:**
- Skise ki índisi ta kumesa na **0** (ka na 1)
- Uza `len(lista)` kumo índisi (último índisi válidu é `len(lista) - 1`)
- Lista váziu i bu ta tenta asesa kualker índisi

```python
# Koresan: verifika tamañu antis
ilhas = ["Santiago", "São Vicente", "Sal"]
indisi = 3
if indisi < len(ilhas):
    print(ilhas[indisi])
else:
    print(f"Índisi {indisi} ka ezisti! Lista so tene {len(ilhas)} elementus.")
```

<SectionHeading level={3} variant="concept">5. `KeyError` — Xavi ka Ezisti na Disionáriu</SectionHeading>

Bu ta buska un xavi ki ka ezisti na disionáriu.

```python
estudanti = {"nomi": "Kelvin", "idadi": 22}
print(estudanti["nota"])
```

<TerminalBlock
  title="python testa.py"
  lines={[
    { type: "err", t: "Traceback (most recent call last):" },
    { type: "trace", t: '  File "testa.py", line 2, in <module>' },
    { type: "code", t: '    print(estudanti["nota"])' },
    { type: "caret", t: "          ~~~~~~~~~^^^^^^^^" },
    { type: "err", t: "KeyError: 'nota'" },
  ]}
/>

**Koresan: uza `.get()` pa evita eru**

```python
estudanti = {"nomi": "Kelvin", "idadi": 22}

# Manera seguru — retorna None si ka ezisti
nota = estudanti.get("nota")
print(nota)  # None

# Manera seguru ku valór padran
nota = estudanti.get("nota", "Sen nota")
print(nota)  # Sen nota
```

<SectionHeading variant="concept">Estratéjia di 3 Pasus pa Rezolvi Eru</SectionHeading>

Kuandu bo odja un eru, sigi senpri es 3 pasus:

<Steps
  variant="phases"
  showHeader={false}
  phaseWord="Pasu"
  steps={[
    { title: "Ler", content: [{ type: "prose", t: "Última linha di traceback — **tipu** di eru i **mensajen** ki ta splika-l." }] },
    { title: "Atxa", content: [{ type: "prose", t: "Fixeru i número di linha. Seta `^` ta mostra lokalizasan exatu." }] },
    { title: "Pensa", content: [{ type: "prose", t: "Pergunta-bu: **\"Pamodi Python ta pensa ki es é un problema?\"**" }] },
  ]}
/>

Nu pratika ku un izemplu:

:::callout{type=info title="O ki es kódiku ta tenta faze"}
Soma kuatu presu (350, 200, 150, 400) i kalkula média. Ka preokupa ku `for` i `range` — bu ta prende-s na Lisan 10. Repara na traceback.
:::

```python
# Programa ku eru — bo pode atxa?
lista_di_presus = [350, 200, 150, 400]
total = 0

for i in range(1, 5):
    total = total + lista_di_presus[i]

media = total / 4
print(f"Média di presu: {media} ECV")
```

<TerminalBlock
  title="python presus.py"
  lines={[
    { type: "err", t: "Traceback (most recent call last):" },
    { type: "trace", t: '  File "presus.py", line 6, in <module>' },
    { type: "code", t: "    total = total + lista_di_presus[i]" },
    { type: "caret", t: "                    ~~~~~~~~~~~~~~~^^^" },
    { type: "err", t: "IndexError: list index out of range" },
  ]}
/>

Aplika 3 pasus:
1. **LER:** `IndexError: list index out of range` — índisi fora di rangu
2. **ATXA:** Linha 6, `lista_di_presus[i]` — seta mostra ki problema é na asesu pa lista
3. **PENSA:** `range(1, 5)` ta da `1, 2, 3, 4`. Ma `lista_di_presus[4]` ka ezisti pamodi índisis ta bai di 0 a 3. Soluson: `range(len(lista_di_presus))` o `range(0, 4)`.

```python
# Korrejidu!
lista_di_presus = [350, 200, 150, 400]
total = 0

for i in range(len(lista_di_presus)):
    total = total + lista_di_presus[i]

media = total / len(lista_di_presus)
print(f"Média di presu: {media} ECV")  # Média di presu: 275.0 ECV
```

:::callout{type=tip}
Na futuru, bu ta prende ki `sum(lista_di_presus)` é manera más simples di soma un lista. Ma pa gosi, importanti é kompriende kumo ler erus!
:::
