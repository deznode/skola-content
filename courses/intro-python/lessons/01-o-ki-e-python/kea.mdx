# O ki é Python

Imagina bu tene un ferramenta ki ta permiti-bu kria websites, analiza dadus, konstrui intelijensia artifisial, i automatiza tarefas repetitivu — tudu ku un só linguajen. Kel ferramenta é **Python**. Na es lisan, nu ta diskubri kuze é Python, unde é uzadu, i pamodi el pode abri portas pa bu karera na tek.

## Python: Un Linguajen pa Tudu Mundu

<GlossaryText
  text="Python é un [[linguajen di programasan]] ki kriadu na 1991 pa Guido van Rossum, un programador olandez."
  terms={{
    "linguajen di programasan": { en: "programming language", definition: "Un sistema di palavras i regras ki nu ta uza pa skrebe instrusons ki un komputador pode kompriende i kore." },

/>

Nomi "Python" ben di grupu di komedia britaniku *Monty Python* — nton dja bu sabi ki komunidadi di Python ta gosta di bon humor!

Ma kuze ki ta faze Python spesial? Trez kuzas:

1. **Faxi di ler i skrebe** — Python ta paresi ku inglés simples. Konparado ku otus linguajens, kódiku di Python é limpu i diretu.
2. **Versátil** — Bu pode uza Python pa kria website, analiza dadus, konstrui AI, automatiza tarefa, i mutu más.
3. **Komunidadi enormi** — Milhons di programadoris na mundu interu ta uza Python i ta partilha bibliotekas (libraries) di gras.

Odja un izemplu simples:

```python
# Nha primeru programa na Python
print("Ola! Ben-vindu a Python!")
```

Kel kódiku li ta imprimi: `Ola! Ben-vindu a Python!`

Sen símbulus komplikadu, sen pontu-i-vírgula. Python é diretu i limpu.

:::callout{type=quote cite="Tim Peters — O Zen di Python"}
Simples é midjór ki komplikadu.
:::

:::callout{type=tip}
Na Python, `print()` é funsan ki ta mostra textu na tela. É primeru funsan ki tudu programador ta prende!
:::

## Python i Kabu Verdi: Oportunidadis Real

Kabu Verdi ta investi kada bez más na tek. Python pode abri portas pa bo na várius kaminhu:

### Trabadju Remotu

Ku Python, bu pode trabadja pa empresas internasionais diretu di Praia, Mindelo, ou Espargos. Milhares di vagas remotu ta pidi Python kumo skill prinsipál.

```python
# Imagina: bu ta kria un sistema pa jestan di turistas na Sal
turistas = ["Ana", "Pedro", "Carla"]
ilha = "Sal"

for turista in turistas:
    print(f"Ben-vindu a {ilha}, {turista}! Morabeza!")
```

Ka preokupa si bu ka kompriende tudu gosi — `for`, `[ ]`, i `f"…"` é kuzas ki nu ta prende na próximus lisans. Pa gosi, só odja: ku poku linhas di kódiku, Python pode rezolvi un problema real.

### Startups Kabu-Verdianu

- Un app pa kontrola vendas di pexe na merkadu

Ku siénsia di dadus na Python, bu pode analiza dadus públiku pa djuda Kabu Verdi:

- Análisi di klima i agu pa agrikultura na Santo Antão

- Estatístikas di saúdi pa midjora servisu públiku

## Pamodi Python Primeru?

Si bo é novu na programasan, Python é eskolha perfetu pa kumesa:

| Razan | Splikasan |
|--------|-----------|
| **Sintaxi simples** | Kódiku ta paresi ku linguajen natural — faxi di kompriende |
| **Sen kompilasan** | Skrebe kódiku → kore diretu. Sen pasus extra |
| **Erus klaru** | Python ta mostra-bu exatamenti unde, i kuze kore mal |
| **Bibliotekas pa tudu** | Presiza di algu? Provavelmenti algen dja kria un pakoti pa kel |
| **Merkadu di trabadju** | Un di kez linguajens más pididu na vagas di emprego |

Otus linguajens kumo JavaScript, Java ou C++ tudu é bon, ma Python ta dá-bu un fundasan sólidu ki ta fasilita aprendizajen di kualker linguajen dipos.
