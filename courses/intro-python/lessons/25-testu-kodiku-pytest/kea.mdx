## assert вЂ” Funsamentu di Tudu Testu

`assert` ta verifika ki un kondisun Г© verdadeiru. Si ka Г©, Python ta lansa `AssertionError`:

```python

`assert` Г© simples: si kondisun Г© `True`, tudu ben. Si Г© `False`, testu fayo.

## Pensamentu Test-Driven (TDD Leve)

TDD signifika skrebi testus **antis** di skrebi kГіdiku. Siklu Г©:

```

## Rezumu

- **assert** вЂ” verifika ki un kondisun Г© verdadeiru; si ka Г©, testu fayo
- **pytest** вЂ” ferramenta ki ta diskubri i roda tudu funsoens `test_*` automГЎtikamenti

- **`pytest.raises(Error)`** вЂ” testa ki un funsan lansa un exsesun espesГ­fiku
- **Pensamentu TDD:** skrebi testu primeru в†’ implementa в†’ verifika в†’ repeti
