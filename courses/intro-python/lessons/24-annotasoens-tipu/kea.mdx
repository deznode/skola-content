# Annotasons di Tipu

Kuandu bu ta skrebe un funsan, bo sabi ki tipu di dadus el ta espera i ki tipu el ta retorna. Ma un otru programador ki ta ler bo kódiku 3 mez dipos... i sabi? I bo mesmu, dipos di un férias na Sal, bu ta lembra?

**Type hints** é un forma di dokumenta diretamenti na kódiku ki tipu di dadus kada variável i funsan ta uza. É manera un etiketa na pratelera — ka ta muda konteúdu, ma ta ajuda tudu jen atxa o ki ta buska rápidu.

## Primeru Izemplu: Antis i Dipós

**Sen type hints:**

```python
def kalkula_total(presus, diskonto):
    total = sum(presus)
    return total - (total * diskonto / 100)
```

Perguntas ki leitor ta faze: `presus` é un lista? Di ints o floats? `diskonto` é persentajen (0-100) o frasan (0-1)? O ki ta retorna?

**Ku type hints:**

```python
def kalkula_total(presus: list[float], diskonto: float) -> float:
    total = sum(presus)
    return total - (total * diskonto / 100)
```

Gosi é klaru: `presus` é un lista di floats, `diskonto` é un float (persentajen), i funsan ta retorna un float.

## Annotasons di Funsan

Sintaxi é simples: `parámetru: tipu` i `-> tipu_retorno`:

```python
def saudasan(nomi: str) -> str:
    """Retorna un saudasan personalizadu"""
    return f"Ola, {nomi}! Bon dia!"

def soma(a: int, b: int) -> int:
    """Soma dos numerus interu"""
    return a + b

def é_maior_di_idadi(idadi: int) -> bool:
    """Verifika si pesoa é maior di idadi na Kabu Verdi"""
    return idadi >= 18

def mostra_mensajen(mensajen: str) -> None:
    """Funsan ki ka retorna nada (None)"""
    print(f"📢 {mensajen}")
```

:::callout{type=tip}
`-> None` signifika ki funsan ka retorna nada (só ta faze un asan kumo print).
:::

## Annotasons di Variável

Bu pode anotar variáveis tanbe:

```python
# Variáveis simples
nomi: str = "Cesária"
idadi: int = 28
saltu: float = 85000.50
ta_ativi: bool = True

# Sen valór inisiál (só deklarasan di tipu)
rezultadu: float
```

Na prátika, pa variáveis simples, type hint é opisionál pamodi tipu ta ser óbviu. É más útil na funsan i kazus más kompleksu.

## Tipus di Container (Python 3.9+)

Dezdi Python 3.9, bu pode uza tipus di container diretamenti (antis, presizava `from typing import List`):

```python
# Lista di strings
ilhas: list[str] = ["Santiago", "São Vicente", "Sal", "Fogo"]

# Lista di numerus
notas: list[float] = [85.5, 92.0, 78.3, 90.1]

# Disionáriu: xavi é string, valór é int
populasan: dict[str, int] = {
    "Praia": 159_050,
    "Mindelo": 70_468,
    "Santa Maria": 25_765,
}

# Tupla ku tipus fixu
koordenadas: tuple[float, float] = (14.9333, -23.5133)  # Praia

# Set di strings
lingua_ofisial: set[str] = {"portugues", "kriolu"}
```

### Funsan ku Containers

```python
def média_notas(notas: list[float]) -> float:
    """Kalkula média di un lista di notas"""
    if not notas:
        return 0.0
    return sum(notas) / len(notas)

def konta_palavras(tekstu: str) -> dict[str, int]:
    """Konta frekvénsia di kada palavra"""
    palavras = tekstu.lower().split()
    kontador: dict[str, int] = {}
    for palavra in palavras:
        kontador[palavra] = kontador.get(palavra, 0) + 1
    return kontador

def filtra_aprovadus(estudantis: dict[str, float], mínima: float = 10.0) -> list[str]:
    """Retorna nomis di estudantis ku nota >= mínima"""
    return [nomi for nomi, nota in estudantis.items() if nota >= mínima]

# Uza
notas_klasi = {"Maria": 15.5, "João": 8.0, "Ana": 12.3, "Pedro": 9.5}
aprovadus = filtra_aprovadus(notas_klasi)
print(aprovadus)  # ['Maria', 'Ana']
```

## Union: `int | None` (Python 3.10+)

Algunhas vezes, un funsan pode retorna un valór **o** `None` (kuandu ka tene rezultadu). Sintaxi `|` ta espresa es:

```python
def buska_estudanti(nomi: str, bazi: dict[str, int]) -> int | None:
    """Buska nota di estudanti. Retorna None si ka atxa."""
    return bazi.get(nomi)

# Rezultadu pode ser int o None
notas = {"Kelvin": 16, "Edna": 14, "Nilton": 18}

nota_kelvin: int | None = buska_estudanti("Kelvin", notas)
print(nota_kelvin)  # 16

nota_lara: int | None = buska_estudanti("Lara", notas)
print(nota_lara)  # None
```

### Más Ezemplus di Union

```python
def konverti_pa_numeru(tekstu: str) -> float | None:
    """Tenta konverti string pa float. None si faya."""
    try:
        return float(tekstu)
    except ValueError:
        return None

def buska_presu(prudotu: str) -> int | str:
    """Retorna presu o mensajen di eru"""
    presus = {"katxupa": 250, "grogu": 500, "ponche": 300}
    if prudotu in presus:
        return presus[prudotu]
    return f"Prudotu '{prudotu}' ka foi atxadu"

# Parámetru opisionál ku valór padran
def saudasan(nomi: str, lingua: str | None = None) -> str:
    """Sauda na lingua espesifikadu o padran (kriolu)"""
    if lingua == "pt":
        return f"Olá, {nomi}!"
    elif lingua == "en":
        return f"Hello, {nomi}!"
    return f"Ola, {nomi}! Tudu dretu?"
```

## Grandi Segredu: Type Hints Ka Ta Muda NADA na Runtime!

Es é pontu más importanti di es lisan: **type hints é sujestoens, ka regras.** Python ka ta verifika kes kuandu programa ta roda!

```python
def soma(a: int, b: int) -> int:
    return a + b

# Python ka ta da eru — mesmu ku tipus eradu!
rezultadu = soma("Ola", "Mundu")  # Funsiona! Retorna "OlaMundu"
print(rezultadu)  # OlaMundu
```

Entun pa ki é ki type hints ta sirbi?

### 1. Dokumentasan pa Umanus

```python
# Sen hints — ki é ki es funsan ta espera?
def prosesa(dadus, opcoes, callback):
    ...

# Ku hints — klaru!
def prosesa(dadus: list[dict], opcoes: dict[str, bool], callback: callable) -> bool:
    ...
```

### 2. IDE Autocomplete

Kuandu bo skrebe `nomi: str = "Maria"`, bo IDE (VS Code) ta sabi ki `nomi` é string i ta sugerir tudu metodus di string (`.upper()`, `.split()`, etc.) automátikamenti!

### 3. Ferramenta di Análizi (mypy)

```bash
# mypy ta verifika type hints sen roda programa
pip install mypy
mypy nha_programa.py
```

```python
# mypy ta aviza si tipo é inkompatível
def soma(a: int, b: int) -> int:
    return a + b

soma("ola", "mundu")  # mypy: error: Argument 1 has incompatible type "str"
```

:::callout{type=tip}
Pensa na type hints manera sinais di tránzitu. Sinál ta dizi "másimu 60 km/h" — karu PODE bai 100, ma si polísia (mypy) ta kontrola, bu ta leva multa!
:::

## Izemplu Konpletu: Registu di Estudantis

```python
def kria_estudanti(
    nomi: str,
    idadi: int,
    kursu: str,
    nota_final: float | None = None
) -> dict[str, str | int | float | None]:
    """Kria un registu di estudanti"""
    return {
        "nomi": nomi,
        "idadi": idadi,
        "kursu": kursu,
        "nota_final": nota_final,
    }

def klasifika(nota: float) -> str:
    """Retorna klasifikasan bazeadu na nota (0-20)"""
    if nota >= 16:
        return "Exselenti"
    elif nota >= 14:
        return "Mutu Bon"
    elif nota >= 10:
        return "Sufisienti"
    return "Insufisienti"

def relatóriu_klasi(
    estudantis: list[dict[str, str | int | float | None]]
) -> dict[str, float | int]:
    """Jera relatóriu di klasi ku estatístikas"""
    notas: list[float] = []
    for est in estudantis:
        if est["nota_final"] is not None:
            notas.append(est["nota_final"])

    return {
        "total_estudantis": len(estudantis),
        "ku_nota": len(notas),
        "média": sum(notas) / len(notas) if notas else 0.0,
        "más_altu": max(notas) if notas else 0.0,
        "más_baxu": min(notas) if notas else 0.0,
    }

# Uza
klasi: list[dict] = [
    kria_estudanti("Edna", 20, "Informátika", 17.5),
    kria_estudanti("Gil", 22, "Informátika", 14.0),
    kria_estudanti("Lara", 19, "Informátika", 11.5),
    kria_estudanti("Nuno", 21, "Informátika"),  # Inda sen nota
]

relátoriu = relatóriu_klasi(klasi)
print(f"Média di klasi: {relátoriu['média']:.1f}")  # Média di klasi: 14.3
```
