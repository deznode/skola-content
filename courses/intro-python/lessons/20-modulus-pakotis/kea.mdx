# Módulus i Pakotis

Kuandu bo programa ta krési, bo nau podi pô tudu na un ficheru só. Python ta permite organiza kódiku na **módulus** (ficheru .py individuais) e **pakotis** (pasta ku módulus relasionadu). Alen disso, Python tene un **standard library** enormi ku módulus pronto pa uza — i un ekosistema di milháris di pakotis ki komunidadi kria.

## O ki é un Módulu?

Un módulu é simplesmente un ficheru Python (`.py`) ki kontene funsoens, klasis, ó variáveis. Kuandu bo fazi `import`, bo ta trazi kel kódiku pa bo programa.

```python
# Python tene muitius módulus built-in prontu pa uza
import math

print(math.sqrt(16))    # 4.0
print(math.pi)          # 3.141592653589793
print(math.ceil(4.2))   # 5
print(math.floor(4.8))  # 4
```

## Tres Maneira di Importa

### 1. `import modulo` — Importa módulu interu

```python
import math

# Presiza prefixu: math.funsan()
resultado = math.factorial(5)
print(resultado)  # 120
```

### 2. `from modulo import algo` — Importa elementus spesífiku

```python
from math import sqrt, pi, pow

# Uza diretu sin prefixu
print(sqrt(25))     # 5.0
print(pi)           # 3.14159...
print(pow(2, 10))   # 1024.0
```

### 3. `import modulo as alias` — Importa ku nomi kurtu

```python
import datetime as dt

agora = dt.datetime.now()
print(f"Data di oji: {agora.strftime('%d/%m/%Y')}")

# Alias komun ki bo ta enkontra na komunidadi:
# import numpy as np
# import pandas as pd
# import matplotlib.pyplot as plt
```

:::callout{type=tip}
**Dika:** Evita `from modulo import *` — el ta importa TUDU i bo nau ta sabi di undi kada funsan ben. Kel ta kria konfuzun i bugs difísil di atxa.
:::

## Standard Library: Módulus ki Dja Ben ku Python

Python tene un "biblioteca padrãu" enormi — módulus ki dja sta instaladu i prontu pa uza:

### `random` — Numeru aleatóriu

```python
import random

# Numeru interu aleatóriu entre 1 i 10
numeru = random.randint(1, 10)
print(f"Numeru sórtiadu: {numeru}")

# Skolhe un elementu aleatóriu
ilhas = ["Santiago", "Sal", "São Vicente", "Fogo", "Boa Vista"]
ilha_sortida = random.choice(ilhas)
print(f"Destinu di férias: {ilha_sortida}")

# Mistura un lista (shuffle)
numeros = [1, 2, 3, 4, 5]
random.shuffle(numeros)
print(f"Misturadu: {numeros}")
```

### `os` — Sistema operativu

```python
import os

# Diretóriu atual
print(os.getcwd())

# Lista ficheru na pasta
ficheros = os.listdir(".")
print(ficheros)

# Verifica si ficheru ezisti
if os.path.exists("notas.txt"):
    print("Ficheru ezisti!")
else:
    print("Ficheru nau ezisti.")

# Kria pasta nóvu
os.makedirs("projetu/dadus", exist_ok=True)
```

### `json` — Trabadja ku JSON

```python
import json

# Python dict → JSON string
estudanti = {
    "nomi": "Maria",
    "idadi": 22,
    "kursu": "Python",
    "ilha": "Santiago"
}
json_str = json.dumps(estudanti, indent=2, ensure_ascii=False)
print(json_str)

# JSON string → Python dict
dadus = json.loads('{"nomi": "João", "nota": 18}')
print(dadus["nomi"])  # João
print(dadus["nota"])  # 18
```

### `datetime` — Data i ora

```python
from datetime import datetime, timedelta

# Data i ora atual
agora = datetime.now()
print(f"Agora: {agora.strftime('%d/%m/%Y %H:%M')}")

# Kalkula data futura
daki_30_dias = agora + timedelta(days=30)
print(f"Daki 30 dias: {daki_30_dias.strftime('%d/%m/%Y')}")

# Parse un string pa data
aniversáriu = datetime.strptime("15/01/1973", "%d/%m/%Y")
print(f"Dia: {aniversáriu.day}, Mes: {aniversáriu.month}")
```

### `csv` — Lé i skrévi CSV

```python
import csv

# Skrévi un CSV
vendas = [
    ["produto", "prasu", "kuantidadi"],
    ["Grogue", "500 ECV", "10"],
    ["Ponche", "300 ECV", "25"],
    ["Mel di kana", "200 ECV", "15"],
]

with open("vendas.csv", "w", newline="") as f:
    escritor = csv.writer(f)
    escritor.writerows(vendas)

# Lé un CSV
with open("vendas.csv", "r") as f:
    leitor = csv.reader(f)
    for linha in leitor:
        print(linha)
```

### `re` — Expresoens regularis

```python
import re

textu = "Maria tene 22 anu i João tene 35 anu"

# Atxa tudu numeru
numeros = re.findall(r"\d+", textu)
print(numeros)  # ['22', '35']

# Verifica si email é válidu (báziku)
email = "maria@gmail.com"
if re.match(r"^[\w.+-]+@[\w-]+\.[\w.]+$", email):
    print(f"{email} parsé válidu ✓")
```

:::callout{type=tip}
**Dika:** Standard library é enormi! Kel li é só un prova. Vizita [docs.python.org/3/library](https://docs.python.org/3/library/) pa esplora tudu módulus disponível.
:::

## Instala Pakotis ku `pip`

Pa pakotis ki **nau sta** na standard library, uza `pip`:

```bash
# Instala un pakotu
pip install requests

# Instala versan spesífiku
pip install requests==2.31.0

# Instala múltiplus pakotis
pip install requests flask pandas

# Mostra pakotis instaladu
pip list

# Mostra informasun di un pakotu
pip show requests
```

```python
# Dipos di instala, uza normal
import requests

resposta = requests.get("https://api.github.com")
print(f"Status: {resposta.status_code}")
```

:::callout{type=tip}
**Dika:** Sénpri instala pakotis dentru di un **ambiente virtual** (`.venv`). Nôs ta prende kel na prósimu lisun!
:::

## Kria Pakotis Personalizadu

Un **pakotu** é un pasta ku un ficheru `__init__.py` (podi ser vazio) e un ó más módulus:

```
meu_projetu/
├── main.py
└── utilidadis/
    ├── __init__.py
    ├── matematika.py
    └── texto.py
```

```python
# utilidadis/matematika.py
def soma(a, b):
    """Soma dos numeru."""
    return a + b

def media(numeros):
    """Kalkula média di un lista di numeru."""
    return sum(numeros) / len(numeros)
```

```python
# utilidadis/texto.py
def limpa(texto):
    """Removi espasus extra i konverti pa minúskulu."""
    return texto.strip().lower()

def konta_palavras(texto):
    """Konta numeru di palavras."""
    return len(texto.split())
```

```python
# utilidadis/__init__.py (podi ser vazio, ó importa kusas prinsipal)
from .matematika import soma, media
from .texto import limpa, konta_palavras
```

```python
# main.py -- uza bo pakotu
from utilidadis import soma, media, limpa

print(soma(10, 20))           # 30
print(media([15, 18, 12]))    # 15.0
print(limpa("  PRAIA  "))     # "praia"

# Ó importa módulu spesífiku
from utilidadis.texto import konta_palavras
print(konta_palavras("Nu ta kria djuntu"))  # 4
```

## Padrãu `__name__ == "__main__"`

Kuandu Python ta roda un ficheru, el ta define `__name__` kómu `"__main__"`. Ma kuandu el é **importadu**, `__name__` é nomi di módulu:

```python
# kalkuladora.py
def soma(a, b):
    return a + b

def subtrai(a, b):
    return a - b

# Kel bloku só ta roda si kel ficheru é ezekatadu diretu
if __name__ == "__main__":
    # Kódiku di testu / demonstrasun
    print("Testandu kalkuladora:")
    print(f"5 + 3 = {soma(5, 3)}")
    print(f"10 - 4 = {subtrai(10, 4)}")
```

```bash
# Si bo roda diretu:
python kalkuladora.py
# Saída: "Testandu kalkuladora:" + resultadus

# Si bo importa di otu ficheru:
# from kalkuladora import soma  -- bloku if NAU ta roda
```

:::callout{type=tip}
**Dika:** Kel padrãu é perfeitu pa pô kódiku di testu ó ezemplus ki só ta roda kuandu bo ezekuta ficheru diretu, ma nau kuandu el é importadu kómu módulu.
:::

## Struktura di un Projetu Real

Un projetu Python ben organizadu ta parsé ku kel li:

```
gestor_notas/
├── main.py              # Pontu di entrada
├── requirements.txt     # Dependénsia di terseru
├── .venv/               # Ambiente virtual (NAU ta bai pa git!)
├── modulus/
│   ├── __init__.py
│   ├── estudanti.py     # Funsun relasionadu ku estudanti
│   ├── nota.py          # Funsun relasionadu ku nota
│   └── relatoriu.py     # Jera relatórius
└── dadus/
    └── notas.csv        # Ficheru di dadus
```

3. **Exersísiu 3: Pakotu di Utilidadis** — Kria un pasta `utils/` ku `__init__.py`, un módulu `validasun.py` (funsan `valida_email(email)`) e un módulu `formatu.py` (funsan `formata_ecv(valor)` ki retorna "1.500$00 ECV"). Importa tudu di `main.py`.

## Rezumu

- **Módulu** é un ficheru `.py` ku kódiku reutilizável
- Tres formas di import: `import math`, `from math import sqrt`, `import numpy as np`
- **Standard library** tene módulus prontu: `math`, `random`, `os`, `json`, `csv`, `datetime`, `re`
- **pip** ta instala pakotis di terseru: `pip install nome_pakotu`
- **Pakotu** é un pasta ku `__init__.py` + módulus
- `__name__ == "__main__"` ta permite kódiku ki só roda kuandu ficheru é ezekatadu diretu
- Organiza projetu na módulus ta fazi kódiku más fásil di lé, mante, e reusa
