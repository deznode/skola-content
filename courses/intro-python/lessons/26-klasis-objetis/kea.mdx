# Klasis i Objetis

Imagina reseta di katxupa. Reseta ta dize ingredientis (midju, feijun, mandioka, lingisa...), kantidadi, i pasus pa prepara. Ma reseta ka é katxupa — é só instruson. Kada bez ki bo sigi reseta i kuzinha, bu ta kria un **katxupa real** — un pratu diferenti, ku si sabor própriu.

Na programasan, **klasi** é kumo reseta — un moldi ki ta deskrebe kumo un koisa debe ser. **Objetu** é kel ki bu ta kria a parti di moldi — un instánsia real, ku si dadus própriu. Bu pode uza mesmu reseta pa faze katxupa pa almósu na Praia i pa janta na Mindelo — dos pratu diferenti, mesmu reseta!

## Kria Klasi ku `class`

Pa defini un klasi na Python, bo uza keyword `class` seguidu di nomi (ku **PascalCase** — primeru letra di kada palavra maiúskula):

```python
# Klasi más simples posível
class Pesoa:
    pass  # 'pass' signifika "nada aki pa gosi"

# Kria un objetu (instánsia) di klasi Pesoa
maria = Pesoa()
joao = Pesoa()

print(type(maria))  # <class '__main__.Pesoa'>
print(type(joao))   # <class '__main__.Pesoa'>
```

Es klasi ka ta faze nada inda — é kumo un reseta na branku. Nu ta adisiona ingredientis!

## `__init__` — Konstrutor di Klasi

Métudu spesiál `__init__` ta roda **automátikamenti** kuandu bo kria un objetu novu. É konstrutor — ta inisializa atributus di objetu:

```python
class Pesoa:
    def __init__(self, nomi, ilha, idadi):
        self.nomi = nomi      # Atributu: nomi di pesoa
        self.ilha = ilha      # Atributu: ilha di orijem
        self.idadi = idadi    # Atributu: idadi

# Kria objetis — __init__ ta roda automátikamenti
maria = Pesoa("Maria", "Santiago", 25)
nilton = Pesoa("Nilton", "São Vicente", 30)

print(maria.nomi)    # Maria
print(nilton.ilha)   # São Vicente
print(maria.idadi)   # 25
```

```python
# ERU KOMUN: skese di pasa argumentus
# pesoa = Pesoa()
# TypeError: __init__() missing 3 required positional arguments
```

:::callout{type=tip}
**Dika:** `__init__` ka é teknikamenti "kria" objetu — Python ta kria el primeru, dipos `__init__` ta inisializa si atributus. Nomi `__init__` signifika "initialize" (inisializa).
:::

## `self` — O ki é?

`self` é referénsia pa **objetu atuál** — kel ki ta ser kriadu o usadu. Pensa na `self` kumo "**nha**" (meu/nha). Kuandu bo skrebe `self.nomi = nomi`, é kumo dize "nha nomi é esi valór":

```python
class Estudanti:
    def __init__(self, nomi, kursu):
        # "nha nomi é [nomi ki entra]"
        self.nomi = nomi
        # "nha kursu é [kursu ki entra]"
        self.kursu = kursu

# Kuandu Python ta kria 'ana', self = ana
ana = Estudanti("Ana", "Engenharia")
# ana.nomi → "Ana" (nha nomi)
# ana.kursu → "Engenharia" (nha kursu)

# Kuandu Python ta kria 'kelvin', self = kelvin
kelvin = Estudanti("Kelvin", "Medisina")
# kelvin.nomi → "Kelvin"
# kelvin.kursu → "Medisina"
```

```python
# ERU KOMUN: skese di 'self' kumo primeiru parámetru
class Mal:
    def __init__(nomi):  # ← falta self!
        nomi = nomi      # ← isu ka ta guarda nada na objetu!

# TypeError: __init__() takes 1 positional argument but 2 were given
```

:::callout{type=tip}
**Dika:** `self` é **senpri** primeru parámetru di tudu métudu dentu di un klasi. Python ta pasa el automátikamenti — bu ka presiza skrebe el kuandu ta txoma métudu.
:::

## Variáveis di Instánsia

Variáveis di instánsia é atributus ki **pertense a kada objetu individualmenti**. Kada objetu tene si própriu kópia:

```python
class Livru:
    def __init__(self, títulu, autor, presiu):
        self.títulu = títulu
        self.autor = autor
        self.presiu = presiu    # Presu na ECV

# Kada livru é independenti
livru1 = Livru("Python pa Tudu Jenti", "Amilcar", 1500)
livru2 = Livru("Algoritmus na Kriolu", "Cesária", 2000)

print(livru1.títulu)   # Python pa Tudu Jenti
print(livru2.títulu)   # Algoritmus na Kriolu

# Muda un ka ta afeta otru
livru1.presiu = 1200
print(livru1.presiu)   # 1200
print(livru2.presiu)   # 2000 (ka muda!)
```

```python
# Defini tudu atributu dentu di __init__
class Artista:
    def __init__(self, nomi, jeneru):
        self.nomi = nomi
        self.jeneru = jeneru
        self.albuns = []        # Lista vázia — ta enxi depois
        self.fans = 0           # Kumesa na zero

cesaria = Artista("Cesária Évora", "Morna")
print(cesaria.albuns)   # []
print(cesaria.fans)     # 0
```

:::callout{type=tip}
**Dika:** **Senpri** defini tudu atributu dentu di `__init__`, mesmu si valór inisiál é `[]`, `0`, o `None`. Defini fora di `__init__` pode kauza `AttributeError` na otus objetis.
:::

## Metodus di Instánsia

Metodus é funsons definidu **dentu di klasi** ki ta opera sobri dadus di objetu. Es ta resebe `self` kumo primeru parámetru:

```python
class Musiku:
    def __init__(self, nomi, instrumentu):
        self.nomi = nomi
        self.instrumentu = instrumentu
        self.músikus_tokadu = []

    def toka(self, músika):
        """Adisiona músika na lista di tokadu"""
        self.músikus_tokadu.append(músika)
        print(f"🎵 {self.nomi} ta toka '{músika}' na {self.instrumentu}")

    def repertóriu(self):
        """Mostra tudu músikus tokadu"""
        if not self.músikus_tokadu:
            print(f"{self.nomi} inda ka toka nada")
            return
        print(f"Repertóriu di {self.nomi}:")
        for i, m in enumerate(self.músikus_tokadu, 1):
            print(f"  {i}. {m}")

# Kria músiku i uza si metodus
joao = Musiku("João", "Violun")
joao.toka("Sodadi")           # 🎵 João ta toka 'Sodadi' na Violun
joao.toka("Petit Pays")       # 🎵 João ta toka 'Petit Pays' na Violun
joao.toka("Nha Testamentu")   # 🎵 João ta toka 'Nha Testamentu' na Violun
joao.repertóriu()
# Repertóriu di João:
#   1. Sodadi
#   2. Petit Pays
#   3. Nha Testamentu
```

```python
# Métudu ki ta retorna valór
class Retángulu:
    def __init__(self, largura, altura):
        self.largura = largura
        self.altura = altura

    def área(self):
        """Kalkula área di retángulu"""
        return self.largura * self.altura

    def perímetru(self):
        """Kalkula perímetru di retángulu"""
        return 2 * (self.largura + self.altura)

    def deskrebe(self):
        """Retorna deskrisan di retángulu"""
        return f"Retángulu {self.largura}x{self.altura} (área: {self.área()} m²)"

terenu = Retángulu(20, 15)
print(terenu.área())        # 300
print(terenu.perímetru())   # 70
print(terenu.deskrebe())    # Retángulu 20x15 (área: 300 m²)
```

## Izemplu Prátiku: Konta Bankáriu

Nu ta kria un klasi `KontaBankáriu` konpletu ku depózitu, levantamentu, i estrátu:

```python
class KontaBankáriu:
    def __init__(self, titulár, sáldu=0):
        self.titulár = titulár
        self.sáldu = sáldu          # Sáldu na ECV
        self.istóriku = []           # Istóriku di transasons

    def depozita(self, montanti):
        """Depozita dinheru na konta"""
        if montanti <= 0:
            print("❌ Montanti debe ser pozitivu!")
            return
        self.sáldu += montanti
        self.istóriku.append(f"+{montanti} ECV (depózitu)")
        print(f"✅ {montanti} ECV depozitadu. Sáldu: {self.sáldu} ECV")

    def levanta(self, montanti):
        """Levanta dinheru di konta"""
        if montanti <= 0:
            print("❌ Montanti debe ser pozitivu!")
            return
        if montanti > self.sáldu:
            print(f"❌ Sáldu insufisienti! Bo tene {self.sáldu} ECV")
            return
        self.sáldu -= montanti
        self.istóriku.append(f"-{montanti} ECV (levantamentu)")
        print(f"✅ {montanti} ECV levantadu. Sáldu: {self.sáldu} ECV")

    def estrátu(self):
        """Mostra estrátu di konta"""
        print(f"\n{'='*40}")
        print(f"  Konta di {self.titulár}")
        print(f"{'='*40}")
        if not self.istóriku:
            print("  (sen transasons)")
        else:
            for transasan in self.istóriku:
                print(f"  {transasan}")
        print(f"{'─'*40}")
        print(f"  Sáldu atuál: {self.sáldu} ECV")
        print(f"{'='*40}\n")
```

```python
# Maria ta abri konta na Praia
konta_maria = KontaBankáriu("Maria", 50000)
konta_maria.depozita(15000)   # ✅ 15000 ECV depozitadu. Sáldu: 65000 ECV
konta_maria.levanta(8000)     # ✅ 8000 ECV levantadu. Sáldu: 57000 ECV
konta_maria.depozita(25000)   # ✅ 25000 ECV depozitadu. Sáldu: 82000 ECV
konta_maria.levanta(100000)   # ❌ Sáldu insufisienti! Bo tene 82000 ECV

konta_maria.estrátu()
# ========================================
#   Konta di Maria
# ========================================
#   +15000 ECV (depózitu)
#   -8000 ECV (levantamentu)
#   +25000 ECV (depózitu)
# ────────────────────────────────────────
#   Sáldu atuál: 82000 ECV
# ========================================
```

## Kria Vários Objetis

Kada objetu ki bo kria é **totalmenti independenti**. Muda un ka ta afeta otru — mesmu ki es sai di mesmu klasi:

```python
# Kria vários kontas independentis
konta_maria = KontaBankáriu("Maria", 50000)
konta_pedro = KontaBankáriu("Pedro", 30000)
konta_djina = KontaBankáriu("Djina", 75000)

# Operasons na un konta ka ta afeta otrus
konta_maria.depozita(10000)   # Maria: 60000 ECV
konta_pedro.levanta(5000)     # Pedro: 25000 ECV
konta_djina.depozita(20000)   # Djina: 95000 ECV

# Kada un tene si sáldu própriu
print(f"Maria: {konta_maria.sáldu} ECV")   # Maria: 60000 ECV
print(f"Pedro: {konta_pedro.sáldu} ECV")   # Pedro: 25000 ECV
print(f"Djina: {konta_djina.sáldu} ECV")   # Djina: 95000 ECV
```

```python
# Más un ezemplu: Lista di estudantis
class Estudanti:
    def __init__(self, nomi, ilha, nota):
        self.nomi = nomi
        self.ilha = ilha
        self.nota = nota

    def aprovadu(self):
        """Verifika si estudanti pasa (nota >= 10)"""
        return self.nota >= 10

    def deskrebe(self):
        status = "✅ Aprovadu" if self.aprovadu() else "❌ Reprovadu"
        return f"{self.nomi} ({self.ilha}) — Nota: {self.nota} — {status}"

# Kria lista di estudantis
turma = [
    Estudanti("Ana", "Santiago", 16),
    Estudanti("Nilton", "São Vicente", 8),
    Estudanti("Edna", "Sal", 14),
    Estudanti("Nelson", "Fogo", 11),
    Estudanti("Ivete", "Boa Vista", 9),
]

# Mostra tudu
print("📋 Rezultadus di Turma:")
for e in turma:
    print(f"  {e.deskrebe()}")

# Filtra aprovadus
aprovadus = [e for e in turma if e.aprovadu()]
print(f"\n{len(aprovadus)} di {len(turma)} estudantis aprovadu!")
# 3 di 5 estudantis aprovadu!
```

:::callout{type=tip}
**Dika:** Klasis é perfeitu pa modela kualker koisa di mundu real — pesoas, produtus, kontas, veíkulus, enkomendas di Sukupira... Pensa na ki **dadus** i ki **asons** kel koisa tene.
:::

## Erus Komunis

```python
# ERU 1: Skese di self na métudu
class Mal:
    def __init__(self, nomi):
        self.nomi = nomi

    def saudasan():  # ← FALTA self!
        print(f"Ola, N é {self.nomi}")

# p = Mal("João")
# p.saudasan()  # TypeError: saudasan() takes 0 positional arguments but 1 was given
```

```python
# ERU 2: Defini atributu fora di __init__
class Mal:
    def __init__(self, nomi):
        self.nomi = nomi

    def seta_idadi(self, idadi):
        self.idadi = idadi  # Só ezisti si seta_idadi() foi txomadu!

p = Mal("Maria")
# print(p.idadi)  # AttributeError: 'Mal' object has no attribute 'idadi'
# Pamodi? Pamodi seta_idadi() nunka foi txomadu!
```

```python
# KORETU: Defini tudu na __init__
class Bon:
    def __init__(self, nomi, idadi=None):
        self.nomi = nomi
        self.idadi = idadi  # Ezisti senpri, mesmu si é None

p = Bon("Maria")
print(p.idadi)  # None (ka da eru!)
```
