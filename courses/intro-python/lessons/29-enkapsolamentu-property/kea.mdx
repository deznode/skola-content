# Enkapsolamentu i @property

Pensa na un kuzinha di restoranti na Mindelo. Klientis ta senta na sala, ta odjá menu i ta faze pedidu — isu é **públiku**. Garsons ta entra na kuzinha pa buska pratus i ta komuniká ku xefi — es tene asésu **protejidu**. Ma reseita sekretu di katxupa di xefi? Kel só xefi ta sabi — é **privadu**. Ninguén ka pode txiga diretamenti na reseita — si bu kre sabi ingredientis, bu ten ki pidi xefi (el ta disidi si ta fala o nau).

Na programasan, enkapsolamentu é mesmu ideia: **kontrola ken ki pode asésa i modifika dadus di un objetu**. En bez di dexa tudu abertu, bu ta proteje partis sensíveis i ta fornese maneiras kontroladu di asésa-l.

## Asésu Públiku, Protejidu i Privadu

Python tene 3 niveis di asésu, bazadu na **konvensan di nomi**:

| Tipu | Sintaxi | Signifikadu |
|------|---------|-------------|
| Públiku | `self.nomi` | Kualker un pode asésa |
| Protejidu | `self._nomi` | Konvensan: só klasi i fidju debe uza |
| Privadu | `self.__nomi` | Python ta transforma nomi (name mangling) |

```python
class Estudanti:
    def __init__(self, nomi, ilha, nota):
        self.nomi = nomi          # Públiku — kualker un pode asésa
        self._ilha = ilha          # Protejidu — konvensan: "ka toka sen motivo"
        self.__nota = nota         # Privadu — Python ta proteje ku name mangling

# Kria estudanti
maria = Estudanti("Maria", "Santiago", 16)

# Asésu públiku — funsiona normal
print(maria.nomi)       # Maria ✅

# Asésu protejidu — funsiona, ma é konvensan pa ka uza diretu
print(maria._ilha)      # Santiago ⚠️ (funsiona, ma ka debe)

# Asésu privadu — ERU!
# print(maria.__nota)   # AttributeError: 'Estudanti' object has no attribute '__nota'
```

:::callout{type=tip}
Na Python, `_protejidu` é só un **konvensan** — nada ka ta impidi bo di asésa-l. É kumo un avizu: "Oji, ka toka nisi sen motivo!" Ma `__privadu` tene protesan real.
:::

## Name Mangling — Kumo Python ta Proteje Atributus Privadu

Kuandu bo uza `__` (dois underscore) na inísiu di un nomi, Python ta transforma-l internamenti. Isu ta txoma **name mangling**.

```python
class KontaBankaria:
    def __init__(self, titular, saldu):
        self.titular = titular      # Públiku
        self.__saldu = saldu        # Privadu

# Kria konta
konta = KontaBankaria("João", 50_000)

# Tenta asésa diretu — ERU
# print(konta.__saldu)  # AttributeError!

# Ma Python ka ta eskonde TOTALMENTI — el ta muda nomi
# __saldu ta vira _KontaBankaria__saldu
print(konta._KontaBankaria__saldu)  # 50000 ⚠️ (funsiona, ma NUNKA faze isu!)
```

```python
# Uza dir() pa odjá tudu atributus (inklusindu nomi transformadu)
print([attr for attr in dir(konta) if "saldu" in attr])
# ['_KontaBankaria__saldu']
```

:::callout{type=tip}
Name mangling ka é seguránsa — é protesan kontra **eru asidentál**. Si algen kre MESMU asésa, el pode. Ma mensajen é klaru: "Kel atributu li é privadu, ka toka nel!"
:::

## Getters i Setters Tradisionál

Na linguajis kumo Java o C++, bu ta kria metodus espesífikis pa ler (get) i skrebe (set) atributus privadu. Na Python, bu pode faze mesmu koza:

```python
class Produtu:
    def __init__(self, nomi, presu):
        self.__nomi = nomi
        self.__presu = presu

    # Getter — pa ler presu
    def get_presu(self):
        return self.__presu

    # Setter — pa muda presu ku validasan
    def set_presu(self, novu_presu):
        if novu_presu < 0:
            print("Eru: presu ka pode ser negativu!")
            return
        self.__presu = novu_presu

    # Getter pa nomi
    def get_nomi(self):
        return self.__nomi

# Uza
kaxtelu = Produtu("Kaxtelu di Fera", 250)

# Asésa via getters/setters
print(kaxtelu.get_nomi())         # Kaxtelu di Fera
print(kaxtelu.get_presu())        # 250

kaxtelu.set_presu(300)            # Muda presu
print(kaxtelu.get_presu())        # 300

kaxtelu.set_presu(-50)            # Eru: presu ka pode ser negativu!
print(kaxtelu.get_presu())        # 300 (ka mudá)
```

Isu ta funsiona, ma é **txeu kódiku** i ka é "Pythonic". Na Python, tene un maneira más eleganti...

## `@property` — Maneira Modernu

Dekorador `@property` ta transforma un metodu na un atributu ki pode ser lidu kumo si el fosse públiku, ma na verdadi ta roda un funsan pa trás.

```python
class Produtu:
    def __init__(self, nomi, presu):
        self.__nomi = nomi
        self.__presu = presu

    @property
    def nomi(self):
        """Getter — asésa komu atributu, nau komu funsan."""
        return self.__nomi

    @property
    def presu(self):
        """Getter pa presu."""
        return self.__presu

# Uza — limpu i eleganti!
kaxtelu = Produtu("Kaxtelu di Fera", 250)

# Asésa kumo atributu (sen paréntesis!)
print(kaxtelu.nomi)    # Kaxtelu di Fera
print(kaxtelu.presu)   # 250

# Ma ka pode muda (ainda ka tene setter)
# kaxtelu.presu = 300  # AttributeError: property 'presu' has no setter
```

Kompara ku maneira tradisionál:

```python
# ANTIS (tradisionál):
print(kaxtelu.get_presu())    # Txoma kumo funsan
kaxtelu.set_presu(300)        # Txoma kumo funsan

# AGORA (ku @property):
print(kaxtelu.presu)          # Asésa kumo atributu ✨
# kaxtelu.presu = 300         # Atribui kumo atributu (si tene setter)
```

:::callout{type=tip}
`@property` ta da bo milhor di dos mundus — **interfasi limpu di atributu** ku **kontrolu di metodu**. Utilizador ka presiza sabi ki tene un funsan pa trás!
:::

## `@property.setter` ku Validasan

Pa permiti modifikasan, bu ta djunta un setter uzandu `@nomi.setter`:

```python
class Produtu:
    def __init__(self, nomi, presu):
        self.__nomi = nomi
        self.presu = presu  # Nota: ta uza setter pa validasan dja na __init__!

    @property
    def nomi(self):
        return self.__nomi

    @property
    def presu(self):
        return self.__presu

    @presu.setter
    def presu(self, valor):
        """Setter ku validasan — proteje kontra valóris inválidus."""
        if not isinstance(valor, (int, float)):
            raise TypeError("Presu ten ki ser un nùmeru!")
        if valor < 0:
            raise ValueError("Presu ka pode ser negativu!")
        self.__presu = valor
```

```python
# Testa
grogu = Produtu("Grogu di Santo Antão", 1_500)
print(grogu.presu)       # 1500

# Muda presu — validasan ta roda automatikamenti
grogu.presu = 1_800
print(grogu.presu)       # 1800 ✅

# Tenta valor inválidu
# grogu.presu = -500     # ValueError: Presu ka pode ser negativu!
# grogu.presu = "karú"   # TypeError: Presu ten ki ser un nùmeru!
```

Bu pode tanbe kria properties kalkuladu — valoris ki ta depende di otru atributus:

```python
class Retangulu:
    def __init__(self, largura, altura):
        self.largura = largura
        self.altura = altura

    @property
    def largura(self):
        return self.__largura

    @largura.setter
    def largura(self, valor):
        if valor <= 0:
            raise ValueError("Largura ten ki ser positivu!")
        self.__largura = valor

    @property
    def altura(self):
        return self.__altura

    @altura.setter
    def altura(self, valor):
        if valor <= 0:
            raise ValueError("Altura ten ki ser positivu!")
        self.__altura = valor

    @property
    def area(self):
        """Property kalkuladu — ka tene setter, é só leitura."""
        return self.__largura * self.__altura

    @property
    def perimetru(self):
        """Otru property kalkuladu."""
        return 2 * (self.__largura + self.__altura)
```

```python
# Terrenu na Boa Vista
terrenu = Retangulu(30, 20)
print(f"Área: {terrenu.area} m²")           # Área: 600 m²
print(f"Perímetru: {terrenu.perimetru} m")   # Perímetru: 100 m

terrenu.largura = 40
print(f"Nova área: {terrenu.area} m²")       # Nova área: 800 m² (rekalkula automatikamenti!)

# terrenu.area = 999  # AttributeError: property 'area' has no setter
```

:::callout{type=tip}
Property kalkuladu (sen setter) é perfeitu pa valoris ki ta depende di otru atributus. Es ta garantí ki valor ta sta SENPRI atualizadu!
:::

## Izemplu Prátiku: Konta Bankáriu Seguru

Nos ta kria un klasi di konta bankáriu ki ta proteje saldu i ta valida tudu operasons:

```python
class KontaBankaria:
    """Konta bankáriu ku enkapsolamentu — saldu privadu e protejidu."""

    # Atributu di klasi — taxa di juru anuál
    _taxa_juru = 0.02  # 2% (protejidu — pode ser atualizadu pa banku)

    def __init__(self, titular, numeru_konta, saldu_inisiál=0):
        self.__titular = titular
        self.__numeru_konta = numeru_konta
        self.__saldu = 0
        self.__istóriku = []

        # Uza setter pa validasan di saldu inisiál
        if saldu_inisiál > 0:
            self.deposita(saldu_inisiál)

    @property
    def titular(self):
        """Nomi di titular — só leitura."""
        return self.__titular

    @property
    def numeru_konta(self):
        """Nùmeru di konta maskáru pa seguránsa."""
        return f"***{self.__numeru_konta[-4:]}"

    @property
    def saldu(self):
        """Saldu atuál — só leitura (ka pode seta diretamenti)."""
        return self.__saldu

    @property
    def istóriku(self):
        """Kópia di istóriku (ka pode modifika orijinál)."""
        return self.__istóriku.copy()

    def deposita(self, montanti):
        """Deposita dinheru na konta."""
        if not isinstance(montanti, (int, float)):
            raise TypeError("Montanti ten ki ser un nùmeru!")
        if montanti <= 0:
            raise ValueError("Montanti di depózitu ten ki ser positivu!")

        self.__saldu += montanti
        self.__istóriku.append(f"Depózitu: +{montanti:,.0f} ECV")
        print(f"✅ Depózitu di {montanti:,.0f} ECV konfirmadu.")
        return True

    def levanta(self, montanti):
        """Levanta dinheru di konta."""
        if not isinstance(montanti, (int, float)):
            raise TypeError("Montanti ten ki ser un nùmeru!")
        if montanti <= 0:
            raise ValueError("Montanti di levantamentu ten ki ser positivu!")
        if montanti > self.__saldu:
            print(f"❌ Saldu insufisienti! Saldu atuál: {self.__saldu:,.0f} ECV")
            return False

        self.__saldu -= montanti
        self.__istóriku.append(f"Levantamentu: -{montanti:,.0f} ECV")
        print(f"✅ Levantamentu di {montanti:,.0f} ECV konfirmadu.")
        return True

    def aplika_juru(self):
        """Aplika juru anuál na saldu."""
        juru = self.__saldu * self._taxa_juru
        self.__saldu += juru
        self.__istóriku.append(f"Juru ({self._taxa_juru*100:.0f}%): +{juru:,.0f} ECV")
        print(f"📈 Juru di {juru:,.0f} ECV aplikadu.")

    def estratu(self):
        """Mostra estratu bankáriu."""
        print(f"\n{'='*45}")
        print(f"  ESTRATU BANKÁRIU — BCA")
        print(f"{'='*45}")
        print(f"  Titular: {self.__titular}")
        print(f"  Konta:   {self.numeru_konta}")
        print(f"{'─'*45}")
        if self.__istóriku:
            for mov in self.__istóriku:
                print(f"  {mov}")
        else:
            print("  (sen movimentus)")
        print(f"{'─'*45}")
        print(f"  SALDU: {self.__saldu:,.0f} ECV")
        print(f"{'='*45}")
```

```python
# Kria konta di Nilton
konta = KontaBankaria("Nilton Ferreira", "BCA-20261234", 100_000)
# ✅ Depózitu di 100,000 ECV konfirmadu.

# Operasons
konta.deposita(25_000)
# ✅ Depózitu di 25,000 ECV konfirmadu.

konta.levanta(40_000)
# ✅ Levantamentu di 40,000 ECV konfirmadu.

konta.levanta(200_000)
# ❌ Saldu insufisienti! Saldu atuál: 85,000 ECV

konta.aplika_juru()
# 📈 Juru di 1,700 ECV aplikadu.

# Tenta modifika saldu diretamenti — KA FUNSIONA!
# konta.saldu = 999_999  # AttributeError: property 'saldu' has no setter
# konta.__saldu = 999_999  # Kria un NOVU atributu, ka muda orijinál!

# Odjá estratu
konta.estratu()
# =============================================
#   ESTRATU BANKÁRIU — BCA
# =============================================
#   Titular: Nilton Ferreira
#   Konta:   ***1234
# ─────────────────────────────────────────────
#   Depózitu: +100,000 ECV
#   Depózitu: +25,000 ECV
#   Levantamentu: -40,000 ECV
#   Juru (2%): +1,700 ECV
# ─────────────────────────────────────────────
#   SALDU: 86,700 ECV
# =============================================
```

```python
# Asésa properties
print(konta.titular)         # Nilton Ferreira
print(konta.numeru_konta)    # ***1234 (maskáru!)
print(konta.saldu)           # 86700

# Istóriku é un kópia — ka pode modifika orijinál
ist = konta.istóriku
ist.append("HAKE!")          # Muda kópia, nau orijinál
print(len(konta.istóriku))   # 4 (orijinál ka mudá)
```

:::callout{type=tip}
Nota kumo `istóriku` ta retorna `self.__istóriku.copy()`. Isu ta previni ki algen modifika lista orijinál di fora! É un padron importanti di enkapsolamentu txomadu "defensive copy".
:::

## Rezumu di @property vs Getters/Setters

```python
# ❌ MANEIRA TRADISIONÁL (verbózu)
class Pesoal_V1:
    def __init__(self, nomi):
        self.__nomi = nomi
    def get_nomi(self):
        return self.__nomi
    def set_nomi(self, nomi):
        self.__nomi = nomi

p = Pesoal_V1("Edna")
print(p.get_nomi())     # Edna
p.set_nomi("Ivete")     # Txeu kódiku pa un koza simples

# ✅ MANEIRA PYTHONIC (limpu i eleganti)
class Pesoal_V2:
    def __init__(self, nomi):
        self.__nomi = nomi
    @property
    def nomi(self):
        return self.__nomi
    @nomi.setter
    def nomi(self, valor):
        self.__nomi = valor

p = Pesoal_V2("Edna")
print(p.nomi)           # Edna (kumo atributu!)
p.nomi = "Ivete"        # Limpu i naturál!
```

  <RezumuItem term="Niveis di asésu">Públiku (`self.nomi`), protejidu (`self._nomi` — só konvensan), privadu (`self.__nomi` — name mangling).</RezumuItem>
