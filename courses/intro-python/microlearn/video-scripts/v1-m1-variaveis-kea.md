## 1) Hook
Bo kre guarda un nomi, un idadi, un presu? Na Python, variável é bo kaxa — bo ta pô kualker kuza dentu!

## 2) Core Concept
Variável é un nomi ki ta aponta pa un valor na memória. Na Python, ka presiza deklara tipu — bo ta skrévi nomi, pô `=`, e pronto. Python ta diskubri tipu sozinhu.

## 3) Show It
```python
nomi = "Maria"
idadi = 23
altura = 1.68
é_estudanti = True

print(f"{nomi} tene {idadi} anu.")
# Output: Maria tene 23 anu.

print(type(nomi))    # <class 'str'>
print(type(idadi))   # <class 'int'>
print(type(altura))  # <class 'float'>
```

## 4) Takeaway
Variável = nomi ki ta aponta pa un valor. Python ta diskubri tipu automátikamenti — kel é **tipu dinámiku**.
