# Python Tene 4 Tipu Báziku — Kual Bo Ta Uza?
Language: Kriolu (KEA)
Audience: Beginner
Format: micro_blog
Topic: Python

Tudu kuza na Python tene un **tipu**. Kuantu bo intendi és 4 tipu báziku, bo ta intendi kumu Python pensa.

```python
nomi = "Amilka"        # str  — textu
idadi = 28              # int  — númeru interu
altura = 1.72           # float — númeru desimal
é_programador = True    # bool — verdaderu ó falsu

print(type(nomi))    # <class 'str'>
print(type(idadi))   # <class 'int'>
```

**4 tipus, 4 uzu:**
- `str` — nomi, mensajis, kualker textu
- `int` — idadi, kontadoris, índisis
- `float` — presus, médias, medidas
- `bool` — desizun: `True` ó `False`

Kuidadu ku mistura! `"25" + 10` da `TypeError`. Konverti priméru: `int("25") + 10` = `35`.

**Takeaway**: Senpri pensa na tipu di bo dadus. Uza `type()` pa verifica e `int()`/`str()`/`float()` pa konverti.
