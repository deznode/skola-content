# Grid

Flexbox é poderozu pa **komponenti** (navs, kards, botan). Grid é poderozu pa **layouts inteiru** — kuandu bo presiza di kontrolu en **dos eixu** ao mesmu tempu.

Flexbox ta poi filhus na fila ou koluna. Grid ta poi-l na un **mapa di kuadradus**. Filhu pode okupa 1 kuadradu, ou 2, ou un linha kompletu. É kumo Excel pa CSS.

Pa 95% di kazus, **flex é más simples**. Ma kuandu flex ka kabe (galerias komplikadu, dashboards), grid é a resposta.

Es dos ferramenta ka ta luta un kontra otru — es ta kompleta un a otru. A pergunta ka é "kual ki é midjór", ma sin "kual ki es kazu ta pidi":

<SectionHeading variant="concept" seq={2}>Kria un grid ku `grid-cols-N`</SectionHeading>

```html
<div class="grid grid-cols-3">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
  <div>Item 4</div>
  <div>Item 5</div>
  <div>Item 6</div>
</div>
```

- **`grid`** ta poi `display: grid`
- **`grid-cols-3`** ta poi 3 kolunas iguais

Browser ta poi 6 itens en 2 linhas di 3 kolunas. **Otomatikamenti.**

Bo pode skolhe kualker numeru di kolunas iguais, te `grid-cols-12`:

```html
<div class="grid grid-cols-1">1 koluna (default)</div>
<div class="grid grid-cols-2">2 kolunas iguais</div>
<div class="grid grid-cols-3">3 kolunas iguais</div>
<div class="grid grid-cols-4">4 kolunas iguais</div>
<!-- te grid-cols-12 -->
```

12 é máximu padran (kumo Bootstrap, pamodi 12 é dividi pa 2, 3, 4 i 6).

### Responsivu — Padran di Layout Inteiru

```html
<div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 xl:grid-cols-4 gap-4">
  <!-- 1 koluna no telefoni → 2 no tablet → 3 no laptop → 4 no desktop -->
</div>
```

Es é **padran más kumun di grid na web**.

<SectionHeading variant="concept" seq={3}>`gap-*` — Spasu Entri Kuadradus</SectionHeading>

Mesmu kumo flex:

```html
<div class="grid grid-cols-3 gap-4">Espasu di 1rem en tudu direksan</div>
<div class="grid grid-cols-3 gap-x-2 gap-y-8">Espasu diferenti horizontal/vertikal</div>
```

<SectionHeading variant="concept" seq={4}>Spanning — Filhu ki Okupa Multiplu Kuadradus</SectionHeading>

```html
<div class="grid grid-cols-3 gap-4">
  <div class="col-span-3 bg-sky-100">Hero — okupa 3 kolunas</div>
  <div class="bg-white p-4">Karta 1</div>
  <div class="bg-white p-4">Karta 2</div>
  <div class="bg-white p-4">Karta 3</div>
  <div class="col-span-2 bg-white p-4">Karta wider — 2 kolunas</div>
  <div class="bg-white p-4">Karta 4</div>
</div>
```

- **`col-span-N`** — filhu okupa N kolunas
- **`row-span-N`** — filhu okupa N rows
- **`col-span-full`** — okupa **tudu** kolunas disponivel

Es é unde grid brilha — flexbox **ka pode faze es**.

<SectionHeading variant="concept" seq={5}>`grid-rows-N` — Rows Definidu</SectionHeading>

Default, rows ta krese otomatikamenti. Pa kontrola:

```html
<div class="grid grid-cols-3 grid-rows-2 h-96 gap-4">
  <!-- 6 spasus dentu di un altura fíxu di 24rem -->
</div>
```

<SectionHeading variant="concept" seq={6}>Tamanhu Personalizadu — Valor Arbitráriu</SectionHeading>

Kuandu bo presiza di kolunas ki **ka son iguais** — ex: sidebar fiksu di 200px + main flexível — uza valor arbitráriu:

```html
<div class="grid grid-cols-[200px_1fr] gap-4">
  <aside>Sidebar fíxu di 200px</aside>
  <main>Main ki okupa tudu spasu ki sobra (1fr = "1 frasan di kuza ki sobra")</main>
</div>
```

**Sintaxi:**
- Separa kolunas ku `_` (underscore, ka spasu — Tailwind ka aseita spasu na klasi)
- `1fr` = "krese pa okupa spasu ki sobra" (kumo `flex-1`)
- Valoris pode ser `px`, `rem`, `%`, `fr`, `min-content`, `max-content`, `minmax(min, max)`

Un izemplu ku trez tracks, splikadu linha-pa-linha:

### `auto-fit` / `auto-fill` — Grid Auto-Responsivu

```html
<div class="grid grid-cols-[repeat(auto-fit,minmax(250px,1fr))] gap-4">
  <!-- Kuantas kolunas ka 250px kabe? Browser disidi. -->
</div>
```

Es é "Pinterest mode" sen media queries. Otimu pa galerias di kards.

<SectionHeading variant="concept" seq={7}>Itens na Pozisan Fiksu — `col-start-N` / `col-end-N`</SectionHeading>

Pa kontrolu izatu di pozisan:

```html
<div class="grid grid-cols-6 gap-4">
  <div class="col-start-2 col-end-5">Komesa na koluna 2, termina antes di koluna 5 (okupa 2,3,4)</div>
</div>
```

Igual: `col-start-2 col-span-3`.

Mesmu padran pa rows: `row-start-N`, `row-end-N`.
