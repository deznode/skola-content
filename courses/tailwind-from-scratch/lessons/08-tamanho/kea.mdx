# Tamanhu (Width / Height)

Tipografia da voz a Resort Brava. Ma hero ta okupa mesmu spasu ki kualker seksan pikinu — visualmente ka ta **domina**. Tamanhu é o ki ta da estrutura: hero é grandi pamodi é primeru kuza ki vizitanti ta odja. Boton ten largura ki ta kombina ku textu — ka mais, ka menus.

Bom notisia: utilidadis di tamanhu **uza mesma skala** di margin i padding ki bo dja konxe.

<SectionHeading variant="concept" seq={1}>`w-*` i `h-*` — Mesma Skala di Spacing</SectionHeading>

Si bo lembra di Lisan 6, `--spacing` é token base ki ta multiplika kada númeru:

```html
<div class="w-4 h-4">w-4 = 1rem (16px), h-4 = 16px</div>
<div class="w-12 h-12">w-12 = 3rem (48px)</div>
<div class="w-32 h-32">w-32 = 8rem (128px)</div>
<div class="w-96 h-96">w-96 = 24rem (384px) — máximu padraun</div>
```

| Klasi | Tamanhu |
|---|---|
| `w-1` | 4px |
| `w-4` | 16px |
| `w-8` | 32px |
| `w-12` | 48px |
| `w-16` | 64px |
| `w-24` | 96px |
| `w-32` | 128px |
| `w-48` | 192px |
| `w-64` | 256px |
| `w-96` | 384px |

`h-*` ta funsiona identikamente.

<SectionHeading variant="concept" seq={2}>Valoris Spesial — Frasan, Tela, Total</SectionHeading>

### Frasan — `w-1/2`, `w-1/3`, `w-1/4`...

```html
<div class="w-1/2">Metadi di kontainer pai</div>
<div class="w-1/3">Un tersu</div>
<div class="w-2/3">Dos tersu</div>
<div class="w-1/4">Un kuartu</div>
<div class="w-3/4">Trez kuartu</div>
<div class="w-1/5">Un kintu</div>
<div class="w-1/6">Un sestu</div>
<div class="w-1/12">Un duzia</div>
```

Suporta dinominador 2, 3, 4, 5, 6 i 12. (12 é poderozu pa grids — 12 é divisivel pa 2, 3, 4 i 6.)

### Tela — `w-screen`, `h-screen`

```html
<div class="w-screen">Toma tudu largura di viewport (100vw)</div>
<div class="h-screen">Toma tudu altura di viewport (100vh)</div>
```

`h-screen` é klásiku pa **hero ki ta okupa tudu ekran** kuandu página karrega.

### Total — `w-full`, `h-full`

```html
<div class="w-full">100% di kontainer pai (ka di viewport)</div>
<div class="h-full">100% di altura di pai</div>
```

Kel diferensa entri `w-full` i `w-screen` é un di kez ki más ta konfundi. Es tabela ta separa-s:

### Auto i 0

```html
<div class="w-auto">Browser ta dezidi (default)</div>
<div class="w-0">Sen largura (ezatu zero)</div>
<div class="w-px">1px (kasu spesial)</div>
```

<SectionHeading variant="concept" seq={3}>`min-*` i `max-*` — Limita</SectionHeading>

Largura/altura mínimu i máximu funsiona ku mesmu família di valoris:

```html
<div class="min-w-32">Minimu di 8rem — ka inkolhe abaixu di kel</div>
<div class="max-w-2xl">Máximu di 42rem — ka stika alén di kel</div>
<div class="min-h-screen">Minimu di 100vh — Bom pa página intera ku konteúdu pikinu</div>
```

**Valoris spesial pa max-width** (similar a `container`):

| Klasi | Tamanhu |
|---|---|
| `max-w-xs` | 20rem (320px) |
| `max-w-sm` | 24rem (384px) |
| `max-w-md` | 28rem (448px) |
| `max-w-lg` | 32rem (512px) |
| `max-w-xl` | 36rem (576px) |
| `max-w-2xl` | 42rem (672px) |
| `max-w-3xl` | 48rem (768px) |
| `max-w-4xl` | 56rem (896px) |
| `max-w-5xl` | 64rem (1024px) |
| `max-w-6xl` | 72rem (1152px) |
| `max-w-7xl` | 80rem (1280px) |
| `max-w-prose` | ~65 caraktér (ótimu pa ler) |

:::callout{type=tip}
`max-w-prose` é otimizadu pa parágrafu di le. Limita largura na ~65 caraktér — ki é largura fásil pa odju (ideal pa leitura). Otimu pa artigus i blog posts.
:::

<SectionHeading variant="concept" seq={4}>Valoris Arbitráriu — `w-[300px]`</SectionHeading>

Si ninhun valor padran di skala kabe:

```html
<div class="w-[300px]">Egzatamenti 300px</div>
<div class="h-[42vh]">42% di viewport height</div>
<div class="min-h-[calc(100vh-4rem)]">Tudu viewport menos 4rem di header</div>
```

Valoris arbitráriu é último rekursu — si bo ta uza un valor más di un, defina token na `@theme`:

```css
@theme {
  --spacing-hero: 42vh;
}
```

I aplika `h-hero`.

<SectionHeading variant="concept" seq={5}>`size-*` — Largura + Altura Djuntu (v4-current)</SectionHeading>

Si bo kre kuadradu (igual largura i altura), v4 ten `size-*` shorthand:

```html
<!-- Antis -->
<img class="w-16 h-16" src="avatar.jpg">

<!-- Modernu (v4) -->
<img class="size-16" src="avatar.jpg">
```

`size-*` só ta poi largura i altura djuntu. Pa mínimu o máximu **ka ten** `min-size-*`/`max-size-*` — uza `min-w-*` + `min-h-*` (o `max-w-*` + `max-h-*`) separadu. `size-*` é ótimu pa ícons, avatars, thumbnails.

<SectionHeading variant="concept" seq={6}>Propriedadis Lógiku (v4.2)</SectionHeading>

v4.2 traze suporti pa CSS logical properties pa internasionalizasan. Pa língua ki ta le di direita pa skerda (Arábi, Hebraicu), `left/right` ta troka. Logical properties uza `start/end`:

```html
<!-- Físiku — sempri skerda i direita -->
<div class="w-1/2 ml-4">Skerda</div>

<!-- Lójiku — adapta a língua -->
<div class="w-1/2 ms-4">"margin-inline-start"</div>
```

Pa kazus internasional (kumo Skola.dev ku Portuguese, Kriolu i Inglés), físiku é seguru — tudu es língua é LTR. Si bo um dia adisiona Árabi, propriedadis lógiku ta brilha.

<SectionHeading variant="concept" seq={7}>Aplika Tamanhu a Resort Brava</SectionHeading>

### Antis (di Lisan 7)

```html
<section id="hero" class="space-y-4">
  <h2 class="text-amber-600 text-5xl font-bold tracking-tight">Ben Vindo</h2>
  <p class="text-slate-700 text-lg leading-relaxed">Vive Brava sima nunka...</p>
  <a class="bg-amber-500 text-white px-4 py-2 font-semibold">Olha kuartus</a>
</section>
```

Hero ten tipografia ma é pikinu — só toma altura naturalmenti. Bamus pa el okupa metadi di ekran kumo hero verdaderu.

### Dipos

```html
<section id="hero" class="space-y-4 min-h-[50vh]">
  <h2 class="text-amber-600 text-5xl font-bold tracking-tight">Ben Vindo</h2>
  <p class="text-slate-700 text-lg leading-relaxed max-w-2xl">Vive Brava sima nunka...</p>
  <a class="bg-amber-500 text-white px-4 py-2 font-semibold">Olha kuartus</a>
</section>
```

**Mudansas:**

- **Hero:** `min-h-[50vh]` — metadi di altura di viewport, sente importanti
- **Hero p:** `max-w-2xl` — parágrafu ka stika alén di 42rem, fika fásil pa ler

Es é pikinu mudansas, ma na sentidu vizual mudansa é grandi: hero ten **prezensa**.

Inche os dos spasus pa reproduzi kel mudansa na hero:
