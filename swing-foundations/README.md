# Java Swing Foundations

Learn the fundamentals of building desktop applications with Java Swing.

## Prerequisites

- Java 21+ (JDK)
- Maven 3.9+
- IDE with Java support (IntelliJ IDEA or VS Code)

## Setup

```bash
# Navigate to starter code
cd initial/

# Build the project
mvn compile

# Run the application
mvn exec:java
```

## Learning Objectives

By completing this tutorial, you will learn:

- JFrame and window basics
- Common Swing components (JButton, JLabel, JTextField, etc.)
- Component hierarchy and containers
- Basic styling and look-and-feel
- MVC pattern in Swing applications
